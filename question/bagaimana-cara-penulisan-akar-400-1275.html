Diketahui f(4x+5)=8x-2 dan g(x+7)=3x+1.Tentukan f(x)dan g(x)

Jawaban

F(4x + 5) = 8x - 2
g(x + 7) = 3x + 1

f(x) dan g(x) = ... ?

*misal
4x + 5 = a
4x = a - 5
x = (a - 5)/4

f(4x + 5) = 8x - 2
f(a) = 8 [(a - 5)/4] - 2
f(a) = 2 (a - 5) - 2
f(a) = 2a - 10 - 2
f(x) = 2x - 12

*misal
x + 7 = a
x = a - 7

g(x + 7) = 3x + 1
g(a) = 3 (a - 7) + 1
g(a) = 3a - 21 + 1
g(x) = 3x - 20

Jika x1 & x2 adalah akar akar dari persamaan 2x2-3x+5=0 maka persamaan kuadrat baru yg akar akarnya x1-3 dan x2-3 adalah

Jawaban

2x^2 - 3x + 5 = 0
a = 2, b = -3, c = 5
X1 × X2 = c/a = 5/2
X1 + X2 = -b/a = 3/2
• (X1 - 3)(X2 - 3) = X1.X2 - 3X1 - 3X2 + 9
= X1.X2 - 3(X1 + X2) + 9
= 5/2 - 3(3/2) + 9
= 5/2 - 9/2 + 9
= 4/2
= 2
• (X1 - 3) + (X2 - 3) = X1 + X2 - 6
= 3/2 - 6
= -9/2
Persamaan Kuadrat Baru
x^2 - (X1.X2)x + (X1 + X2)
= x^2 - 2x - 9/2
= 2x^2 - 4x - 9
