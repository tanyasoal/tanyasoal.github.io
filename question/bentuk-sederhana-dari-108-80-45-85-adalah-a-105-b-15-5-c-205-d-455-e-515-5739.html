Tentukan himpunan penyelesaian dari
|x - 3| ≥ 6

Jawaban

Bentuk |x - 3| ≥ 6 artinya

a) x - 3 ≥ 6, yaitu
x ≥ 6 + 3
x ≥ 9

atau
b) x - 3 ≤ -6
x ≤ -6 + 3
x ≤ -3

Sehingga himpunan penyelesaiannya
HP = { x | x ≤ -3 atau x ≥ 9, x € R }

Semoga jelas dan membantu
Mapel : Matematika
Materi : Nilai Mutlak

Pembahasan :

Kalau menurut saya begini.

Misal : 1 × 1 ≥ a → x ≤ -a atau x ≥ a

X ≤-a
|x-3|≤ -6
X-3≤-6
X≤ -6+3
X≤ -3


Atau
X ≥a
|x-3|≥6
X≥ 6+3
X≥9

*Jadi HP*
X ≥ 9, x ≤ -3
