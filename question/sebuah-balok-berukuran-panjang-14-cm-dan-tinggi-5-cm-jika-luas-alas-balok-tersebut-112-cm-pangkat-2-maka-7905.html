= 2x (14x8 + 14x5 + 8x5)
= 2x (112 + 70 + 40)
= 2 x 222
= 444 cm²
Dik p = 14, t = 5cm, luas alas(p.l) = 112 cm^2, lp....?

Luas permukaan = 2(pl+pt+lt)

112 = p x l
112 = 14 x l
L = 112/14
L = 8cm

Lp = 2(pl + pt + lt) = 2(14.8 + 14.5 + 8.5) = 2(112+70+40) = 444 cm^2

Jika p=7a-8b-c dan Q= - 3a + 2b - 6c,hasil dari p-2Q adalah....

Jawaban

Materi : Aljabar 
Tingkat : SMP 

p = 7a - 8b - c
q = -3a + 2b - 6c 

Maka : 
⇒ p - 2q 
⇒ 7a - 8b - c - 2(-3a + 2b - 6c)
⇒ 7a - 8b - c + 6a - 4b + 12c
⇒ 7a + 6a - 8b - 4b - c + 12c
⇒ 13a - 12b + 11c
