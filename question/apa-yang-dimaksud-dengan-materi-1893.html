Tiga buah pegas disusun seri, setiap pegas memiliki konstanta sebesar 1300 N/m, 650N/m, dan 425 N/m . Ketiga pegas tersebut diberi gaya sebesar 50 N . Berapakah K total pegas-pegas tersebut?

ㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡ

k1 = 1300 N/m
k2 = 650 N/m
k3 = 425 N/m
F = 50 N

ktotal = ?

Penyelesaian :
< Ketiga pegas disusun secara seri, maka >
1/ktotal = 1/k1 + 1/k2 + 1/k3
1/ktotal = 1/1300 + 1/650 + 1/425
1/ktotal = 0,000769 + 0,00153 + 0,00235
1/ktotal = 0,004652
ktotal = 1/0,004652
ktotal = 214,9
ktotal = 215 N/m

∴ Jadi, konstanta total pegas-pegas tsb adalah 215 N/m atau yg mendekati

jumlah lilitan primer sebuah trafo 5kali dari jumlah lilitan sekunder. Jika kuat arus 2A dan daya keluaran trafo 5W, tegangan primer dan arus sekunder masing-masing adalah?

Jawaban

Diket :
Np = 5 x Ns = Np : Ns = 5 : 1
Ip = 2 A
Ps = 5 w


Dit : Vp &Is.......?


Jwb :

# Arus sekunder

Is = ( Np . Ip ) / Ns
Is = ( 5 . 2 ) /1
Is = 10 A


# Tegangan sekunder

Vs = Ps / Is
Vs = 5 / 10
Vs = 0,5 volt

# Tegangan primer

Vp = ( Np . Vs ) / Ns
Vp = ( 5 . 0,5 ) / 1
Vp = 2,5 volt

Jadi tegangan primernya 2,5 v dan arus sekunder sebesar 10 A
