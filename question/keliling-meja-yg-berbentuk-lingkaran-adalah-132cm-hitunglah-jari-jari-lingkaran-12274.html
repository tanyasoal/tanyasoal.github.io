Keliling meja yg berbentuk lingkaran adalah 132cm hitunglah jari jari lingkaran

Jawaban

K = n D
132 = 22/7 × D
132 ÷ 22/7 = D
132 x 7/22 = D
6 x 7 = D
42 = D
r = 1/2D
r = 1/2 *42
r = 21
Diket : kll = 132 cm
Ditanya : r...?
jawab :
kll=2πr
132=2*22/7*r
r = 132*2*7/22
r = 6*2*7
r = 84 cm

Jawaban

Panjang sisi = √3600 = 60
volume kubus = s x s x s = 60 x 60 x 60 = 216.000cm3 = 216 liter
banyak ember = volume kubus / volume ember
= 216 / 5
= 43,2
jadi banyak ember yang diperlukan kurang lebih 43 buah
