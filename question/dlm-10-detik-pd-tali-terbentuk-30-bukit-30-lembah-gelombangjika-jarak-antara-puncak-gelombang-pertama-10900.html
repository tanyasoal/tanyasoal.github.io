Seseorang dapat membaca dengan jelas pada jarak paling dekat 50cm.agar ia dapat membaca dengan jelas pada jarak 30 cm,kekuatan lensa kacamata yang d perlukan adalah....dioptri

Jawaban

Seseorang dengan jelas membaca pada jarak paling dekat sejauh Punctum Proximum (PP) atau titik dekat
PP = 50 cm
ingin melihat jelas pada jarak baca normal
Sn = 30 cm
maka kekuatan lensa (Power) yang harus digunakan adalah

P = 100/Sn - 100/PP
P = 100/30 - 100/50
P = 4/3 Dioptri
atau P = 1,33 Dioptri

benda masa 3kg kalori jenisnya 460j/kg°c dipanaskan dari suhu 10° menjadi 60°c tentukan banyaknya kalori yg di perlukan

Jawaban

Mapel : IPA Fisika 
Kelas : 8 SMP/VIII SMP
Bab : Kalor dan Suhu 
Tentang : Kalor Jenis 

Pembahasan Soal :
Benda masa 3 kg kalori jenisnya 460 J/kg°C dipanaskan dari suhu 10°C menjadi 60°C tentukan banyaknya kalori yg di perlukan...

Pembahasan Jawaban :
DiKetahui :
m(benda) = 3 kg 
C = 460 J/kg°C 
T1 = 10°C 
T2 = 60°C 

DiTanya :
Q =....?

DiJawab :
Rumus mencari Q
Yaitu :
Q = m(benda) x C x ΔT 
Q = m(benda) x C x ( T2 - T1 )
Q = 3 kg x 460 J/kg°C x ( 60°C - 10°C )
Q = 3 kg x 460 J/kg°C x 50°C 
Q = 1.380 x 50 
Q = 69.000 Joule 
Q = 69 kJ.
Jadi, Kalor yang dibutuhkan adalah 69.000 J/ 69 kJ.

Keterangan :
m(benda) = Massa benda ( kg )
T1 = Suhu pertama ( °C )
T2 = Suhu kedua ( °C )
ΔT = Selisih suhu ( °C )
C = Kalor Jenis benda ( J/kg°C )
Q = Banyak kalor ( Joule )
