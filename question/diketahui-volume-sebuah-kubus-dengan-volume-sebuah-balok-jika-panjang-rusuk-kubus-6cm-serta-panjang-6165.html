Persamaan garis lurus mellaui tituk N(-2,5) dan tegak lurus dengan garis 4x+3y=12 adalah

Jawaban

Cari gradien dari persamaan 4x + 3y = 12

Bentuk umum persamaan garis
ax + by + c = 0

untuk persamaan tersebut :
a = 4
b = 3
c = -12

m = -a/b = -4/3

Karena tegak lurus,
m1 . m2 = -1
-4/3 . m2 = -1
m2 = 3/4

Persamaan garis melalui titik (-2, 5)
(y - y1) = m(x - x1)
(y - 5) = 3/4(x - (-2))
4(y - 5) = 3(x + 2)
4y - 20 = 3x + 6
3x - 4y = -20 - 6
3x - 4y = -26

Jawaban

A = 6
b = 9 - 6 = 3
Un = 183

Un = a + (n - 1)b
183 = 6 + (n - 1)3
3(n - 1) = 183 - 6
3(n - 1) = 177
n - 1 = 177 / 3
n - 1 = 59
n = 59 + 1
n = 60

Sn = n/2(a + Un)
S60 = 60/2(6 + 183)
S60 = 30 x 189
S60 = 5.670
