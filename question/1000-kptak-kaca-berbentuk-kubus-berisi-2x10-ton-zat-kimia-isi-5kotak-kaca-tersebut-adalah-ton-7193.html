= 112 : 14 = 8 cm

= 2x (14x8 + 14x5 + 8x5)
= 2x (112 + 70 + 40)
= 2 x 222
= 444 cm²
Dik p = 14, t = 5cm, luas alas(p.l) = 112 cm^2, lp....?

112 = p x l
112 = 14 x l
L = 112/14
L = 8cm
