Jawaban

Maaf untuk soal, terpaksa ada yang saya edit supaya penyelesaiannya dapat ditentukan.
sin a = 8/17
cos a = 15/17
tan a = 8/15
tan b = 12/5 <--- ini bagian yang saya edit
cos b = 5/13
sin b = 12/13
sin (a + b)
= sin a.cos b + cos a.sin b
= (8/17)(5/13) + (15/17)(12/13)
= 40/221 + 180/221
= (40 + 180)/221
= 220/221

sin (a - b)
= sin a.cos b - cos a.sin b
= (8/17)(5/13) - (15/17)(12/13)
= 40/221 - 180/221
= (40 - 180)/221
= -140/221
