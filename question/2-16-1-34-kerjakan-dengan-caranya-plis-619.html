Melewati Rumah Neila *

berapakah jarak rumah C ke rumah B = 4.600 m - 2.500 m = 2.100 m

hasil m:n adalah ...

Jawaban

Jawabannya '


Mapel : Matematika
Bab : Operasi Bilangan
Sub Bab : Perbandingan

Pembahasan :

m = 16 2/3% = 50%
n = 66 2/3% = 200%

Maka '

m : n
= 50% : 200%
= 50/100 : 200/100
= 1/2 : 2/1
= 1/2 × 1/2
= 1/4 → Jwbnya
