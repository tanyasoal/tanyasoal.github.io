Diketahui suku ke2 dan suku ke5 dari dari suatu deret geometri berturut-turut adalah 81 dan 3.Jumlah 6 suku pertama deret tersebut adalah...

Jawaban

Cari U1 dulu
U1 + b = 81
U1 + 4b = 3 -
        -3b = 78
           b = -26 

U1 + b = 81
U1 - 26 = 81
U1 = 81 + 26
U1 = 107 

Sn = n/2 (2U1 + (n-1)b)
S6 = 6/2 (2.107 + (5)-26)
S6 = 3 (214 - 130)
S6 = 3 (84)
S6 = 252

Jadi, jumlah 6 suku pertama 252
