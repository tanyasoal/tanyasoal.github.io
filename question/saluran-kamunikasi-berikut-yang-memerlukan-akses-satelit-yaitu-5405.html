1. Mengapa malware tidak bisa terdeteksi menggunakan antivirus?

Karena Malware merupakan sejenis perangkat lunak berbahaya, tidak seperti virus yang merupakan kode program yang menempel pada sebuah program atau file dan dapat diatasi dengan Antivirus. Untuk mengatasi masalah malware, dibutuhkan software Antimalware.

2. Kenapa malware bisa meyebabkan keusakan sistem yang lebih parah ketimbang virus?

Karena Malware merupakan perangkat lunak berbahaya yang sengaja diciptakan untuk mengganggu operasi komputer, mengumpulkan data sensitif atau mendapatkan akses ke sistem pribadi. Malware dapat menyebar secara otomatis atau membutuhkan campur tangan pengguna, tidak seperti virus yang hanya memasangkan dirinya ke file, folder atau program yang dapat dijalankan. Beberapa jenis malware juga dapat dikendalikan dari jarak jauh dan dapat menyebar dengan cepat.
1. Tidak semua malware dapat terdeteksi dengan antivirus karena ada malware yang lebih canggih dari antivirus. selain itu, tidak semua antivirus dapat mendeteksi malware
2. malware dapat menghapus file sistem atau memodifikasi sistem mu sehingga kerusakan sistem akan jauh lebih rentan. contohnya adalah MEMZ. dia mengubah file untuk booting sehingga komputer tidak dapat boot
