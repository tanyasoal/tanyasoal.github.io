1.Diketahui suatu balok dengan ukuran 26 cm×14 cm×10 cm.Hitunglah: A.luas permukaan balok
B.panjang rusuk/kerangka

Jawaban

 A.       LP balok = 2 * (p*l) + (p*t) + (l*t)
= 2 * ( 26*14 ) + ( 26*10 ) + ( 14*10 )
= 2 * 364 + 260 + 140
= 2 * 764 
= 1528 cm

B.      panjang rusuk balok
= 4 * ( p + l + t )
= 4 * ( 26 + 14 + 10 )
= 4 * 50
= 200 cm

Suatu karton berukuran 3m×2m akan dibuat 5 kubus dengan panjang rusuk 50cm, maka ukuran karton minimal untuk menambah kekurangan karton adalah...

Jawaban

1 luas kubus = 6 × S^2
= 6 × 50^2
= 6 x 2500
= 15000 cm2
maka jika lima kubus = 5 × 15000 = 75000 cm2
= 7,5 m2
luas karton = 3 × 2 = 6 m2

ukuran kantor minimal untuk menambah kekurangan karton = 7,5 - 6 = 1,5 m2
