Pusat jari jari lingkaran dengan persamaan x^+y^-4x-8y+15=0 berturut turut adalah

Jawaban

Pembahasan :

diberikan persamaan lingkaran
x² + y² - 4x - 8y + 15 = 0
ditanya pusat dan jari-jari lingkaran

persamaan umum lingkaran
(x - a)² + (y - b)² = r²
dengan pusat (a, b)

x² + y² - 4x - 8y + 15 = 0 → kita buat menjadi bentuk persamaan umum lingkaran
x² - 4x + y² - 8y = -15
x² - 4x + (1/2 x -4)² + y² - 8y + (1/2 x -8)² = -15 + (1/2 x -4)² + (1/2 x -8)²
x² - 4x + 4 + y² - 8y + 16 = -15 + 4 + 16
(x - 2)² + (y - 4)² = 5

pusat lingkaran = (2, 4)
jari-jari lingkaran = √5

Dari sebuah deret geometri diketahui suku pertama dan rasip masing masing 3 dan 2 jika s=765 maka nilai n

Jawaban

Dari soal telah kita ketahui bahwa
a=3
r=2, r>1
sn=765
maka
sn=a((r^n)-1)/r-1=765
sn=3((2^n)-1)/2-1=765
sn=3((2^n)-1)=765
sn=((2^n)-1)=765/3
sn=(2^n)-1=255
sn=2^n=256
sn=2^n=2^8
maka kita peroleh n=8
