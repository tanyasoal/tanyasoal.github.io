sebuah balok berukuran panjang 14 cm dan tinggi 5 cm jika luas alas balok tersebut 112 cm pangkat 2 maka luas permukaan nya adalah

Jawaban

Lebar = Luas : panjang
= 112 : 14 = 8 cm
Luas permukaan balok..
= 2x (pxl + pxl + lxt)
= 2x (14x8 + 14x5 + 8x5)
= 2x (112 + 70 + 40)
= 2 x 222
= 444 cm²
Dik p = 14, t = 5cm, luas alas(p.l) = 112 cm^2, lp....?

Luas permukaan = 2(pl+pt+lt)

112 = p x l
112 = 14 x l
L = 112/14
L = 8cm

Lp = 2(pl + pt + lt) = 2(14.8 + 14.5 + 8.5) = 2(112+70+40) = 444 cm^2

Diagonal ruang suatu kubus adalah akar 108 cm hitunglah panjang rusuk dan luas permukaan

Jawaban

Diketahui :
Kubus
Diagonal ruang = √108

Ditanya :
Rusuk dan Luas Permukaan

Dijawab :
Panjang Diagonal Ruang = √(3s²)

√(3s²) = √108
3s² = 108
s² = 36
s = 6

Jadi, rusuk = 6 cm.

Luas Permukaan = 6s²
Lp = 6 × 6²
Lp = 216 cm²
