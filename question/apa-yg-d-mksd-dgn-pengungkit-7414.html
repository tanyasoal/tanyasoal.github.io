Sebuah bola bermassa 0,6 kg mula-mula diam, kemudian setelah dipukul dengan tongkat dan kecepatan bola menjadi 40 m/s. Hitunglah besarnya impuls dari gaya pemukul tersebut!
Pembahasan:
Diketahui:
m = 0,5 kg
v1 = 0 m/s (karena bola mula-mula dalam keadaan diam)
v2 = 40 m/s
Ditanya: Impuls (I)
Jawab:
I = p2 – p1
I = m (v2 – v1)
I = 0,5 (40 – 0) = 20 Ns
Jadi impuls dari gaya pemukul tersebut adalah 20 Ns.

Sebuah Transformator memiliki perbandingan lilitan primer dan lilitan sekunder 3:5 di hubungkan dengan sumber tegangan ac dengan tegangan 220 volt dan kuat arus 4 apmere.tentukan: efisiensi trafo

Jawaban

Diket :
Np : Ns = 3 : 5
Vp = 220 v
Ip = 4 A


Dit : Efisiensi trafo.......?


Jwb :

# Tegangan sekunder

Vs = ( Ns . Vp ) / Np
Vs = ( 5 . 220 ) / 3
Vs = 1100 / 3
Vs = 367 volt

# Arus sekunder

Is = ( Vp . Ip ) / Vs
Is = ( 220 . 4 ) /367
Is = 880 / 367
Is = 2,4 A

# Efisiensi

ef = ( Vs . Is / Vp . Ip ) . 100%
ef = ( 367 . 2,4 / 220 . 4) . 100%
ef =( 880,8 / 880) .100%
ef = 100%

Jadi efisiensi trafo tsb 100%
