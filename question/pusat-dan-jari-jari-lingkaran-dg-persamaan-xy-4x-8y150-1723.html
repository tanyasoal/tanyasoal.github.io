Maka
Pusat (2,4)
r = √5
Kelas 11 Matematika
Bab Persamaan Lingkaran

x² + y² - 4x - 8y + 15 = 0
x² - 4x + y² - 8y = -15
(x - 2)² + (y - 4)² = -15 + 2² + 4²
(x - 2)² + (y - 4)² = 5

r² = 5
r = √5

Pusat = (2, 4)
Jari-jari = √5
