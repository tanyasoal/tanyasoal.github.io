Diketahui barisan geometri dengan U1 = 1 dan U4 = 8 jumlah suku pertama deret geometri adalah

Jawaban

Diketahui :
Deret Geometri
U1 = 1
U4 = 8

Ditanya :
S1 = ?

Dijawab :
U1 = a = 1
U4 = a.r^3 = 8

       U4/U1 = 8/1
a.r^3/a = 8
            r^3 = 2^3
                r = 2
 a.r^3 = 8
a.2^3 = 8
   a. 8 = 8
       a = 8/8
       a = 1

Sn = a.(r^n -1)/(r - 1)
S1 = 1.(2^1 -1)/(2 - 1)
S1 = 1.(2 - 1))/(1)
S1 = 1.1/1
S1 = 1

Jadi jumlah suku pertama deret tersebut adalah 1

Harga 3kg mangga dan 2kg jeruk Rp116.000,00, harga 4kg mangga dan 5kg jeruk Rp199.000,00. Jadi berapa harga 1kg jeruk dan 2kg mangga?

Jawaban

Jawaban ⇒ Rp 64.000,00

❏ Misalkan
mangga » x
jeruk » y

sehingga
3 kg mangga + 2 kg jeruk = 116.000
=> 3x + 2y = 116.000

4 kg mangga + 5 kg jeruk = 199.000
=> 4x + 5y = 199.000

❏ Eliminasi variabel x
4x + 5y = 199.000 ( ×3 )
3x + 2y = 116.000 ( ×4 )

12x + 15y = 597.000
12x + 8y = 464.000 --
-----------------------------
7y = 133.000
y = 133.000 ÷ 7 = 19.000

❏ Substitusikan y = 19.000 ke

3x + 2y = 116.000
3x + 2 (19.000) = 116.000
3x + 38.000 = 116.000
3x = 116.000- 38.000
3x = 78.000
x = 78.000 ÷ 3
x = 26.000

❏ Harga 1 kg jeruk dan 2 kg mangga
= 1x + 2y
= 1 (26.000) + 2 (19.000)
= 26.000 + 38.000
= 64.000

Jadi, harganya adalah Rp 64.000,00

Jawaban

Jumlah seluruh ayam = 40 ekor
25 betina dengan (15 berwarna putih)
jumlah ayam jantan :
jumlah seluruh ayam - jumlah ayam betina = 40 - 25 = 15 ayam jantan

jika ada 22 ekor ayam putih maka,
22-15(jumlah ayam betina berwarna putih) = 7 ekor ayam jantan berwarna putih

jumlah ayam jantan yang Tidak berwarna putih adalah 15 - 7 = 8 ekor
