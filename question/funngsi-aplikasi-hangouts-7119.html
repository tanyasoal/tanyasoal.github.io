Jawaban

Pusat (a,b) = (-1,-4)
Titik (x,y) = (1,-2)
(x-a)^2 + (y-b)^2 = r^2
(1-(-1))^2 + ((-2)-(-4))^2 =r^2
2^2 + 2^2 = r^2
4 + 4 = r^2
8 = r^2
(x-(-1))^2 + (y-(-4))^2 = 8
(x+1)^2 + (y+4)^2 = 8
(x^2 + 2x + 1) + (y^2 + 8y + 16) = 8
x^2 + y^2 + 2x + 8y + 9 = 0
