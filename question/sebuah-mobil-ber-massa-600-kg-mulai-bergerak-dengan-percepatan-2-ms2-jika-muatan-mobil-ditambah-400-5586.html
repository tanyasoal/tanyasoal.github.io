b. 1200 N
c. 1600 N
d. 1800 N
PAKE CARA YA,,,
MAKASIH,,,

Jawaban

F = m x a = 600 x 2 = 1200 N
agar percepatan menjadi 3 m/s² ⇒ F = 1000 x 3 = 3000 N
gaya tambahan  = 3000-1200 = 1800N

daya setrika di bandingkan daya kipas angin 5:2.Kedua alat digunakan selama 4 jam per hari.Dalam 1 bulan (30hari)kedua alat memerlukan energi listrik 58,8 kwh.Daya setrika dan daya kipas angin masing-masing adalah.....

Jawaban

Diket :
P1 : P2 = 5 : 2
t = 4 jam / hr
W = 58,8 kwh = 58.800 wh/ bln ( 30 hr )

Dit : Daya alat masing masing .....?


Jwb :

# Total daya kedua alat

P = W / t . 30
P = 58.800 / 4 . 30
P = 58.800 / 120
P = 490 watt

# Daya setrika

P1 = (5 / (5+2 ) . P
P1 = (5 / 7 ) . 490
P1 = 0,714 . 490
P1 = 350 watt


# Daya kipas angin

P2 = (2 / ( 5+ 2 ) . P
P2 = ( 2 / 7 ) . 490
P2 = 0,285 . 490
P2 = 140 watt


Jadi daya setrika 350 w dan kipas angin
