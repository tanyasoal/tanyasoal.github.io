1. diket:
F1= 100 N
F2= 250 kg
d1= 40 cm

ditanya: d1?
jawab:
F1/A1 = F2/A2
d1= F2.F1 / d2^2
d1= 250.100 / 40^2
d1= 250.100 / 1600
d1= 250 / 16
d1= 15,625cm
d1= 15,6cm (bila dibulatkan)

2. diket:
wu(udara)= 500 N
wa(air)= 250 N
pair= 1000 kg/m^3

ditanya: Fa & pb?
jawab:
Fa= wa-wu
Fa= 500 - 250
Fa= 250N <= gaya apung

=> cari vol.air/va?
Fa= pair.g.va
250 = 1000.10.va
250 = 10000va
va= 250 / 10000
va= 0,025 m^3

=> cari massa benda/m?
m= wu/g
m= 500/10
m= 50 kg

=> penyelesaian:
pb= m/va
pb= 50 / 0,025
pb= 800 kg/m^3

3. diket:
pair= 1000 kg/m^3
v= 60%

ditanya: pb?
jawab:
pb= p.V / Vb
pb= 1000. 3/5 / Vb
pb= 600 kg/m3
