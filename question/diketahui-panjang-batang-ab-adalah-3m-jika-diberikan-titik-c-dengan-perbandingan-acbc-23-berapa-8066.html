Pusat jari jari lingkaran dengan persamaan x^+y^-4x-8y+15=0 berturut turut adalah

Jawaban

Pembahasan :

diberikan persamaan lingkaran
x² + y² - 4x - 8y + 15 = 0
ditanya pusat dan jari-jari lingkaran

persamaan umum lingkaran
(x - a)² + (y - b)² = r²
dengan pusat (a, b)

x² + y² - 4x - 8y + 15 = 0 → kita buat menjadi bentuk persamaan umum lingkaran
x² - 4x + y² - 8y = -15
x² - 4x + (1/2 x -4)² + y² - 8y + (1/2 x -8)² = -15 + (1/2 x -4)² + (1/2 x -8)²
x² - 4x + 4 + y² - 8y + 16 = -15 + 4 + 16
(x - 2)² + (y - 4)² = 5

pusat lingkaran = (2, 4)
jari-jari lingkaran = √5

panjang jari jari dua lingkaran masing masing adalah 20 cm dan 15 cm. jika jarak pusat kedua lingkaran adalah 13 cm, tentukan panjang garis singgung persekutuan luarnya?

Jawaban

D = panjang garis singgung
p = jarak kedua titik pusat
r1 = jari-jari 1
r2 = jari-jari 2

d luar
= √p kuadrat - (r1 - r2) kuadrat
= √13 kuadrat - (20 - 15) kuadrat
= √13 kuadrat - 5 kuadrat
= √169 - 25
= √144
= 12 cm
