Harga 3kg mangga dan 2kg jeruk Rp116.000,00, harga 4kg mangga dan 5kg jeruk Rp199.000,00. Jadi berapa harga 1kg jeruk dan 2kg mangga?

Jawaban

Jawaban ⇒ Rp 64.000,00

❏ Misalkan
mangga » x
jeruk » y

sehingga
3 kg mangga + 2 kg jeruk = 116.000
=> 3x + 2y = 116.000

4 kg mangga + 5 kg jeruk = 199.000
=> 4x + 5y = 199.000

❏ Eliminasi variabel x
4x + 5y = 199.000 ( ×3 )
3x + 2y = 116.000 ( ×4 )

12x + 15y = 597.000
12x + 8y = 464.000 --
-----------------------------
7y = 133.000
y = 133.000 ÷ 7 = 19.000

❏ Substitusikan y = 19.000 ke

3x + 2y = 116.000
3x + 2 (19.000) = 116.000
3x + 38.000 = 116.000
3x = 116.000- 38.000
3x = 78.000
x = 78.000 ÷ 3
x = 26.000

❏ Harga 1 kg jeruk dan 2 kg mangga
= 1x + 2y
= 1 (26.000) + 2 (19.000)
= 26.000 + 38.000
= 64.000

Jadi, harganya adalah Rp 64.000,00

Pembahasan 

Diminta untuk menentukan persamaan direktris elips dengan persamaan 
4x² + 25y² - 24x - 100y + 36 = 0

Kita olah persamaan elips bentuk implisit di atas menjadi bentuk eksplisit.

⇔ 4x² - 24x + 25y² - 100y + 36 = 0

⇔ 4(x² - 6x) + 25(y² - 4y) + 36 = 0

⇔ 4(x² - 6x + 9 - 9) + 25(y² - 4y + 4 - 4) + 36 = 0

⇔ 4((x - 3)² - 9) + 25((y - 2)² - 4) + 36 = 0

⇔ 4(x - 3)² - 36 + 25(y - 2)² - 100 + 36 = 0

⇔ 4(x - 3)² + 25(y - 2)² = 100; kedua ruas dibagi 100

∴
