Gradien garis dengan persamaan 2y kurang 6 x kurang 5 = 0

Jawaban

2y-6x-5=0

2y=6x+5

y=3x + 5/2

jadi gradien nya adalah 3(koefisien x)

^-^
Bentuk umum persamaan :
ax + by + c = 0

Untuk persamaan 2y - 6x - 5 = 0
a = -6
b = 2
c = -5

gradien = -a/b
             = -(-6)/2
             = 3
