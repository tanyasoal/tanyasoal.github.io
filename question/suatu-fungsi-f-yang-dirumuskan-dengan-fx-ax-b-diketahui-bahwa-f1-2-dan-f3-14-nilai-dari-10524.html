Suatu fungsi f yang dirumuskan dengan f(x) = ax + b diketahui bahwa f(1) = -2 dan f(3) = -14. Nilai dari f(7) adalah...

Jawaban

F(x) = ax + b
f(1) => a.1 + b = -2
f(3) => a.3 + b = -14
________________ -
-2a = 12
a = -6

a + b = -2
-6 + b = -2
b = 4

f(7) => ax + b
= -6.7 + 4
= -42+4
= -38
