Jawaban

Panjang tangga = 2,5m
panjang alas tangga ke dasar tembok = 1,5m
panjang tembok = 3,5m

panjang tembok dibawah tangga
= √((2,5)²-(1,5)²) = √(6,25 - 2,25)
= √4 = 2

panjang tembok diatas tangga
= 3,5 - 2 = 1,5m (b)

Arbi Akan membuat dua buah aquarium tanpa tutup berbentuk balok dengan panjang 120cm, lebar 50cm dance tinggi 60cm berapa meter persegi, kaca yan diperlukan arbi untuk membuat dua akuarium tersebut?

Jawaban

P = 120 cm
l = 50 cm
t = 60 cm

Luas permukaan balok tanpa tutup
= (p x l) + 2(pt + lt)
= (120 x 50) + 2(120x60 + 50x60)
= 6.000 + 2(7.200 + 3.000)
= 6.000 + 20.400
= 26.400 cm²
= 2,64 m²
