reni menabungdi bank sebesar RP4.000.000,00dengan suku bunga 15%setahun. jika ia ingin menginginkan tabungannya menjadi Rp4.400.000,00, lama ia harus menabung adalah..

Jawaban

Mapel : Matematika
Materi : Aritmatika Sosial
Kelas  : 7

Tabungan (M)   = Rp. 4.000.000
Suku bunga (p) = 15 %
Besar Bunga     = 4.400.000 - 4.000.000
                          = Rp. 400.000

Lama Menabung :

n = (Besar Bunga x 12 bulan) / ( p x M)
n = (400.000 x 12) / (15% x 4.000.000)
n = (4.800.000) / 600.000
n = 8 bulan


Jadi, Lama menabung ada 8 bulan

Persamaan garis melalui titik A(-3,4) fan B(2,-5) adalah

Jawaban

Melalui titik A(-3, 4) dan B(2, -5)

x1 = -3
x2 = 2
y1 = 4
y2 = -5

Persamaan garis
(y - y1) / (y2 - y1) = (x - x1) / (x2 - x1)
(y - 4) / (-5 - 4) = (x - (-3)) / (2 - (-3))
(y - 4) / (-9) = (x + 3) / 5
5(y - 4) = -9(x + 3)
5y - 20 = -9x - 27
9x + 5y = -27 + 20
9x + 5y = -7
atau
9x + 5y + 7 = 0

Jawaban

Diketahui =
Rama membutuhkan 15 1/3 kg beras
Beras yg dimiliki 2 1/2 dan 5 4/8
Ditanyakan =
Berapa beras yg dibutuhkan Rama agar sesuai dgn kebutuhannya ?
Jawab =
15 1/3 - (2 1/2 + 5 4/8) =
46/3 - (5/2 + 44/8) =
368/24 - (60/24 + 132/24) =
368/24 - 192/24 =
176/24 = 7 8/24
= 7 1/3
Jadi , beras yg dibutuhkan Rama agar sesuai dgn kebutuhannya adalah 7 1/3 kg
