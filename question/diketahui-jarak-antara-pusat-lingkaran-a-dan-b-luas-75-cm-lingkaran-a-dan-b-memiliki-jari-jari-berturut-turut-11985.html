Grafik fungsi f(x) =x³-3x²-9x+15-5 naik pada interval ...

Jawaban

Pembahasan :

f(x) = x³ - 3x² - 9x + 15 - 5
naik pada interval ...

kita gunakan turunan
f(x) = x³ - 3x² - 9x + 15 - 5
f(x) = x³ - 3x² - 9x + 10
f'(x) = 3x² - 6x - 9 
naik pada saat f'(x) > 0

f'(x) = 3x² - 6x - 9 kita kecilkan dengan membagi 3
x² - 2x - 3 > 0
(x - 3)(x + 1) > 0
kita dapati dua bilangan -1 dan 3
kita cek nol fungsi atau x = 0

3x² - 6x - 9 > 0
3(0)² - 6(0) - 9 > 0
0 - 0 - 9 > 0
-9 > 0 salah
maka interval naik pada
x < -1 atau x > 3


jadi f(x) = x³ - 3x² - 9x + 15 - 5 naik pada interval x < -1 atau x > 3

Suku ke 20 dari barisan 4, 6, 10, 16, 24,...., adalah ...........

Jawaban

Kelas 9 Matematika
Bab 6 - Barisan dan Deret

Penyelesaian :
4, 6, 10, 16, 24, .....
--2--4---6---8
----2--2---2

a = 4
b = 2
c = 2

Aritmatika tingkat 2
Suku ke-20
Un = a + (n - 1) b + c (n - 1) (n - 2) / 2
U20 = 4 + (20 - 1) 2 + 2 (20 - 1) (20 - 2) / 2
U20 = 4 + (19) 2 + (19) (18)
U20 = 4 + 38 + 342
U20 = 42 + 342
U20 = 384

Jadi, suku ke-20 barisan tersebut adalah 384.
