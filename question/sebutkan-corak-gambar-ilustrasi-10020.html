frekuensi nada c pada suatu alat musik sebesar 264 Hz.Apabila perbandingan antara nada c dengan nada G adalah 24:36,frekuensi nada G sebesar...Hz

Jawaban

~Gelombang Bunyi~

fc = 264 Hz
c : g = 24 : 36

fg =__?

fg = C / G • fc
fg = 24 / 36 • 264
fg = 176 Hz

sebuah sumber bunyi menghasilkan frekuensi 1000 hz.jika kecepatan gelombang bunyi 320m/s,berapakah jumlah gelombang yang merambat selama 5 sekon?

Jawaban

◾ Materi : Getaran dan Gelombang
◾ Sub materi : Gelombang
◾ Mapel : Fisika
◾ Keyword : Gelombang, waves

ㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡ

Diketahui :
f = 1000 Hz
v = 320 m/s
t = 5 sekon

Ditanya :
n = ?

Penyelesaian :
f = n/t
1000 = n/5
n = 5000 gelombang

∴ Jadi, banyaknya gelombang yg merambat selama waktu tsb adalah 5000 gelombang

pennelitian hendak mengukur kedalaman laut , sinyal bunyidikirim ke dasar laut dan ditangkap setelah 3detik kemudian. bila kelajuan bunyi di air laut 1800m/s tentukan kedalaman laut di yempat itu

Jawaban

◾ Materi : Mengukur Kedalaman Laut
◾ Sub materi :
◾ Mapel : Fisika
◾ Keyword :

ㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡ

Diketahui :
v = 1800 m/s
t = 3 detik

Ditanya :
X = ?

Penyelesaian :
X = ½(v. t)
X = ½(1800 . 3)
X = 900(3)
X = 2700 meter

∴ Jadi, kedalaman laut tempat itu adalah 2700 meter
