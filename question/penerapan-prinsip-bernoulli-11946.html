Jawaban

ㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡ

F1 = 300N
F2 = 400 N
F3 = 500 N
t = 4 sekon
μs = 0,4
μk = 0,2
m = 1 ton = 1000 kg
g = 10 m/s²

Ditanya :
W = ?

Penyelesaian :
*Cari gaya gesek maksimum
fsmaks = μs. N
fsmaks = 0,4 (1000.10)
fsmaks = 4000 N

*Cari resultan gaya
Fr = F1+F2+F3
Fr = 300+400+500
Fr = 1200 N

Karena Fr < fsmaks maka mobil tidak berpindah. s = 0

W = F. s
W = 0

∴ Jadi, usaha yg dilakukan oleh ketiga gaya tsb adalah 0
