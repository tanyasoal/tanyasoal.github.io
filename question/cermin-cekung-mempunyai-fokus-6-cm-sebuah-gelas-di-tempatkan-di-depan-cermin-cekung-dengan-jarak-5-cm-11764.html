Byk kalor untuk memanaskan 4 kg air dari suhu 30'c spi mendidih, kjair=4200 J/kg'C

Jawaban

Materi Kelas : Vll
Materi Bab : Kalor

Diketahui

T1 = 30 derajat C
massa air = 4kg
Kalor jenis air (c) = 4200 J/kg derajat C

Karena titik didih air adalah 100 derajat C
maka

delta t = 100 - 30 = 70 derajat C

Ditanya banyak kalor (Q) untuk memanaskan air???

Q = m × c × delta t

Q = 4 × 4200 × 70

Q = 16800 × 70

Q = 1,176000 Joule

Jadi untuk memanaskan air dibutuhkan kalor sebesar 1,176000 Joule atau 1176 KiloJoule.

Q= m.c. delta T
Q = 4 . 4200 . ( 100 - 30 )
Q = 1 176 000 j

benda masa 3kg kalori jenisnya 460j/kg°c dipanaskan dari suhu 10° menjadi 60°c tentukan banyaknya kalori yg di perlukan

Jawaban

Mapel : IPA Fisika 
Kelas : 8 SMP/VIII SMP
Bab : Kalor dan Suhu 
Tentang : Kalor Jenis 

Pembahasan Soal :
Benda masa 3 kg kalori jenisnya 460 J/kg°C dipanaskan dari suhu 10°C menjadi 60°C tentukan banyaknya kalori yg di perlukan...

Pembahasan Jawaban :
DiKetahui :
m(benda) = 3 kg 
C = 460 J/kg°C 
T1 = 10°C 
T2 = 60°C 

DiTanya :
Q =....?

DiJawab :
Rumus mencari Q
Yaitu :
Q = m(benda) x C x ΔT 
Q = m(benda) x C x ( T2 - T1 )
Q = 3 kg x 460 J/kg°C x ( 60°C - 10°C )
Q = 3 kg x 460 J/kg°C x 50°C 
Q = 1.380 x 50 
Q = 69.000 Joule 
Q = 69 kJ.
Jadi, Kalor yang dibutuhkan adalah 69.000 J/ 69 kJ.

Keterangan :
m(benda) = Massa benda ( kg )
T1 = Suhu pertama ( °C )
T2 = Suhu kedua ( °C )
ΔT = Selisih suhu ( °C )
C = Kalor Jenis benda ( J/kg°C )
Q = Banyak kalor ( Joule )
