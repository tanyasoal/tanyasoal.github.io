Persamaan garis lurus mellaui tituk N(-2,5) dan tegak lurus dengan garis 4x+3y=12 adalah

Jawaban

Cari gradien dari persamaan 4x + 3y = 12

Bentuk umum persamaan garis
ax + by + c = 0

untuk persamaan tersebut :
a = 4
b = 3
c = -12

m = -a/b = -4/3

Karena tegak lurus,
m1 . m2 = -1
-4/3 . m2 = -1
m2 = 3/4

Persamaan garis melalui titik (-2, 5)
(y - y1) = m(x - x1)
(y - 5) = 3/4(x - (-2))
4(y - 5) = 3(x + 2)
4y - 20 = 3x + 6
3x - 4y = -20 - 6
3x - 4y = -26

Jawaban

Penyelesaian :
180° = 4x° + (3x + 1)° + (2x - 1)°
180° = 4x° + 3x° + 2x° + 1° - 1°
180° = 9x°
-9x° = -180°
x = -180° / -9°
x = 20

Jadi, nilai x adalah 20
