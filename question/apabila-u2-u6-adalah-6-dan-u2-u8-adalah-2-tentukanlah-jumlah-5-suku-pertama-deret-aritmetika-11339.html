Jawaban

Dik : U2 + U6 = -6
U2 + U8 = -2
Dit : S5?

Jawab :
•U2 + U6 = -6 dipecah menjadi
a + b + a + 5b = -6
2a + 6b = -6
___________ : 2
a + 3b = -3

• U2 + U8 = -2 dipecah menjadi
a + b + a + 7b = -2
2a + 8b = -2
__________ :2
a + 4b = -1

•Dieliminasi
a + 3b = -3
a + 4b = -1
_________ -
-b = -2
b = 2

• Disubtitusi
a + 3b = -3
a = -3 - 3(2)
a = - 3 - 6
a = -9

•Mencari S5
Sn = n/2 (2a + (n-1) b)

S5 = 5/2 (-18 + 8)
S5 = 5/2 (-10)
S5 = -25

Sebuah balok luas pemukaannya 342 cm pesegi. Jika p : l : t = 4 : 3 : 1, maka volume balok itu adalah a.104 cm2 b.124cm2 c.224cm2 d.324cm2

Jawaban

Luas permukaan = 342 cm²
p : l : t = 4 : 3 : 1

misal :
p = 4x
l = 3x
t = x

Luas permukaan = 2(pl + pt + lt)
342 = 2(4x . 3x + 4x . x + 3x . x)
342 = 2(12x² + 4x² + 3x²)
342 / 2 = 19x²
19x² = 171
x² = 171 / 19
x² = 9
x = √9
x = 3

maka
panjang = 4x = 4(3) = 12 cm
lebar = 3x = 3(3)= 9 cm
t = x = 3 cm

Volume = p x l x t
             = 12 x 9 x 3
             = 324 cm³
