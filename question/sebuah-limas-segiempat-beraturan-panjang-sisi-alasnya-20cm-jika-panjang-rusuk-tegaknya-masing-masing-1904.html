Luas permukaan = luas alas + 4 (luas Δ tegak)
                          =( 20 X 20 ) + 4 ( 20 x (1/2 x 20 )
                         =  ( 400 ) + 4( 26 X 10)
                         = 400 + 4 (260)
                        = 400 + 1040
                        = 1440 cm²

Jadi, luas permukaannya adalah 1440 cm²
