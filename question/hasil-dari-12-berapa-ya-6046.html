Jawaban

Pembahasan 

Diminta untuk menentukan persamaan direktris elips dengan persamaan 
4x² + 25y² - 24x - 100y + 36 = 0

Kita olah persamaan elips bentuk implisit di atas menjadi bentuk eksplisit.

⇔ 4x² - 24x + 25y² - 100y + 36 = 0

⇔ 4(x² - 6x) + 25(y² - 4y) + 36 = 0

⇔ 4(x² - 6x + 9 - 9) + 25(y² - 4y + 4 - 4) + 36 = 0

⇔ 4((x - 3)² - 9) + 25((y - 2)² - 4) + 36 = 0

⇔ 4(x - 3)² - 36 + 25(y - 2)² - 100 + 36 = 0

⇔ 4(x - 3)² + 25(y - 2)² = 100; kedua ruas dibagi 100

∴

Diketahui ΔABC sudut A = 30 derajat, sudut B = 45 derajat, dan a = 10cm. Tentukan sudut c, sisi b, dan sisi c

Jawaban

Sudut c
180 - (30+45) = 105 derajat

sisi b

10/ sin 30 = b/ sin 45
10/ ½ = b/ ½√2
kali silang

5√2 = ½b
b = 10√2 cm

sisi c

10/ sin 30 = c/ sin 105
10/ ½ = c/ 0.9659
kali silang

9.659 = ½c
c = 9.659 * 2 = 19.318 cm
