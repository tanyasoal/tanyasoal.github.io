Pusat dari jari jari lingkaran dengan persamaan x2 +y2 -4x -8y +15=0 berturut turut adl

Jawaban

Pembahasan :

diberikan persamaan lingkaran
x² + y² - 4x - 8y + 15 = 0
ditanya pusat dan jari-jari lingkaran

persamaan umu lingkaran
(x - a)² + (y - b)² = r²
dengan pusat (a, b)

x² + y² - 4x - 8y + 15 = 0 → kita buat menjadi bentuk persamaan umum lingkaran
x² - 4x + y² - 8y = -15
x² - 4x + (1/2 x -4)² + y² - 8y + (1/2 x -8)² = -15 + (1/2 x -4)² + (1/2 x -8)²
x² - 4x + 4 + y² - 8y + 16 = -15 + 4 + 16
(x - 2)² + (y - 4)² = 5

pusat lingkaran = (2, 4)
jari-jari lingkaran = √5

Suku ke 3 dan suku 6 suatu deret aritmetika berturut turut adalah 40 dan 60. Jumlah sepuluh suku pertama deret tersebut adalah

Jawaban

Diketahui :
U3 = 40
U6 = 60

Ditanya :
S10 = ...?

Jawab :
Un = a + b(n-1)
U3 = a + b(3-1)
40 = a + 2b ...(1)
U6 = a + b(6-1)
60 = a + 5b ... (2)

=> subsitusikan ke persamaan (1) dan (2)
40 = a + 2b
60 = a + 5b
_________-
-20 = -3b
b = -20/-3
b = 6,6

=> eliminasikan ke persamaan (1) atau (2)
60 = a + 5b
60 = a + 5(6,6)
60 = a + 33
a = 60 - 33
a = 27

=> rumus Un :
Un = a + b(n-1)
Un = 27 + 6,6(n-1)
Un = 27 + 6,6n - 6,6
Un = 20,4 + 6,6n
U10 = 20,4 + 6,6(10)
U10 = 20,4 + 66
U10 = 86,4 => dibulatkan menjadi 86

=> Jumlah sepuluh suku pertama deret tersebut adalah :
Sn = n/2 (a + Un)
S10 = 10/2 (27+ U10)
S10 = 10/2 (27 + 86)
S10 = 5(113)
S10 = 565
