jiak massa benda tersebut 2 kg, percepatan yang akan terus jadi pada benda 10 N ke kiri 10N ke kanan dan 5N ke kanan adalah

Jawaban

Kelas 8 Fisika
Materi Hukum Newton

Penyelesaian :

Massa (m) = 2 kg
F1 = 10 N ke kiri
F2 = 10 N ke kanan
F3 = 5 N ke kanan

Sigma F
= 10 N + 5 N - 10 N
= 15 N - 10 N
= 5 N

Percepatan
= sigma F / massa
= 5 / 2
= 2,5 m/s²

Jadi, percepatan yang terjadi pada benda tersebut adalah 2,5 m/s²

sebuah benda di udara beratnya 100N,volume benda adalah 2000 cm3 jika massa jenis minyak 0,8 gr/cm3 dan g=10m/s2,maka berat benda itu didalam minyak adalah

Jawaban

◾ Materi : Mekanika Fluida
◾ Sub materi : Hukum Archimedes
◾ Mapel : Fisika
◾ Keyword : Hk Archimedes, Gaya Angkat

ㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡ

Diketahui :
Wb = 100 N
V = 2000 cm³ = 0,002 m³
p = 0,8 g/cm³ = 0,8 x 10^-3 kg / 10^-6 m³ = 800 kg/m³
g = 10 m/s²

Ditanya :
Wbf = ?

Penyelesaian :
*Cari gaya angkat ke atas yg dialami benda
Fa = pgV
Fa = (800)(10)(0,002)
Fa = 16 N

*Cari berat benda di fluida
Wbf = Wb - Fa
Wbf = 100 - 16
Wbf = 84 N

∴ Jadi, berat benda di dalam minyak / fluida adalah 84 N

benda masa 3kg kalori jenisnya 460j/kg°c dipanaskan dari suhu 10° menjadi 60°c tentukan banyaknya kalori yg di perlukan

Jawaban

Mapel : IPA Fisika 
Kelas : 8 SMP/VIII SMP
Bab : Kalor dan Suhu 
Tentang : Kalor Jenis 

Pembahasan Soal :
Benda masa 3 kg kalori jenisnya 460 J/kg°C dipanaskan dari suhu 10°C menjadi 60°C tentukan banyaknya kalori yg di perlukan...

Pembahasan Jawaban :
DiKetahui :
m(benda) = 3 kg 
C = 460 J/kg°C 
T1 = 10°C 
T2 = 60°C 

DiTanya :
Q =....?

DiJawab :
Rumus mencari Q
Yaitu :
Q = m(benda) x C x ΔT 
Q = m(benda) x C x ( T2 - T1 )
Q = 3 kg x 460 J/kg°C x ( 60°C - 10°C )
Q = 3 kg x 460 J/kg°C x 50°C 
Q = 1.380 x 50 
Q = 69.000 Joule 
Q = 69 kJ.
Jadi, Kalor yang dibutuhkan adalah 69.000 J/ 69 kJ.

Keterangan :
m(benda) = Massa benda ( kg )
T1 = Suhu pertama ( °C )
T2 = Suhu kedua ( °C )
ΔT = Selisih suhu ( °C )
C = Kalor Jenis benda ( J/kg°C )
Q = Banyak kalor ( Joule )
