Jawaban

Cara 1
Dik : T1 = 28°c
M2 = 200g
T2 = 45°c
T = 42°c
C2 = 2100J/KG°C
C1 = 450J/KG°C
DIT : M1 = ?
Dij: Q1 = Q2
= M1.C1.Delta(T1) = M2.C2.Delta(T2)
= M1×450(42-28)
= 200×2100(45-42)
= M1×450(14)
= 1.260.000/6300= 200 G
