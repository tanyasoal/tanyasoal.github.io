Air dengan massa 50 gram dan suhua 30 derajat C dicampur dengan air yang massanya 100 gram dan suhu 90 derajat C hitunglah suhu akhirnya

Jawaban

Dik : m1 = 50 gram
T1 = 30 C
m2 = 100 gr
mT2 = 90 C
Dit : Suhu akhir campuran (TC) ?
Jawab :
TC = (m1T1 + m2T2) / (m1+m2)
= (50 x 30 + 100 x 90)/(50 + 100)
= (1500 + 9000)/150
= 10500/150
= 70 C

Jadi, suhu akhir campuran = 70 C

batang kuningan pada suhu 25°c memiliki panjang 100cm dan pada suhu 160°c memiliki panjang 100,2526 cm koefisien muai panjang kuningan adalah

Jawaban

◾ Materi : Kalor, Suhu dan Pemuaian
◾ Sub materi : Pemuaian
◾ Mapel : Fisika
◾ Keyword : Pemuaian Panjang

ㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡ

Diketahui :
ΔT = 160-25 = 135°c
Lo = 100 cm
L = 100,2526 cm

Ditanya :
α = ?

Penyelesaian :
L = Lo ( 1 + αΔT )
100,2526 = 100 ( 1 + α.135)
1,002526 = 1 + α.135
135α = 0,002526
α = 0,002526/135
α = 0,000018711 /°c
α = 1,8 x 10^-5 /°c

∴ Jadi, koefisien muai panjang kuningan tsb adalah 1,8 x 10^-5 /°c

Sebuah benda dengan massa 10 kg ingin dileburkan dan dalam kalor jenisnya 4200j/kg menggunakans suhu 50 menjadi 80. Hitunglah kalor yang diperlukan dan suhu terakhir

Jawaban

◾ Materi : Kalor, Suhu dan Pemuaian
◾ Sub materi : Kalor
◾ Mapel : Fisika
◾ Keyword : Kalor

ㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡ

Diketahui :
m = 10 kg
c = 4200 J/kg°c
ΔT = 80 - 50 = 30°c

Ditanya :
Q = ?

Penyelesaian :
Q = mcΔT
Q = (10)(4200)(30)
Q = 1.260.000 J
Q = 1.260 kJ

∴ Jadi, kalor yg diperlukan adalah 1.260.000 J atau 1.260 kJ
