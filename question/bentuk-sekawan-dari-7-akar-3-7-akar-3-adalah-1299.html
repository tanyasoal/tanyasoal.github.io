= 2x (pxl + pxl + lxt)
= 2x (14x8 + 14x5 + 8x5)
= 2x (112 + 70 + 40)
= 2 x 222
= 444 cm²

112 = p x l
112 = 14 x l
L = 112/14
L = 8cm
