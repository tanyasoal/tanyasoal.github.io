Arjuna menabung pada sebuah bank yang memberikan suku bunga 9% pertahun. Jika besar tabungan Bima setelah 30 bulan Rp 2.205.000,00, berapakah tabungan awal Arjuna…

Jawaban

Misal saldo awal = a

suku bunga 30 bulan = 30/12 x 9%
                                   = 5/2 x 9%
                                   = 22,5 %

saldo akhir = saldo awal + 22.5% x saldo awal
2.205.000 = a + 22,5% a
2.205.000 = 122,5 % a
a = 2.205.000 : 122,5 / 100
a = 2.205.000 x 100 / 122,5
a = 1.800.000

tabungan awal Arjuna adalah Rp. 1.800.000
→ Mapel : Math
→ Materi : Tabungan Awal
→ Bab : Aritmatika Sosial
————————————————

Persentase bung,
= 30/12 x 9%
= 22,5%

Tabungan akhir
= 100% + 22,5%
= 122,5%

Tabungan awal
= 100/122,5 x 2.205.000
= Rp 1.800.000,00

Balok dengan panjang 9 cm,lebar12 cm,dan diagonal ruang 17 cm.berapa volumenya

Jawaban

Diagonal sisi
= √(12² + 9²)
= √(144 + 81)
= √225
= 15 cm

Tinggi
= √(diagonal ruang² - diagonal sisi²)
= √(17² - 15²)
= √(289 - 225)
= √64
= 8 cm

Volume balok
= p x l x t
= 9 cm x 12 cm x 8 cm
= 864 cm³
