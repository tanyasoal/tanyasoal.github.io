Jawaban

Pembahasan 

Diminta untuk menentukan persamaan direktris elips dengan persamaan 
4x² + 25y² - 24x - 100y + 36 = 0

Kita olah persamaan elips bentuk implisit di atas menjadi bentuk eksplisit.

⇔ 4x² - 24x + 25y² - 100y + 36 = 0

⇔ 4(x² - 6x) + 25(y² - 4y) + 36 = 0

⇔ 4(x² - 6x + 9 - 9) + 25(y² - 4y + 4 - 4) + 36 = 0

⇔ 4((x - 3)² - 9) + 25((y - 2)² - 4) + 36 = 0

⇔ 4(x - 3)² - 36 + 25(y - 2)² - 100 + 36 = 0

⇔ 4(x - 3)² + 25(y - 2)² = 100; kedua ruas dibagi 100

∴

, dengan a > b, sehingga bentuknya merupakan elips horisontal atau sumbu utama elips sejajar sumbu x.

⇒ a² = 25 → a = 5

Rumus suku ke-n dari barisan 3,8,13,18,... adalah... a.Un=n-2
b.Un=2n-2
c.Un=3n+1
d.Un=4n-3
e.Un=5n-2

tolong dibantu kakak2 disertai penyelesaianya untuk remedial soalnya terimakasih

Jawaban

Beda = 5
uji tiap pilahan
a. Un = n - 2
U1 = 1 - 2 = - 1 ( tdak terbukti pada suku pertama
b. Un = 2n - 2
U1 = 2.1 - 2 = 0( tdak terbukti)
c. Un = 3n + 1
U1 = 3.1 + 1 = 4 (tidak terbukti)
d. Un = 4n - 3
U1 = 4.1 - 3 =1 ( tidak terbukti)
e. Un = 5n - 2
U1 = 5.1 - 2 = 3 (terbukti)
U2 = 5.2 - 2= 8 (terbukti)
U3 = 5 .3 - 2= 13 ( terbukti)
Jadi, rumus suku ke-n dari barisan tersebut adalah Un = 5n - 2
