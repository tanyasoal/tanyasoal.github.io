Menarik kertas dengan cepat

daya setrika di bandingkan daya kipas angin 5:2.Kedua alat digunakan selama 4 jam per hari.Dalam 1 bulan (30hari)kedua alat memerlukan energi listrik 58,8 kwh.Daya setrika dan daya kipas angin masing-masing adalah.....

Jawaban

Diket :
P1 : P2 = 5 : 2
t = 4 jam / hr
W = 58,8 kwh = 58.800 wh/ bln ( 30 hr )

Dit : Daya alat masing masing .....?


Jwb :

# Total daya kedua alat

P = W / t . 30
P = 58.800 / 4 . 30
P = 58.800 / 120
P = 490 watt

# Daya setrika

P1 = (5 / (5+2 ) . P
P1 = (5 / 7 ) . 490
P1 = 0,714 . 490
P1 = 350 watt


# Daya kipas angin

P2 = (2 / ( 5+ 2 ) . P
P2 = ( 2 / 7 ) . 490
P2 = 0,285 . 490
P2 = 140 watt


Jadi daya setrika 350 w dan kipas angin
140 w

ΔT = 10°c

Q = ?

Penyelesaian :

Sebuah Transformator memiliki perbandingan lilitan primer dan lilitan sekunder 3:5 di hubungkan dengan sumber tegangan ac dengan tegangan 220 volt dan kuat arus 4 apmere.tentukan: efisiensi trafo

Jawaban

Diket :
Np : Ns = 3 : 5
Vp = 220 v
Ip = 4 A


Dit : Efisiensi trafo.......?


Jwb :

# Tegangan sekunder

Vs = ( Ns . Vp ) / Np
Vs = ( 5 . 220 ) / 3
Vs = 1100 / 3
Vs = 367 volt

# Arus sekunder

Is = ( Vp . Ip ) / Vs
Is = ( 220 . 4 ) /367
Is = 880 / 367
Is = 2,4 A

# Efisiensi

ef = ( Vs . Is / Vp . Ip ) . 100%
ef = ( 367 . 2,4 / 220 . 4) . 100%
ef =( 880,8 / 880) .100%
ef = 100%

Jadi efisiensi trafo tsb 100%
