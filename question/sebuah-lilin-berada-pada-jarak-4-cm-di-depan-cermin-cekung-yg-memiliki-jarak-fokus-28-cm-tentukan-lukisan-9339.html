jika perbandingan jumlah lilitan primer dan sekunder pada sebuah trafo adalah 1 : 9, jika tegangan primernya adalah 100 volt dan dayanya 750 watt.. berapa besar kuat arus sekundernya?

Jawaban

Diket :

Np : Ns = 1 : 9
Vp = 100 v
Pp = 750 w


Dit : Arus sekunder.......?


Jwb :

# Tegangan sekunder

Vs = ( Ns . Vp ) / Np
Vs = ( 9 . 100 ) / 1
Vs = 900 volt

# Arus primer

Ip = Pp / Vp
Ip = 750 / 100
Ip = 7,5 A


# Arus sekunder

Is = ( Vp . Ip ) / Vs
Is = ( 100 . 7,5 ) / 900
Is = 750 / 900
Is = 0,83 A


Jadi arus sekundernya 0,83 A
