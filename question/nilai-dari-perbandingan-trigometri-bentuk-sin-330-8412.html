Diketahui f(4x+5)=8x-2 dan g(x+7)=3x+1.Tentukan f(x)dan g(x)

Jawaban

F(4x + 5) = 8x - 2
g(x + 7) = 3x + 1

f(x) dan g(x) = ... ?

*misal
4x + 5 = a
4x = a - 5
x = (a - 5)/4

f(4x + 5) = 8x - 2
f(a) = 8 [(a - 5)/4] - 2
f(a) = 2 (a - 5) - 2
f(a) = 2a - 10 - 2
f(x) = 2x - 12

*misal
x + 7 = a
x = a - 7

g(x + 7) = 3x + 1
g(a) = 3 (a - 7) + 1
g(a) = 3a - 21 + 1
g(x) = 3x - 20

Hasil dari (4a^2b^5)^3/(6a^4n^3)^2 adalah

Jawaban

(4a^2b^5)^3/(6a^4b^3)^2
= (4^3.(a^2)^3.(b^5)^3)/(6^2.(a^4)^2.(b^3)^2)
= (64.a^6.b^15)/(36.a^8.b^6)
= 16/9.a^(6 - 8).b^(15 - 6)
=16/9.a^-2.b^9
= (16.b^9)/a^2
