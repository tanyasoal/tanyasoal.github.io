20 buah pulpen harganya rp35.000. Supaya dapet untung 30%, maka harga jual pulpen per batangnya adalah

Jawaban

20 Pulpen 35000 ( 100% )
Supaya Untung 30% Jadi ,

100% + 30% = 130%

130% per 100 dikali 35000
>> 45.500

45.500 dibagi 20 Pulpen elu
Jadi harga jual setiap pulpen : 2.275 rebu
Laba = 30%x35,000 = 10,500
harga jual pulpen per batang = (35,000+10,500)/20
                                               = 45,500/20
                                               = 2,275

Grafik fungsi f(x) =x³-3x²-9x+15-5 naik pada interval ...

Jawaban

Pembahasan :

f(x) = x³ - 3x² - 9x + 15 - 5
naik pada interval ...

kita gunakan turunan
f(x) = x³ - 3x² - 9x + 15 - 5
f(x) = x³ - 3x² - 9x + 10
f'(x) = 3x² - 6x - 9 
naik pada saat f'(x) > 0

f'(x) = 3x² - 6x - 9 kita kecilkan dengan membagi 3
x² - 2x - 3 > 0
(x - 3)(x + 1) > 0
kita dapati dua bilangan -1 dan 3
kita cek nol fungsi atau x = 0

3x² - 6x - 9 > 0
3(0)² - 6(0) - 9 > 0
0 - 0 - 9 > 0
-9 > 0 salah
maka interval naik pada
x < -1 atau x > 3


jadi f(x) = x³ - 3x² - 9x + 15 - 5 naik pada interval x < -1 atau x > 3
