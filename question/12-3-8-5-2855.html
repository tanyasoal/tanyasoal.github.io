Diketahui panjang jari jari kedua lingkaran adalah 7 cm dan 3 cm. Jika panjang garis singgung persekutuan dalam kedua lingkaran itu 24 cm. Hitunglah jarak titik pusat kedua lingkaran tersebut

Jawaban

Mapel          : Matematika
Kategori      : Garis Singgung Lingkaran
Kata Kunci  : Lingkaran, Garis singgung, Pusat, Jari - jari
Kelas           : 8
Kode            : 8.2.7 - [ Bab - Garis Singgung Lingkaran ]

≡ Jari-jari lingkaran pertama - R₁ = 7 cm
≡ Jari-jari lingkaran kedua    - R₂ = 3 cm
≡ Gspd                                         = 24 cm

Jarak Titik pusat kedua lingkaran tersebut adalah
     = √[Gspd² + (R₁ + R₂)²]
     = √[24² + (7 + 3)²]
     = √[576 + 10²]
     = √[576 + 100]
     = √[676]
     = 26 cm

sebuah mobil dibeli dengan harga 80.000 setiap tahun nilai jualnya menjadi 3/4 dari jarga tahun sebelumnya berapa nilai jualnya setelah dipakai 3 tahun

Jawaban

Harga = Rp.80.000.000
Harga jual per tahun = 3/4 × Rp.80.000.000
= Rp.60.000.000

Selisih = Rp.80.000.000 - Rp.60.000.000
= Rp.20.000.000

Harga jual setelah 3 tahun
= Rp.80.000.000 - (Rp.20.000.000 × 3)
= Rp.80.000.000 - Rp.60.000.000
= Rp.20.000.000
