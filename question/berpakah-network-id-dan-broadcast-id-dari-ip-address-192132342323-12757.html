Jawaban

Kesalahan pada baris pertama dimana:
preprocessor directive #include belum lengkap.
Yang sehsrusnya diisi dengan format: #include <nama-library.h>
Seharusnya diisi dengan <stdio.h> dimana library itu berfungsi untuk input / output suatu data.

Jadi koreksi pada baris pertama:
#include <stdio.h>

Motherboard cpu baru, ram cpu baru, pau cpu baru saat dinyalakan mau nyala cuma tidak mau tampil layar. Bagaimana solusinya?

Permasalahan:
Monitor yang tidak tampil disebabkan karena adanya masalah pada RAM ataupun kartu grafis (VGA).

Solusi:
Mengecek apakah RAM sudah terpasang dengan benar, jikalau perlu lepas dan bersihkan lempengan kuning pada RAM agar tidak kotor, lalu pasang kembali. Dan cek pada kartu grafis (VGA) sudah terpasang dengan benar.
