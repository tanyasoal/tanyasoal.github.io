Sebuah garis melalui titik a (4p, 5) dan b (2p, 8), jika gradien garis itu = 3 , maka nilai p...?

Jawaban

Kelas 8 Matematika
Bab Persamaan Garis

m = 3
(y2 - y1)/(x2 - x1) = 3
(8 - 5) / (2p - 4p) = 3
3 = 3 (-2p)
p = -1/2

Seseorang meminjam uang dikoperasi sebesar Rp. 6.000.000 dan diangsur selama 12 bulan dengan bunga 1,5% per bulan. besar angsuran tiap bulan adalah... a. Rp.640.000
b. Rp.507.500
c. Rp.590.000
d. Rp.650.000

Tolong jawab dengan caranya juga ya..

Jawaban

→ Mapel : Math
→ Materi : Angsuran
→ Bab : Aritmartika Sosial
————————————————

Bunga
= 12/1 x 1,5/100 x 6.000.000
= 18/100 x 6.000.0000
= 1.080.000

Angsuran
= (6.000.000 + 1.080.000) ÷ 12
= 7.080.000 ÷ 12
= Rp 590.000,00
Opsi → C
Angsuran per bulan (belum termasuk bunga)
= 6.000.000 : 12
= 500.000
Bunga per bulan = 1,5% × 6.000.000
= 1,5/100 × 6.000.000
= 90.000

Angsuran per bulan (sudah termasuk bunga)
= Rp.500.000 + Rp.90.000
= Rp.590.000

Jawabannya : c. Rp.590.000

jika sudut P dan sudut Q saling berpelurus dengan besar sudut P = 2×° dan sudut Q adalah 5×+ 26° maka besar sudut Q adalah

Jawaban

Saling berpelurus, maka kedua sudut tersebut berjumlah 180°

2x + 5x + 26 = 180
7x + 26 = 180
7x = 180 - 26
7x = 154
x = 154 / 7
x = 22°

Q = 5x + 26
= 5(22) + 26
= 110 + 26
= 136°
