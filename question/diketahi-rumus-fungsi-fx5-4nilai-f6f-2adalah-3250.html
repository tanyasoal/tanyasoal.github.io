Jawaban

Jawab :

d^2 = (AB)^2 - (R - r)^2
15^2 = (AB)^2 - (10 - 2)^2
225 = (AB)^2 - 64
(AB)^2 = 225 + 64
(AB)^2 = 289
AB = √289
AB = 17 cm
