Perangkat Input

• Penjelasan: 

Perangkat Input merupaka perangkan yang berfungsi untuk memasukkan data atau perintah dari pengguna ke komputer.

Anda dapat membatalkan pemasukan data dengan menekan tombol A. Esc
B. Enter
C. Tab
D. Caps lock
E. Alt
