Diketahui panjang jari jari kedua lingkaran adalah 7 cm dan 3 cm. Jika panjang garis singgung persekutuan dalam kedua lingkaran itu 24 cm. Hitunglah jarak titik pusat kedua lingkaran tersebut

Jawaban

Mapel          : Matematika
Kategori      : Garis Singgung Lingkaran
Kata Kunci  : Lingkaran, Garis singgung, Pusat, Jari - jari
Kelas           : 8
Kode            : 8.2.7 - [ Bab - Garis Singgung Lingkaran ]

≡ Jari-jari lingkaran pertama - R₁ = 7 cm
≡ Jari-jari lingkaran kedua    - R₂ = 3 cm
≡ Gspd                                         = 24 cm

Jarak Titik pusat kedua lingkaran tersebut adalah
     = √[Gspd² + (R₁ + R₂)²]
     = √[24² + (7 + 3)²]
     = √[576 + 10²]
     = √[576 + 100]
     = √[676]
     = 26 cm

Sebuah tangga yang panjangnya 13meter disandarkan keujung tiang listrik jika jarak 5meter maka tinggi tiang listrik adalah?

Jawaban

Mapel          : Matematika
Kategori      : Teorema Pythagoras
Kata Kunci  : Teorema Pythagoras, Segitiga
Kelas           : 8
Kode            : 8.2.9 - [ Bab - Teorema Pythagoras ]


≡ Panjang Tangga                                = 13 meter
≡ Jarak pangkal tangga ke tiang listrik =  5 meter


Maka, Tinggi Tiang Listrik tersebut dihitung dengan menggunakan teorema pythagoras, dimana :
      b² = c² - a²
      b  = √(c² - a²)
      b  = √(13² - 5²)
      b  = √(169 - 25)
      b  = √(144)
      b  = 12 cm

Jadi, Tinggi tiang listrik adalah 12 cm
