Jika 1/a + 1/b + 1/c = o dan a+b+c =-8 berapa nilai a² + b² +c

Jawaban

1/a + 1/b + 1/c = 0 dan a+b+c = -8. Nilai a²+b²+c²? (saya anggap c²) 

1. ubah 1/a + 1/b + 1/c = 0 menjadi tidak pecahan dengan dikali abc
    1/a + 1/b + 1/c = 0  (×abc)
     bc + ac + ab = 0

2. cari nilai a²+b²+c²
    (a+b+c)² = (a+b+c)(a+b+c)
                   = a²+b²+c²+ 2ab+ 2bc + 2ac
                   = a²+b²+c² + 2 (ab+bc+ac)
           (-8)² = a²+b²+c² + 2 (0)
              64 = a²+b²+c²
     a²+b²+c² = 64
    
jadi, nilai a²+b²+c² adalah 64

3. Suhu udara di puncak gunung pada pukul 03.00 adalah -10° C. Setelah matahari terbit, energi matahari menaikkan suhu udara di puncak gunung tersebut. Jika setiap jam suhu udara di puncak guning naik 5° C, suhu udara pada pukul 15.00 menjadi .... C° PAKAI CARANYA YAH KALO BISA ADA PENJELASAN

Jawaban

Pukul 03.00 = -10°c
setiap 1 jam naik 5°c
Pada pujul 15.00 suhunya adalah...

Selisih jam : 15.00-03.00 = 12jam
Suhu jam 15.00 = -10°c + (5°c x 12jam) = -10°c + 60°c = 50°c

Jadi suhu pada jam 14.00 adalah 50°c
