Jawaban

panjang tembok = 3,5m

panjang tembok diatas tangga

Suku ke 3 dan suku 6 suatu deret aritmetika berturut turut adalah 40 dan 60. Jumlah sepuluh suku pertama deret tersebut adalah

Jawaban

Diketahui :
U3 = 40
U6 = 60

Ditanya :
S10 = ...?

Jawab :
Un = a + b(n-1)
U3 = a + b(3-1)
40 = a + 2b ...(1)
U6 = a + b(6-1)
60 = a + 5b ... (2)

=> subsitusikan ke persamaan (1) dan (2)
40 = a + 2b
60 = a + 5b
_________-
-20 = -3b
b = -20/-3
b = 6,6

=> eliminasikan ke persamaan (1) atau (2)
60 = a + 5b
60 = a + 5(6,6)
60 = a + 33
a = 60 - 33
a = 27

=> rumus Un :
Un = a + b(n-1)
Un = 27 + 6,6(n-1)
Un = 27 + 6,6n - 6,6
Un = 20,4 + 6,6n
U10 = 20,4 + 6,6(10)
U10 = 20,4 + 66
U10 = 86,4 => dibulatkan menjadi 86

=> Jumlah sepuluh suku pertama deret tersebut adalah :
Sn = n/2 (a + Un)
S10 = 10/2 (27+ U10)
S10 = 10/2 (27 + 86)
S10 = 5(113)
S10 = 565
