sebuah mobil mogok didiorong oleh tiga orang dengan gaya masing masing 300 N, 400 N dan 500 N selama 4 sekon. massa mobil tersebut 1 ton sedangkan koefisien gesekan antara ban dan jalan μs = 0,4 dan μk= 0,2. jika mobil tersebut berada di jalan yang datar, hitunglah usaha yang dilakukan oleh ketiga orang tersebut

Jawaban

◾ Materi : Usaha dan Energi
◾ Sub materi : Usaha
◾ Mapel : Fisika
◾ Keyword : Usaha, Kerja

ㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡ

Diketahui :
F1 = 300N
F2 = 400 N
F3 = 500 N
t = 4 sekon
μs = 0,4
μk = 0,2
m = 1 ton = 1000 kg
g = 10 m/s²

Ditanya :
W = ?

Penyelesaian :
*Cari gaya gesek maksimum
fsmaks = μs. N
fsmaks = 0,4 (1000.10)
fsmaks = 4000 N

*Cari resultan gaya
Fr = F1+F2+F3
Fr = 300+400+500
Fr = 1200 N

Karena Fr < fsmaks maka mobil tidak berpindah. s = 0

W = F. s
W = 0

∴ Jadi, usaha yg dilakukan oleh ketiga gaya tsb adalah 0

ㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡ

Diketahui :
ΔT = 160-25 = 135°c
Lo = 100 cm
L = 100,2526 cm

Ditanya :
α = ?

Penyelesaian :
L = Lo ( 1 + αΔT )
100,2526 = 100 ( 1 + α.135)
1,002526 = 1 + α.135
135α = 0,002526
α = 0,002526/135
α = 0,000018711 /°c
α = 1,8 x 10^-5 /°c

∴ Jadi, koefisien muai panjang kuningan tsb adalah 1,8 x 10^-5 /°c
