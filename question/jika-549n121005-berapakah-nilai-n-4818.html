Pakai cara

Jawaban

→ Mapel : Math
→ Materi : Luas
→ Bab : Perbandingan
————————————————

Selisih → 3 - 1 → 2

p = 81 cm

l = 27 cm

Luas = p x l
Luas = 81 cm x 27 cm
Luas = 2187 cm²
