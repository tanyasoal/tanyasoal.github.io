Diketahui panjang rusuk kubus = 3 cm. tentukan:
a) panjang seluruh rusuk kubus b) panjang diagonal bidang c) panjang diagonal ruang

Jawaban

Mapel : Matematika
Materi : Balok dan Kubus
Kelas : 8

⇔ Panjang rusuk (s) = 3 cm

a] Panjang seluruh rusuk kubus

x = 12s
x = 12(3)
x = 36 cm


b] Panjang diagonal bidang

= s√2
= 3√2 cm


c] Panjang Diagonal Ruang

= s√3
= 3√3 cm
