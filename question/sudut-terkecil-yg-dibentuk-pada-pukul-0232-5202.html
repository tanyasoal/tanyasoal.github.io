di tempat parkir sebuah pertokoan terdapat 75 kendaran yang terdiri dari mobil dan motor.banyak roda seluruh nya ada 210.jika tarif parkir mobil Rp.5.000 dan motor Rp 2.000 maka pendapatan uang parkir saat itu adalah

Jawaban

4mobil = a
motor = b

roda mobil = 4 = 4a
roda motor = 2 = 2b

mencari banyak motor dan mobil

cara eliminasi
a + b = 75 ~ x 4
4a + 2b = 210 ~ x 1

menjadi
4a + 4b = 300
4a + 2b = 210
___________-
2b = 70
b = 70/2 = 35

a + b = 75 ~ x 2
4a + 2b = 210 ~ x 1

menjadi
2a + 2b = 150
4a + 2b = 210
___________-
-2a = -60
a = -60/-2 = 30

pendapatan = (30 x 5000) + (35 x 2000)
= 150.000 + 70.000
= 220.000

Jadi pendapatan uang parkir adalah Rp220.000,00.

reni menabungdi bank sebesar RP4.000.000,00dengan suku bunga 15%setahun. jika ia ingin menginginkan tabungannya menjadi Rp4.400.000,00, lama ia harus menabung adalah..

Jawaban

Mapel : Matematika
Materi : Aritmatika Sosial
Kelas  : 7

Tabungan (M)   = Rp. 4.000.000
Suku bunga (p) = 15 %
Besar Bunga     = 4.400.000 - 4.000.000
                          = Rp. 400.000

Lama Menabung :

n = (Besar Bunga x 12 bulan) / ( p x M)
n = (400.000 x 12) / (15% x 4.000.000)
n = (4.800.000) / 600.000
n = 8 bulan


Jadi, Lama menabung ada 8 bulan
