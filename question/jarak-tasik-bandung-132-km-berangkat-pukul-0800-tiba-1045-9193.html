Jarak tasik bandung 132 km berangkat pukul 08.00 tiba 10.45

Jawaban

S = 132 km
t = 10.45 - 08.00
  = 2 jam 45 menit
  = 2 3/4 jam

v = s/t
   = 132 / 2 3/4
   = 132 / 11/4
   = 132 x 4/11
   = 48 km/jam

Kecepatan

Jawab :
Sudut a = sudut c, jadi sudut c = 120.

a + d = 180
120 + d = 180
d = 180 - 120
d = 60, jadi sudut d = 60.

sudut d = sudut b, maka sudut b = 60.
