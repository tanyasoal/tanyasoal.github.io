Dari ketiga bilangan diketahui jumlah dua bilangan pertama adalah 7, jumlah bilangan dua bilangan terakhir adalah 5, sedangkan jumlah bilangan pertama dan terakhir adalah 4. Jumlah ketiga bilangan adalah.. A.11
B.8
C.10
D.9
E.12

Jawaban

Anggap saja bilangan pertama itu x, kedua y, ketiga z

1) x+y= 7
2)y+z= 5
3)x+z= 4

eleminasikan (1) terhadap (3)
x+y=7
x+z=4
---------- (-)
y-z= 3 (4)

eleminasikan (2) dan (4)
y+z=5
y-z=3
---------- (-)
2z=2
z =1

substitusikan z=1 ke (2)
y+z=5
y+1=5
y=4

substitusikan z=1 ke (3)
x+z=4
x+1=4
x=3

maka x+y+z=3+4+1= 8(B)

Harga 3kg mangga dan 2kg jeruk Rp116.000,00, harga 4kg mangga dan 5kg jeruk Rp199.000,00. Jadi berapa harga 1kg jeruk dan 2kg mangga?

Jawaban

Jawaban ⇒ Rp 64.000,00

❏ Misalkan
mangga » x
jeruk » y

sehingga
3 kg mangga + 2 kg jeruk = 116.000
=> 3x + 2y = 116.000

4 kg mangga + 5 kg jeruk = 199.000
=> 4x + 5y = 199.000

❏ Eliminasi variabel x
4x + 5y = 199.000 ( ×3 )
3x + 2y = 116.000 ( ×4 )

12x + 15y = 597.000
12x + 8y = 464.000 --
-----------------------------
7y = 133.000
y = 133.000 ÷ 7 = 19.000

❏ Substitusikan y = 19.000 ke

3x + 2y = 116.000
3x + 2 (19.000) = 116.000
3x + 38.000 = 116.000
3x = 116.000- 38.000
3x = 78.000
x = 78.000 ÷ 3
x = 26.000

❏ Harga 1 kg jeruk dan 2 kg mangga
= 1x + 2y
= 1 (26.000) + 2 (19.000)
= 26.000 + 38.000
= 64.000

Jadi, harganya adalah Rp 64.000,00
