Jika a= (2x-3y) dan b=(3x+5y) maka hasil dari a^2+b^2 adalah...

Jawaban

A = (2x-3y) dan b = (3x+5y)
Maka,
a² + b² = (2x-3y)² + (3x+5y)²
           = (2x-3y)(2x-3y) + (3x+5y)(3x+5y)
           = 4x² - 6xy - 6xy + 9y² + 9x² + 15xy + 15xy + 25y²
           = (4+9)x² +(30-12)xy + (9+25)y² 
           = 13x² + 18xy + 34y²

jadi, hasil dari a² + b²  adalah 13x² + 18xy + 34y²

Jawaban

Volume = 3 m³
             
waktu = 1/4 jam

Debit = volume / waktu
          = 3 / 1/4
          = 3 x 4
          = 12 m³/jam
Volume (v) = 3 m³
waktu (t) = 1/4 jam
debit = .....?
debit = v/t
debit = 3 m³ / ¼ jam
