Kerangka layang-layang dengan diagonal 21cm dan 40cm akan ditutup kertas.tersedia kertas berukuran 63cm x 80cm dengan harga Rp3000/lembar tentukan harga kertas untuk tiap layang layang!

Jawaban

Diketahui : k. layang layang = 21 × 40 cm
kertas sebesar 63 × 80 cm
harga 3000/ lembar
ditanya : harga kertas untuk tiap layang layang
jawab:
kerangka layang layang = 21 × 40 = 840 cm2
ditutup kertas = 63 × 80 = 5040 cm2

5040 cm2 : 840 cm2 = 6 layang layang
harga = 3000 , maka harga kertas untuk tiap layang-layang adalah
3000 : 6 = 500 rupiah

sebuah balok berukuran panjang 14 cm dan tinggi 5 cm jika luas alas balok tersebut 112 cm pangkat 2 maka luas permukaan nya adalah

Jawaban

Lebar = Luas : panjang
= 112 : 14 = 8 cm
Luas permukaan balok..
= 2x (pxl + pxl + lxt)
= 2x (14x8 + 14x5 + 8x5)
= 2x (112 + 70 + 40)
= 2 x 222
= 444 cm²
Dik p = 14, t = 5cm, luas alas(p.l) = 112 cm^2, lp....?

Luas permukaan = 2(pl+pt+lt)

112 = p x l
112 = 14 x l
L = 112/14
L = 8cm

Lp = 2(pl + pt + lt) = 2(14.8 + 14.5 + 8.5) = 2(112+70+40) = 444 cm^2

Aritematika sosial keuntungan dan kerugian pada hari ini matahari Energi akan mengadakan discount untuk setiap pembelian kemeja mendapatkan diskon 25% untuk setiap pembelian kaos atau sandal mendapat diskon 50% Sedangkan untuk pembelian sepatu mendapat diskon 20% + 10% jika keluarga pak byoman memborong 4 kemeja dengan harga rp325.000 perpotong ,8potong kaos seharga Rp125.000 perpotong,5 pasang sandal seharga 280.000 pasang serta 3 pasang sepatu seharga rp659.000 per pasang maka berapa rupiah uang pak byoman setorkan

Berapa besar keuntungan/kerugian yang diderita pak berapa persentase keuntungan dan kerugian

Jawaban

Kemeja
harga beli = 4 x Rp. 325.000
                 = Rp. 1.300.000

diskon = 25% x Rp. 1.300.000
            = Rp. 325.000

yang dibayarkan = RP. 1.300.000 - Rp. 325.000
                            = Rp. 975.000

Kaos
harga beli = 8 x Rp. 125.000
                 = Rp. 1.000.000

diskon = 50% x Rp. 1.000.000
            = Rp. 500.000

yang dibayarkan = Rp. 1.000.000 - Rp. 500.000
                            = Rp. 500.000

Sandal
Harga beli = 5 x Rp. 280.000
                  = Rp. 1.400.000

Diskon = 50% x Rp. 1.400.000
            = Rp. 700.000

Yang dibayarkan = Rp. 1.400.000 - Rp. 700.000
                            = Rp. 700.000

Sepatu
Harga beli = 3 x Rp. 659.000
                  = Rp. 1.977.000

Diskon 1 = 20% x Rp. 1.977.000
               = Rp. 395.400

Diskon 2 = 10% x (Rp. 1.977.000 - Rp. 395.400)
               = 10% x Rp. 1.581.600
               = Rp. 158.160

yang dibayarkan = Rp. 1.581.600 - Rp. 158.160
                            = Rp. 1.423.440

Total pembelian = Rp. 975.000 + Rp. 500.000 + Rp. 700.000 + Rp, 1.423.440
                          = Rp. 3.598.440
