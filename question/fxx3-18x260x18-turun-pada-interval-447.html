Rata rata badan 15 siswa 40 kg. Jika masuk beberapa siswa baru rata-rata berat badannya menjadi 40,5 kg. Jika rata-rata berat badan siswa baru 42 kg hitung berapa orang banyak siswa yg baru

Jawaban

Caranya pakai rata - rata gabungan (x)
x = rata - rata gabungan
xa = rata - rata awal
na = banyak siswa awal
xb = rata - rata setelah ditambahkan
nb = banyak siswa yang ditambahkan
x = (xa. na + xb. nb) /(na + nb)
40,5 = (40.15 + 42.nb)/(15 + nb) ---> kali silang
40,5(15 + nb) = 40.15 + 42nb
607,5 + 40,5nb = 600 + 42nb
607,5 - 600 = 42nb - 40,5nb
7,5 = 1,5nb
7,5/1.5 = nb
5 = nb
jadi banyak siswa baru adalah 5 siswa

Kordinat titik balik minimum dari grafik fungsi kuadrat f(x)=x²-2x-3 adalah

Jawaban

BAB Fungsi Kuadrat
Matematika SMA

f(x) = x²-2x-3
titik balik terjadi jika f'(x) = 0
f'(x) = 2x-2 = 0
2x = 2
x = 1
f(1) = 1²-2.1-3
= 1-2-3
= -4

Jadi koordinat titik balik dari grafik fungsi kuadrat f(x) = x²-2x-3 adalah (1,-4)
