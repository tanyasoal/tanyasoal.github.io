efesiensi sebuah trafo 90% apabila di pasang pada tegangan 220 volt dan arus 2 A ,hitung besar arus sekunder yg di hasilkan bila trafo tersebut menghasilkan tegangan 20 volt

Jawaban

Efisiensi trafo = daya (P) sekunder / daya primer * 100%
90%               = P sekunder / (V/I) * 100%
90%               = P sekunder / (220/2) * 100%
90%               = P sekunder / 110 * 100%       
P sekunder    = 90% * 110 / 100%
P sekunder    = 99 watt

Kuat Arus      = P sekunder / V sekunder
                      = 99 watt / 20 Volt
                      = 4,95 Ampere

(V berbanding terbalik dengan I)

Sebuah peluru bermassa 0,02 kg ditembakkan dengan kelajuan 200m/s. energi kinetik peluru adalah

Jawaban

M = 0,02 kg
v = 200 m/s
Ek?

penyelesaian:
Ek = m . v² / 2

Ek = 0,02 . 200² / 2

Ek = 40000 . 0,01

Ek = 400 Joule

jadi energi kinetiknya adalah 400 Joule

benda masa 3kg kalori jenisnya 460j/kg°c dipanaskan dari suhu 10° menjadi 60°c tentukan banyaknya kalori yg di perlukan

Jawaban

Mapel : IPA Fisika 
Kelas : 8 SMP/VIII SMP
Bab : Kalor dan Suhu 
Tentang : Kalor Jenis 

Pembahasan Soal :
Benda masa 3 kg kalori jenisnya 460 J/kg°C dipanaskan dari suhu 10°C menjadi 60°C tentukan banyaknya kalori yg di perlukan...

Pembahasan Jawaban :
DiKetahui :
m(benda) = 3 kg 
C = 460 J/kg°C 
T1 = 10°C 
T2 = 60°C 

DiTanya :
Q =....?

DiJawab :
Rumus mencari Q
Yaitu :
Q = m(benda) x C x ΔT 
Q = m(benda) x C x ( T2 - T1 )
Q = 3 kg x 460 J/kg°C x ( 60°C - 10°C )
Q = 3 kg x 460 J/kg°C x 50°C 
Q = 1.380 x 50 
Q = 69.000 Joule 
Q = 69 kJ.
Jadi, Kalor yang dibutuhkan adalah 69.000 J/ 69 kJ.

Keterangan :
m(benda) = Massa benda ( kg )
T1 = Suhu pertama ( °C )
T2 = Suhu kedua ( °C )
ΔT = Selisih suhu ( °C )
C = Kalor Jenis benda ( J/kg°C )
Q = Banyak kalor ( Joule )
