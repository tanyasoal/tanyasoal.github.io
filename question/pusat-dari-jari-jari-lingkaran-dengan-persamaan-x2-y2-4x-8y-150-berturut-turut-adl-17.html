Pusat dari jari jari lingkaran dengan persamaan x2 +y2 -4x -8y +15=0 berturut turut adl

Jawaban

Pembahasan :

diberikan persamaan lingkaran
x² + y² - 4x - 8y + 15 = 0
ditanya pusat dan jari-jari lingkaran

persamaan umu lingkaran
(x - a)² + (y - b)² = r²
dengan pusat (a, b)

x² + y² - 4x - 8y + 15 = 0 → kita buat menjadi bentuk persamaan umum lingkaran
x² - 4x + y² - 8y = -15
x² - 4x + (1/2 x -4)² + y² - 8y + (1/2 x -8)² = -15 + (1/2 x -4)² + (1/2 x -8)²
x² - 4x + 4 + y² - 8y + 16 = -15 + 4 + 16
(x - 2)² + (y - 4)² = 5

pusat lingkaran = (2, 4)
jari-jari lingkaran = √5

Di ketahui rumus fungsi f(x)=4x-12.jika f (a)=8 dan f(-2)=b. tentukan nilai a+b !

Jawaban

Diketahui rumus : f (x) = 4x - 12

f (a) = 4a -12 = 8
4a = 20
a = 5

f (-2) = 4 . (-2) - 12= b
= -8 - 12 = b
= -20 = b

Jadi, hasil dari a + b = 5 + (-20)= -15

17 = (7p + 1) / 5

7p + 1 = 85
7p = 85 - 1
7p = 84
p = 84 / 7
