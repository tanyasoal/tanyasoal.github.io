Suatu deret geometri memiliki jumlah suku pertama dan kedua adalah 45. Jumlah suku ketiga dan keempat adalah 20. Jumlah 5 suku pertama adalah .. beserta caranya :)

Jawaban

U1 + U2 = 44
a + (a+(2-1)b) = 44
2a+b = 44
2a = 44-b

U3+U4 = 20
a+(3-1)b + a+(4-1)b = 20
2a + 5b = 20

44-b + 5b = 20
44 + 4b = 20
4b = 20-44
b = (-24):4
b = (-6)

2a+b = 44
2a-6 = 44
2a = 44+6
a = 50:2
a = 25

Sn = n:2 (2a +(n-1)b)
S5 = 5:2 (2(25) + (5-1)(-6))
S5 = 5:2 (50 + 4(-6))
S5 = 5:2 (50 -24)
S5 = 5:2 (26)
S5 = 65
