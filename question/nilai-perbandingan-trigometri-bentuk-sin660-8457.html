Menentukan persamaan direktriks elips dengan persamaan 4x²+25y²-24x+100y+36=0

Jawaban

Pembahasan 

Diminta untuk menentukan persamaan direktris elips dengan persamaan 
4x² + 25y² - 24x - 100y + 36 = 0

Kita olah persamaan elips bentuk implisit di atas menjadi bentuk eksplisit.

⇔ 4x² - 24x + 25y² - 100y + 36 = 0

⇔ 4(x² - 6x) + 25(y² - 4y) + 36 = 0

⇔ 4(x² - 6x + 9 - 9) + 25(y² - 4y + 4 - 4) + 36 = 0

⇔ 4((x - 3)² - 9) + 25((y - 2)² - 4) + 36 = 0

⇔ 4(x - 3)² - 36 + 25(y - 2)² - 100 + 36 = 0

⇔ 4(x - 3)² + 25(y - 2)² = 100; kedua ruas dibagi 100

∴ 

Merupakan persamaan elips bentuk eksplisit yang mengikuti format
, dengan a > b, sehingga bentuknya merupakan elips horisontal atau sumbu utama elips sejajar sumbu x.

Informasi utama sebagai berikut:
⇒ a² = 25 → a = 5
⇒ b² = 4 → b = 2

Dari segitiga ABC, garis berat ma membagi sudut A menjadi dua bagian, yaitu A1 dan A2. Buktikanlah bahwa sin A1/sin A2 = sin B/sin C !

Jawaban

Misal ma memotong BC di D. Dengan aturan sinus BD/sinA1=ma/sinB ekuivalen sinB/sinA1=ma/BD. Dengan aturan sinus juga DC/sinA2=ma/sinC ekuivalen DC/ma=sinA2/sinC. Kalikan kedua persamaan dan dari fakta BD=DC didapat (sinB/sinA1)x(sinA2/sinC)=1 yang ekuivalen dg soal.

|x - 3| ≥ 6

Jawaban

Bentuk |x - 3| ≥ 6 artinya

a) x - 3 ≥ 6, yaitu
x ≥ 6 + 3
x ≥ 9

atau
b) x - 3 ≤ -6
x ≤ -6 + 3
x ≤ -3

Sehingga himpunan penyelesaiannya
HP = { x | x ≤ -3 atau x ≥ 9, x € R }
