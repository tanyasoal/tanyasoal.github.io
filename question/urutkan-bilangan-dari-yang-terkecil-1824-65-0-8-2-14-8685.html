Perbandingan uang amin dan budi adalah 2:3 jika uang mereka 200.000.00 berapa selisih uang mereka

Jawaban

Uang amin = 2/5 x 200.000
= Rp 80.000

Uang budi = 3/5 x 200.000
= Rp 120.000

selisih uang mereka = 120.000 - 80.000
= Rp 40.000
amin : budi
   2    :    3 ⇒ jumlah uang = 200.000.00
                 ⇒ jumlah perbandingan = 2 + 3 = 5
                 ⇒ selisih perbandingan = 3 + 2 = 1
jadi , selisih perbandingan / jumlah perbandingan  x jumlah uang

= 1 / 5 x 200.000.00 = 4000.000
