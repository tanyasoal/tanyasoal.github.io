D.) 95

Jawaban

K = 2 x ( p + l )
280 = 2 x ( p + 55 )
280 = 2p + 110
2p = 280 - 110
2p = 170
p = 85 cm
Opsi → B
K = 2(p+l)
280 = 2(p+55)
