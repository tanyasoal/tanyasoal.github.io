Jika α = 4/3 π maka cos α -1/√3 sin α = ..

Jawaban

Α = 4/3 π = 4/3 (180°) = 240° (diubah ke bentuk derajat untuk memudahkan)
cos α = -1/2
sin α = -√3/2 

maka:
cos α - 1/√3 sin α = -1/2 - 1/√3 (- √3/2) = -1/2 + 1/2 = 0

(sin x +cos x) (sin x-cos x)

Jawaban

Jawab :

(a + b)(a - b) = a^2 - b^2
Sehingga :
(sin x + cos x)(sin x + cos x)
= sin^2 x - cos^2 x
