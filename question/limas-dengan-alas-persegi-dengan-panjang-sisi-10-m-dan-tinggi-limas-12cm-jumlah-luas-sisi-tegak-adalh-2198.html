Grafik fungsi f(x) =x³-3x²-9x+15-5 naik pada interval ...

Jawaban

Pembahasan :

f(x) = x³ - 3x² - 9x + 15 - 5
naik pada interval ...

kita gunakan turunan
f(x) = x³ - 3x² - 9x + 15 - 5
f(x) = x³ - 3x² - 9x + 10
f'(x) = 3x² - 6x - 9 
naik pada saat f'(x) > 0

f'(x) = 3x² - 6x - 9 kita kecilkan dengan membagi 3
x² - 2x - 3 > 0
(x - 3)(x + 1) > 0
kita dapati dua bilangan -1 dan 3
kita cek nol fungsi atau x = 0

3x² - 6x - 9 > 0
3(0)² - 6(0) - 9 > 0
0 - 0 - 9 > 0
-9 > 0 salah
maka interval naik pada
x < -1 atau x > 3


jadi f(x) = x³ - 3x² - 9x + 15 - 5 naik pada interval x < -1 atau x > 3

(3x+6)+5=2x+30

nilai 2(x-1)
=2x-2
=2(19)-2
=38-3
=36

Bab : Operasi Aljabar
Materi : Penjumlahan dan Pengurangan Bentuk Aljabar

Penyelesaian:
3(x + 2) + 5 = 2(x + 15)
3x + 6 + 5 = 2x + 30
3x + 11 = 2x + 30
3x - 2x = 30 - 11
x = 19

Maka:
2(x - 1)
= 2(19 - 1)
= 2(18)
= 36

Jawabannya adalah 36

Sebuah tangga yang panjangnya 13meter disandarkan keujung tiang listrik jika jarak 5meter maka tinggi tiang listrik adalah?

Jawaban

Mapel          : Matematika
Kategori      : Teorema Pythagoras
Kata Kunci  : Teorema Pythagoras, Segitiga
Kelas           : 8
Kode            : 8.2.9 - [ Bab - Teorema Pythagoras ]


≡ Panjang Tangga                                = 13 meter
≡ Jarak pangkal tangga ke tiang listrik =  5 meter


Maka, Tinggi Tiang Listrik tersebut dihitung dengan menggunakan teorema pythagoras, dimana :
      b² = c² - a²
      b  = √(c² - a²)
      b  = √(13² - 5²)
      b  = √(169 - 25)
      b  = √(144)
      b  = 12 cm

Jadi, Tinggi tiang listrik adalah 12 cm
