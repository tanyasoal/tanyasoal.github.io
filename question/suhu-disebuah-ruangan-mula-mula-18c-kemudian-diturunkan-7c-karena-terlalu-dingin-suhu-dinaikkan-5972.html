Jawaban

Suhu ruangan = - 18 C
Turun = 7 C
Naik = 5 C

Penyelesaian :
= - 18 - 7 = - 11
= - 11 + 5 = 11 -5 = - 6 C

Jadi suhu udara diruangan tersebut adalah 6 C.

hasil m:n adalah ...

Jawaban

Jawabannya '


Mapel : Matematika
Bab : Operasi Bilangan
Sub Bab : Perbandingan

Pembahasan :

m = 16 2/3% = 50%
n = 66 2/3% = 200%

Maka '

m : n
= 50% : 200%
= 50/100 : 200/100
= 1/2 : 2/1
= 1/2 × 1/2
= 1/4 → Jwbnya
