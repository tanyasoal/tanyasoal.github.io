Jika sudutA berpelurus dengan sudutB.dan besar sudutA=1\3B maka besar sudut A adalah

Jawaban

A + B = 180° (berpelurus)
1/3 × B + B = 180°
1/3B + 1B = 180°
1/3B + 3/3B = 180°
4/3B = 180°
B = 180° : 4/3
B = 180° × 3/4
B = 135°

sudut A :
A + B = 180°
A + 135° = 180°
A = 180° - 135°
A = 45°

atau

A = 1/3 B
A = 1/3 × 135°
A = 45°
