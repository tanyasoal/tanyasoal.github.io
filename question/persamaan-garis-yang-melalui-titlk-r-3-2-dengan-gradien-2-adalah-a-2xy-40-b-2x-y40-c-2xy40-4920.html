jumlah dua bilangan bulat 120 jika bilangan kedua sama dengan sama dengan dua kali bilangan pertama. tentukan kedua bilangan tersebut

Jawaban

Misal
bilangan pertama = a
bilangan kedua = b

a + b = 120
b = 2a

Substitusi
a + b = 120
a + 2a = 120
3a = 120
a = 120 / 3
a = 40

b = 2a
b = 2(40)
b = 80

maka :
bilangan pertama = 40
bilangan kedua = 80

diketahui sebuah prisma alasnya berbentuk belah ketupat yang panjang sisi sisinya 20cm dan panjang salah satu diagonalnya 32cm.jika volume prisma tersebut 9.600cm3 tentukan A luas alas prisma B luas seluruh permukaan prisma

Jawaban

S = 20 cm
d1 = 32 cm
d2 = 2 (√20²-(32/2)²)= 2 (√20²-16²) = 2(√400-256) = 2(√144) = 2(12)=24cm
A. Luas alas = 1/2 x d1 x d2 = 1/2 x 32 x 24 = 384 cm²
B. V = L. alas x tinggi
  9600 = 384 x t
     t = 9600 / 384 = 25 cm
Luas seluruh permukaan = 2 luas alas + 4 luas sisi
                                            = 2(384) + 4 (s x t)
                                            = 768 + 4 (20 x 25)
                                           = 768 + 2000 = 2768 cm ²
