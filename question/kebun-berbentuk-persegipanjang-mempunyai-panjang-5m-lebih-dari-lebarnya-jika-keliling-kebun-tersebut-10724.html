Kebun berbentuk persegipanjang mempunyai panjang 5m lebih dari lebarnya jika keliling kebun tersebut 70m tentukan lebar dari kebun tersebut

Jawaban

→ Mapel : Math
→ Materi : Keliling
→ Bab : Bangun Datar
————————————————

K = 2 x ( p + l )
70 = 2 x ( l + 5 + l )
70 = 2 x ( 2l + 5 )
70 = 4l + 10
4l = 70 - 10
4l = 60
lebar = 15 m
Jadi lebar kebun tersebut adalah 15 m.

→ Mapel : Math
→ Materi : Pecahan
→ Bab : Perbandingan

Jumlah kelereng ketiganya
= 62/38 x 57
= 93 buah
