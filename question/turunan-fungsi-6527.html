Diketahui barisan geometri dengan U1 = 1 dan U4 = 8 jumlah suku pertama deret geometri adalah

Jawaban

Diketahui :
Deret Geometri
U1 = 1
U4 = 8

Ditanya :
S1 = ?

Dijawab :
U1 = a = 1
U4 = a.r^3 = 8

       U4/U1 = 8/1
a.r^3/a = 8
            r^3 = 2^3
                r = 2
 a.r^3 = 8
a.2^3 = 8
   a. 8 = 8
       a = 8/8
       a = 1

Sn = a.(r^n -1)/(r - 1)
S1 = 1.(2^1 -1)/(2 - 1)
S1 = 1.(2 - 1))/(1)
S1 = 1.1/1
S1 = 1

Jadi jumlah suku pertama deret tersebut adalah 1

Kordinat titik balik minimum dari grafik fungsi kuadrat f(x)=x²-2x-3 adalah

Jawaban

BAB Fungsi Kuadrat
Matematika SMA

f(x) = x²-2x-3
titik balik terjadi jika f'(x) = 0
f'(x) = 2x-2 = 0
2x = 2
x = 1
f(1) = 1²-2.1-3
= 1-2-3
= -4

Jadi koordinat titik balik dari grafik fungsi kuadrat f(x) = x²-2x-3 adalah (1,-4)

Jawaban

25 betina dengan (15 berwarna putih)
jumlah ayam jantan :
jumlah seluruh ayam - jumlah ayam betina = 40 - 25 = 15 ayam jantan

jika ada 22 ekor ayam putih maka,
22-15(jumlah ayam betina berwarna putih) = 7 ekor ayam jantan berwarna putih
