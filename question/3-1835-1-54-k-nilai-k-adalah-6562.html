bidang alas sebuah balok berbentuk persegi panjang rusuk alas 8 cm dan tingginya 10 cm Tentukan luas permukaan balok

Jawaban

Rusuk = 8 cm
t = 10 cm

Luas permukaan
= (2x Luas alas) + (Keliling alas x tinggi)
= (2 x rusuk²) + (4 x rusuk x tinggi)
= (2 x 8²) + (4 x 8 x 10)
= (2 x 64) + (320)
= 128 + 320
= 448 cm²
LP=2 x ( (p X l ) + (p X t ) + (l X t) )
=2 x ( (8 x 8)+(8x10)+(8x10) )
=2 x (64+80+80)
=2 x 224=448 cm 2
