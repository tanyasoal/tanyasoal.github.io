bidang alas sebuah balok berbentuk persegi panjang rusuk alas 8 cm dan tingginya 10 cm Tentukan luas permukaan balok

Jawaban

Rusuk = 8 cm
t = 10 cm

Luas permukaan
= (2x Luas alas) + (Keliling alas x tinggi)
= (2 x rusuk²) + (4 x rusuk x tinggi)
= (2 x 8²) + (4 x 8 x 10)
= (2 x 64) + (320)
= 128 + 320
= 448 cm²
LP=2 x ( (p X l ) + (p X t ) + (l X t) )
=2 x ( (8 x 8)+(8x10)+(8x10) )
=2 x (64+80+80)
=2 x 224=448 cm 2

(10cmx13cmx½)x4
65x4=260cm'
luas persegi 10cmx10cm= 100cm'
jadi luas seluruhnya adalah 260cm'+100cm'=360cm'

Jarak sebenarnya kota tangerang dan jakarta adalah 45 km.jika jarak pada peta kedua kota 9 cm.maka berapa skala peta tersebut ? (Tolong di jawab pakai cara ya)

Jawaban

Diketahui : JS = 45km
JpP = 9 cm

skala = jarak sebenarnya/jarak pada peta

jawab : JS/JpP = 45/9 = 5

skala = 1 : 5
Diket :
js : 45 km diubah cm ×100.000 = 4500.000
jp : 9 cm
dit : skala...?
jawab : skala = jp : js
= 9 : 4500.000
= 1 : 500.000
