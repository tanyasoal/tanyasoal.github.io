Diketahui ΔABC sudut A = 30 derajat, sudut B = 45 derajat, dan a = 10cm. Tentukan sudut c, sisi b, dan sisi c

Jawaban

Sudut c
180 - (30+45) = 105 derajat

sisi b

10/ sin 30 = b/ sin 45
10/ ½ = b/ ½√2
kali silang

5√2 = ½b
b = 10√2 cm

sisi c

10/ sin 30 = c/ sin 105
10/ ½ = c/ 0.9659
kali silang

9.659 = ½c
c = 9.659 * 2 = 19.318 cm
