Tentukan persamaan garis melalui titik 3.1 bergradien 1/3

Jawaban

Y-y1= m(×-×1)
y-1 = 1/3(×-3)
y-1 =1/3× -1
dikali 3 supaya tidak pecahan
3y-3=×-3
3y-×-3+3=0
3y-×=0

semoga membantu

atau bisa juga

Gradien garis dengan persamaan 2y kurang 6 x kurang 5 = 0

Jawaban

2y-6x-5=0

2y=6x+5

y=3x + 5/2

jadi gradien nya adalah 3(koefisien x)

^-^
Bentuk umum persamaan :
ax + by + c = 0

Untuk persamaan 2y - 6x - 5 = 0
a = -6
b = 2
c = -5

gradien = -a/b
             = -(-6)/2
             = 3
