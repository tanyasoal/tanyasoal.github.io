Jawaban

1,5 gros = (1,5 x 144) buah
              = 216 buah

8 lusin = (8 x 12) buah
            = 96 buah

1,5 gros + 8 lusin = 216 + 96
                             = 312 buah
1,5  x 144 = 216 buku tulis
8 x 12 = 96 pensil

a+ b = { (8+7)/(8-7)} x 5
        = (15/1)  x 5
