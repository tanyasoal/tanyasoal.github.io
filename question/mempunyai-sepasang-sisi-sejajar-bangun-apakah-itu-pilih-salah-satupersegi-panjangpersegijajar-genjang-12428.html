Koordinat titik puncak dan fokus parabola dengan persamaan x2 + 12y – 4x + 16 adalah ….

Jawaban

Kelas : 11
Mapel : Matematika
Kategori : Irisan Kerucut
Kata Kunci : Persamaan Parabola
Kode : 11.2.10 (Kelas 11 Matematika Bab 10 - Irisan Kerucut)

Materi :

Parabola adalah tempat kedudukan titik-titik yang jaraknya ke satu titik tertentu (titik fokus) sama dengan jaraknya ke sebuah garis tertentu (garis direktriks).

Persamaan parabola horizontal yang terbuka ke kanan (p > 0) atau terbuka ke kiri (p < 0)
(y - b)² = 4p (x - a)
Titik Puncak = (a, b)
Titik Fokus = (a + p, b)

Persamaan parabola vertikal yang terbuka ke atas (p > 0) atau terbuka ke bawah (p < 0)
(x - a)² = 4p (y - b)
Titik Puncak = (a, b)
Titik Fokus = (a, b + p)

Pembahasan :

x² + 12y – 4x + 16 = 0
x² - 4x = -12y - 16
x² - 4x + ... = -12y - 16 + ...
x² - 4x + 4 = -12y - 16 + 4
(x - 2)² = -12y - 12
(x - 2)² = -12 (y + 1)
(x - a)² = 4p (y - b)

Koordinat titik puncak :
(a, b) = (2, -1)

4p = -12 => p = -3

Koordinat titik fokus :
(a, b + p)
= (2, -1 + (-3))
= (2, -4)
