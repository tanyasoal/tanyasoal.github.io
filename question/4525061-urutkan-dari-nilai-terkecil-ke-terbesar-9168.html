Volume limas segi empat beraturan yang panjang sisi alasnya 4 cm dan tingginya 9 cm adalah

Jawaban

V = 1/3 x L alas x t
= 1/3 x (s²) x t
= 1/3 x (4 x 4) x 9
= 1/3 x 16 x 9
= 16 x 3
= 48 cm³

Uno menabung di koperasi sekolah sebesar 750000 dengan bunga 10% setahun. Jika uno menerima bunga sebesar 112500.tentukan lamanya uang yang disimpan di koperasi

Jawaban

M = 750.000
b = 10%
L = 112.500
n = ?

L = M x b% x n/12
n = Lx12/(M x b%)
n = 112.500x12/(750.000 x 10%)
n = 1.350.000/75.000
n = 18 bulan
n = 18/12
n = 1½ tahun
