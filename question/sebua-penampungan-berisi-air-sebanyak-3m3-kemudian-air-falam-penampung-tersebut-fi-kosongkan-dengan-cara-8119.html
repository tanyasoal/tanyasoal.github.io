Jika a= (2x-3y) dan b=(3x+5y) maka hasil dari a^2+b^2 adalah...

Jawaban

A = (2x-3y) dan b = (3x+5y)
Maka,
a² + b² = (2x-3y)² + (3x+5y)²
           = (2x-3y)(2x-3y) + (3x+5y)(3x+5y)
           = 4x² - 6xy - 6xy + 9y² + 9x² + 15xy + 15xy + 25y²
           = (4+9)x² +(30-12)xy + (9+25)y² 
           = 13x² + 18xy + 34y²

jadi, hasil dari a² + b²  adalah 13x² + 18xy + 34y²

Hasil dari 5cm³ + 1.500liter - 750dm³ adalah

pakai caranya

Jawaban

5 cm³ = 5 cm³
1.500 liter = 1.500.000 cm³
750 dm³ = 750.000 cm³
= 5 cm³ + 1.500.000 cm³ = 1.500.005 cm³
=1.500.005 cm³ - 750.000 cm³ = 750.005 cm³

Jadi 5 cm³+1.500 liter-750 dm³ = 750.005 cm³
