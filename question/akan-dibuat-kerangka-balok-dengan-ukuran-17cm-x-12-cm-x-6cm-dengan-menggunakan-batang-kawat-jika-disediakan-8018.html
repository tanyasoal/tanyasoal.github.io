Jawaban

Coba saya bantu ya..

Banyak siswa = 30 anak dan χ = 66 kg
Banyak siswa perempuan = p anak dan χ 1 = 60 kg
Banyak siswa laki laki = (30-p) anak dan χ 2 = 70 kg

χ = pχ1 + (30-p) χ2 / 30

66 = p × 60 + (30-p) × 70 / 30

66 × 30 = 60p + 2.100 - 70p
1.980 = 2.100 - 10p
10p = 2.100 - 1.980
10p = 120
p = 120/10
p = 12

Jadi, banyak siswa wanita yaitu 12 anak.

Luas sebuah persegi panjang sama dengan luas persegi yang panjang sisinya 8 cm .jika lebar persegi panjang 4 cm,hitunglah panjang persegi panjang tersebut.

Jawaban

Diket :
L persegi = L persegi panjang
s = 8 cm
l = 4 cm

Maka,
L persegi = s²
               = 8²
               = 64 cm²

Karena L persegi = L persegi panjang,
Sehingga,
L persegi panjang = p x l
64 = p x 4
p = 64/4
p = 16 cm

Jadi, panjangnya adalah 16 cm
L. Persegi = L. Persegi Panjang
L. Persegi = S×S
                = 8cm×8cm
                = 64cm²
L. Persegi Panjang  = P×l
              64cm²       = P×4cm
         64÷4              = P
              P              = 16 cm

Arjuna menabung pada sebuah bank yang memberikan suku bunga 9% pertahun. Jika besar tabungan Bima setelah 30 bulan Rp 2.205.000,00, berapakah tabungan awal Arjuna…

Jawaban

Misal saldo awal = a

suku bunga 30 bulan = 30/12 x 9%
                                   = 5/2 x 9%
                                   = 22,5 %

saldo akhir = saldo awal + 22.5% x saldo awal
2.205.000 = a + 22,5% a
2.205.000 = 122,5 % a
a = 2.205.000 : 122,5 / 100
a = 2.205.000 x 100 / 122,5
a = 1.800.000

tabungan awal Arjuna adalah Rp. 1.800.000
→ Mapel : Math
→ Materi : Tabungan Awal
→ Bab : Aritmatika Sosial
————————————————

Persentase bung,
= 30/12 x 9%
= 22,5%

Tabungan akhir
= 100% + 22,5%
= 122,5%

Tabungan awal
= 100/122,5 x 2.205.000
= Rp 1.800.000,00
