p = 7a - 8b - c
q = -3a + 2b - 6c 

Maka : 
⇒ p - 2q 
⇒ 7a - 8b - c - 2(-3a + 2b - 6c)
⇒ 7a - 8b - c + 6a - 4b + 12c
