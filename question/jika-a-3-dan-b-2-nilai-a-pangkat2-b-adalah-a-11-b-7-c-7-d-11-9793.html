Pada sebuah toko, hikmah dan hartitik membeli terigu dan beras dengan merek yang sama. Hikmah membeli 5kg terigu dan 7kg beras seharga Rp136.000,00 sedangkan hartitik membeli 10 kg terigu Dan 5 kg beras seharga Rp155.000,00. Harga 3kg terigu dan 3kg beras adalah

Jawaban

→ Mapel : Math
→ Materi : Eliminasi dan Substitusi
→ Bab : SPLDV
————————————————

Terigu = x
Beras = y

Eliminasi
5x + 7y = 136.000 ¦ x 10
10x + 5y = 155.000 ¦ x 5

50x + 70y = 1.360.000
50x + 25y = 775.000
————————————— (kurang)
45y = 585.000
y = Rp 13.000,00

Substitusikan..
5x + 7y = 136.000
5x + 7(13.000) = 136.000
5x = 136.000 - 91.000
5x = 45.000
x = Rp 9.000,00

Maka :
3x + 3y
= 3(9.000) + 3(13.000)
= 27.000 + 39.000
= Rp 66.000,00

Himpunan penyelesaian dari himpunan 7 tanx = 2√3 + tan x, untuk 0 < x <360 adalah A. 30,90
B. 30,210
C. 90,120
D. 90,180
E. 135, 180

Jawaban

7 tan x = 2√3 + tan x
6 tan x = 2√3
tan x = 2√3 / 6
tan x = 1/3 √3
tan x = 30°

x = 30° + n.180°
Untuk n = 0 → 30° + 0.180° = 30°
Untuk n = 1 → 30° + 1.180° = 210°

Himpunan penyelesaiannya adalah 30° dan 210°
