B.RP 11.500,00
C.RP 12.500,00
D.RP 15.500,00
beserta penjelasanya

Jawaban

3B = 2P + 9.000, atau B = 2/3P+ 3.000
3P = 2B - 3.500

* substitusi pers1 ke pers2
3P = 2B - 3.500
3P = 2(2/3P + 3000) - 3.500
3P = 4/3P + 6.000 - 3.500
3P = 4/3P + 2.500
--------------------------------------- tiap ruas x 3
9P = 4P + 7.500
5P = 7.500
P = 1.500

* cari harga buku
B =2/3P + 3.000
B = 2/3 (1.500) + 3.000
B = 1.000 + 3.000
B = 4.000

* harga 2 buku + 3 pulpen
= 2B + 3P
= 2(4.000) + 3(1.500)
= 8.000 + 4.500
= 12.500
