= 1/3 x (s²) x t
= 1/3 x (4 x 4) x 9

= 16 x 3
= 48 cm³

Diketahui luas permukaan balok 104cm²,lebar 6cm dan tinggi 4cm.berapa panjang baloknya?

Jawaban

Luas permukaan = 104 cm²
l = 6 cm
t = 4 cm

L = 2(pl + pt + lt)
104 = 2(px6 + px4 + 6x4)
104 = 2(6p + 4p + 24)
104 = 2(10p + 24)
10p + 24 = 104 / 2
10p + 24 = 52
10p = 52 - 24
10p = 28
p = 28 / 10
p = 2,8 cm

Panjang jari jari dua lingkaran 5 cm dan 12 cm. jarak kedua tiik pusat linkaran 25 cm. hitunglah: A.panjang garis singgung lingkaran dalam. B. panjang garis singgung lingkaran luar

Jawaban

Jawaban :

dik : s = 25 cm
: r1 = 5 cm
: r2 = 12 cm

dit :

a. panjang garis singgung persekutuan dalam (d)
b. panjang garis singgung persekutuan luar (l)

penyelesaian :

__________________________

a. panjang garis singgung persekutuan dalam

d² = s² - ( r1 + r2 )²
d² = 25² - ( 5 + 12 )²
d² = 625 - (17)²
d² = 625 - 289
d = √336
d = 18,3 cm

___________________________
b. panjang garis singgung persekutuan luar

l² = s² - ( r1 - r2 )²
l² = 25² - ( 5 - 12 )²
l² = 625 - (-7)²
l² = 625 + 49
l = √674
l = 25,96 cm
