Apakah hasil yang di dapat psedocode program C++ di bawah ini!
if (A> 0)
cout << "A adalah nilai positif";
else if (A< 0)
cout << "A adalah nilai negativf";
else
cout << "A adalah 0";

Kode diatas tidak akan menghasilkan output, dan akan Error.

• Penjelasan: 

Pada potongan pseudocode tersebut, variabel A tidak didefinisikan, dimana percabngan IF tidak akan dieksekusi. Dan juga akan menghasilkan error karena bertemuan dengan percabangan IF yang mengecek variabel A.
