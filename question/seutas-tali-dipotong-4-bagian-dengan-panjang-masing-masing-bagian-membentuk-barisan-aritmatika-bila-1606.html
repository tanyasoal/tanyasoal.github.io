S4=n/2(2a+(n-1)*b)

smoga mmbantu
≡ penyelesaian :

⇒Tentukan suku pertama ( a ) :

potongan terpendek = U₁ = 5 cm

⇒Tentukan suku ke - n ( Un ) :

potongan terpanjang = Un = U₄ = 14 cm

⇒panjang tali semula sebelum dipotong ( S₄ ) :

Sn = n/2 ( a + Un )
S₄ = 4/2 ( 5 + 14 )
S₄ = 2 ( 19 )
S₄ = 38 cm
