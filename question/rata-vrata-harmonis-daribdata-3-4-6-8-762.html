dalam sebuh barusan aritmatika diketahui U10=45 dan U13=57 jumlah 8 suku pertama dari barusan itu adalah

Jawaban

Diketahui :
U10 = 45
U13 = 57

Ditanya :
S8 = ...?

Jawab :
Un = a + b(n-1)
U10 = a + b(10-1)
45 = a +9b ...(1)
U13 = a + b(13-1)
57 = a + 12b


45 = a +9b
57 = a + 12b
__________-
-12 = -3b
b = -12/-3
b = 4

45 = a + 9b
45 = a + 9(4)
a = 27 - 36
a = -9

Un = a + b(n-1)
U8 = -9 + 4(8-1)
U8 = -9 + 28
U8 = 19

S8 = n/2 (a + U8)
S8 = 8/2 (-9 + 19)
S8 = 4(10)
S8 = 40

jumlah 8 suku pertama dari barusan itu adalah 40.
