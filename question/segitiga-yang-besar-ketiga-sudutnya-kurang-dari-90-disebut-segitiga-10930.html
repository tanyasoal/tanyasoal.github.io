Luas lingkaran 1.386 cm .jari-jari lingkaran tersebut adalah

Jawaban

Mapel : Matematika
Materi : Lingkaran

◼Jari Jari Lingkaran

r = √[L/π]
r = √[1.386/22/7]
r = √[1.386 x 7/22]
r = √[441]
r = 21 cm
Luas Lingkaran = π.r²
1.386 = 22/7 . r²
r² = 1.386 : 22/7
r² = 1.386 x 7/22
r² = 63 x 7
r² = 441
r = √441
r = 21 cm

Apa rumus bangun balok,kubus,tabung,dan prisma segitiga


Bagi bagi poin

Jawaban

Rumus kubus:


Luas permukaan kubus : 6 x luas sisi = 6 x rusuk x rusuk

Panjang diagonal ruang : akar dari (3 x rusuk kuadrat)

Volume Kubus : rusuk x rusuk x rusuk = rusuk^3


Rumus Balok :


Luas Permukaan balok : 2 x { (pxl) + (pxt) + (lxt) }

Panjang diagonal ruang : akar dari (p kuadrat + l kuadrat + t kuadrat)

Volume balok : p x l x t

umus Prisma :


Luas Permukaan Prisma segitiga:


= luas alas + luas selimut + luas atap

= ( 2 x luas alas) + (kell alas x tinggi)


Volume Prisma : luas alas x tinggi

Ket:
La = luas alas
K = keliling alas
t = tinggi prisma
Rumus Tabung/Silinder :


Luas Permukaan Tabung :


= luas alas + luas selimut + luas tutup

= luas lingkaran + luas segi empat + luas lingkaran

= (2 x luas lingkaran) + luas segi empat

= { (2 x π x r^2) + (π x d x t) }

                                          

Volume Tabung :


= luas alas x tinggi

= luas lingkaran x t

= π x r^2 x t
