Jawaban

7 tan x = 2√3 + tan x
6 tan x = 2√3
tan x = 2√3 / 6
tan x = 1/3 √3
tan x = 30°

x = 30° + n.180°
Untuk n = 0 → 30° + 0.180° = 30°
Untuk n = 1 → 30° + 1.180° = 210°
