Halaman rumah miluk pak umar berbentuk persegi panjang berukuean panjang (3x-2) meter dan lebar (x + 4) meter.jika kelilingnya 44 meter,panjang dan lebar halaman tersebut adalah

Jawaban

➰ Mapel Math
➰ Bab Bangun Datar

Keliling persegi panjang
k = 2 ( p + l )
44 = 2 ( 3x - 2 + x + 4 )
44 = 2 ( 4x + 2 )
44 = 8x + 4
44 - 4 = 8x
40 = 8x
x = 40 / 8
x = 5

Ukuran panjang
= 3x - 2
= 3(5) - 2
= 15 - 2
= 13 meter

Ukuran lebar
= x + 4
= 5 + 4
= 9 meter


Good Luck !

Jawaban

Mapel : Matematika
Materi : Barisan dan Deret
Kelas  : 9

U₄ = 8
U₇ = 17

≡ Beda Barisan
    U₇ - U₄ = 17 - 8
    7b - 4b = 9
    3b        = 9
      b        = 9/3
      b        = 3

≡ Nilai suku Pertama
     U₄         = 8
     a + 3b   = 8
     a + 3(3) = 8
     a + 9     = 8
     a           = 8 - 9 
     a           = -1

Sehingga, Nilai suku ke - 10 barisan adalah :

Un  = a + (n - 1)b
U₁₀ = -1 + (10 - 1)(3)
U₁₀ = -1 + 9(3)
U₁₀ = -1 + 27
U₁₀ = 26
...,...,...,8,...,...,17,...,...,...,
mencari beda : 17 - 8 = 9 : 3 =3
beda = 3
maka:
-1, 2, 5, 8, 11, 14, 17, 20, 23, 26

atau dengan rumus:
Un = a + (n-1) b
U10 = -1 + (9 x 3)
U10 = -1 + 27
U10 = 26
