dinda membeli pita sepanjang 5,4 meter dinda memnbeli lagi sepanjang 2 1/2 meter kemudian pita tersebut diberikan adiknya sepanjang 4,6 meter berapa sisa pita yg dimiliki dinda

Jawaban

2 1/2 meter = 2,5 meter
Panjang pita dinda = 5,4 + 2,5 - 4,6 = 3,3 meter
Jawaban :

5,4 meter = 27/5 meter
2½ meter = 5/2 meter
4,6 meter = 23/5 meter

Sisa Pita Dinda
( 5,4 + 2½ ) - 4,6
= ( 27/5 + 5/2 ) - 23/5
= ( 54/10 + 25/10 ) - 23/5
= 79/10 - 23/5
= 79/10 - 46/10
= 33/10
= 3 3/10 meter

Jawaban

Harga sebelum diskon = Rp. 350.000

Diskon = 5,5% x Rp. 350.000
            = 5,5/100 x Rp. 350.000
            = Rp. 19.250

Harga setelah diskon = Rp. 350.000 - Rp. 19.250
                                   = Rp. 330.750

Alfan akan pergi ke rumah ibu Tati yang jaraknya 4,6 km. Jalan menuju rumah Bu Tati melewati rumah neila.jika jarak antara rumah Alfan dan neila 25 hm. Berapakah jarak antara rumah neila dan Bu Tati..... Meter tolong di jawab pakai cara

Jawaban

Alfan - Bu Tati = 4,6 km

Melewati Rumah Neila *
Rumah Neila Dengan Bu Tati = 25hm
25hm = 2,5km

Jadi 4,6 km - 2,5km = 2,1 km ( jarak rumah Neila dan Bu Tati )
Ket. A = alfan, B = bu tati, dan C = neila
Jarak rumah A ke rumah B = 4,6 km = 4.600 m
rumah B melewati rumah C
Jarak rumah A ke rumah C = 25 hm = 2.500 m
berapakah jarak rumah C ke rumah B = 4.600 m - 2.500 m = 2.100 m

Sebuah pabrik mempunyai persediaan minyak goreng 0,24 m3.pabrik menerima kiriman minyak goreng sebanyak 150 dm3.jika pabrik menggunakan minyak goreng sebanyak 32 liter. Persediaan minyak goreng sekarang adalah??? Dengan caranya

Jawaban

Diket : persediaan minyak goreng 0,24 m^3 (2.4 dm^3)
menerima kiriman 150 dm^3
minyak digunakan 32 liter
ditanya : persediaan minyak goreng sekarang?
jawab ; dm^3 = liter
2.4 + 150 - 32 = 120,4 liter
Persediaan minyak goreng =
= 0,24 m3 = 240 liter
= 150 dm3 = 15.000 dl(desi liter) : 10 = 1.500 liter
= 32 liter

= 240 liter + 1.500 liter - 32 liter (karena digunakan)
= 1.708 liter

Jadi persediaan minyak goreng sekarang adalah 1.708 liter
