Jawaban

Panjang kerangka 1 balok = 4 x (15+10+8) = 4 x 33 = 132 cm
banyak kerangka = 11x100/132 = 8 buah
sisa panjang kawat = 1.100 - 132x8 = 1.100 - 1.056 = 44 cm

Jawaban



Lp = ( 2 × luas alas ) + ( keliling alas × tinggi )
= ( 2 × 1/2 × d1 × d2 ) + ( 4 × s × t )
= ( 2 × 1/2 × 24 × 18 ) + ( 4 × 12.7 × 20 )
= 432 + ( 50.8 × 20 )
= 432 + 1016
= 1448 cm2
