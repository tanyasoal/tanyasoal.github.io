Kumparan primer trafo terdiri dari 800 lilitan dan kumpatan sekundernya terdiri dari 200 lilitan. Pada kumparan sekunderterpasang penghambat 100. Jika kumparan primer terhubung pada sumber tegangan 200 volt, kuat arus pada kumparan sekunder dan kuat arus pada kumparan primer berturut- turut A.20 A dan 5 A
B. 5 A dan 20 A
C. 5 A dan 1,25 A
D. 1,25 A dan 5 A

Jawaban

Diket :
Np = 800
Ns = 200
Vp = 200 v
Rs = 100 ohm


Dit : Is & Ip.........?


Jwb :

# Tegangan sekunder

Vs = ( Ns . Vp ) / Np
Vs = ( 200 . 200 ) / 800
Vs = 40.000 / 800
Vs = 50 volt

# Arus sekunder

Is = Vs / Rs
Is = 50 / 100
Is = 0,5 A

# Arus primer

Ip = ( Vs . Is ) / Vp
Ip = ( 50. 0,5 ) / 200
Ip = 25 / 200
Ip = 0,125 A

Jadi arus sekunder 0,5 A dan arus primer 0,125 A

sebuah trafo arus primer dan sekundernya masing masing 3A dan 2A. jumlah lilitan primer dan sekunder masing masing 4.000 dan 1.200, efisiensi trafo sersebut adalah...%

Jawaban

Diket :
Ip = 3A
Is = 2A
Np = 4000
Ns = 1.200

ditanya :
efisiensi trafo = ?

Jawab :
Np/Ns = Is/Ip = Vp/Vs
4000/1.200 = 2A/3A = Vp/Vs
4000 × 3 / 1.200 × 2 = Vp/Vs
12000 /2.400 = Vp/Vs
120 = Vp
24 = Vs

efisiensi trafo
= Ps/Pp × 100%
= Vs × Is / Vp × Ip × 100%
= 24 × 2 / 120 × 3 × 100%
= 48 / 360 × 100%
= 4/30 × 100%
= 13,3 %
