gradiennya = -a / b = - (-2) / 1 = 2

4. 3y = x >> x - 3y = 0

karena semua gradiennya berbeda maka tidak ada yg sejajar

Jawaban

D = panjang garis singgung
p = jarak kedua titik pusat
r1 = jari-jari 1
r2 = jari-jari 2

d luar
= √p kuadrat - (r1 - r2) kuadrat

Tentukan persamaan garis yang melalui titik 2,(-4) dan -3,1

Jawaban

Bila diketahui dua titik yang dilalui yaitu (2,-4) dan (-3,1), maka rumusan persamaan garis singgungnya adalah
((x - x1)/(x2 - x1)) = ((y - y1)/(y2 - y1))
((x - (-3))/(2 - (-3)) = ((y - 1)/(-4 - 1))
(x + 3)/5 = (y - 1)/-5 ---> kali silang
-5(x + 3) = 5(y - 1) ---> kedua ruas dibagi 5
-(x + 3) = y - 1
-x - 3 = y - 1
x + y = -3 + 1
x + y = -2

Jawaban

Lebar = x
panjang = x + 6

52 = x + x + 6 + x + x + 6
52 = 12 + 4x
4x = 40
x = 10

lebar = 10
panjang = 16
luas = 10×16 = 160
