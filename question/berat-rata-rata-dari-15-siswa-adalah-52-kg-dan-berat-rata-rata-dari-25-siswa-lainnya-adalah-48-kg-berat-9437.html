umur Tian 2/3 umur Galang. Umur galang 3/5 umur Aldo. Selisih umur Tian dan Aldo 12thn. Berapa jumlah umur mreka?? tolong lagi ya:)!!

Jawaban

Jawaban :

Selisih Umur Galang dan Aldo = 12 Tahun

Umur Tian
= ⅔ x 12 Tahun
= 24/3
= 8 Tahun

Umur Galang
= 3/5 x 12
= 36/5
= 7,2 Tahun

Umur Aldo
= Selisih Umur Galang dan Aldo - Umur Galang
= 12 Tahun - 7,2 Tahun
= 4,8 Tahun

Jumlah Umur
= 8 Tahun + 7,2 Tahun + 4,8 Tahun
= 20 Tahun

Jawaban

Misal saldo awal = a

suku bunga 30 bulan = 30/12 x 9%
                                   = 5/2 x 9%
                                   = 22,5 %

saldo akhir = saldo awal + 22.5% x saldo awal
2.205.000 = a + 22,5% a
2.205.000 = 122,5 % a
a = 2.205.000 : 122,5 / 100
a = 2.205.000 x 100 / 122,5
a = 1.800.000

tabungan awal Arjuna adalah Rp. 1.800.000

Arbi Akan membuat dua buah aquarium tanpa tutup berbentuk balok dengan panjang 120cm, lebar 50cm dance tinggi 60cm berapa meter persegi, kaca yan diperlukan arbi untuk membuat dua akuarium tersebut?

Jawaban

P = 120 cm
l = 50 cm
t = 60 cm

Luas permukaan balok tanpa tutup
= (p x l) + 2(pt + lt)
= (120 x 50) + 2(120x60 + 50x60)
= 6.000 + 2(7.200 + 3.000)
= 6.000 + 20.400
= 26.400 cm²
= 2,64 m²
