Sebuah balok luas pemukaannya 342 cm pesegi. Jika p : l : t = 4 : 3 : 1, maka volume balok itu adalah a.104 cm2 b.124cm2 c.224cm2 d.324cm2

Jawaban

Luas permukaan = 342 cm²
p : l : t = 4 : 3 : 1

misal :
p = 4x
l = 3x
t = x

Luas permukaan = 2(pl + pt + lt)
342 = 2(4x . 3x + 4x . x + 3x . x)
342 = 2(12x² + 4x² + 3x²)
342 / 2 = 19x²
19x² = 171
x² = 171 / 19
x² = 9
x = √9
x = 3

maka
panjang = 4x = 4(3) = 12 cm
lebar = 3x = 3(3)= 9 cm
t = x = 3 cm

Volume = p x l x t
             = 12 x 9 x 3
             = 324 cm³

Sebuah prisma dengan alas berbentuk belah ketupat memiliki keliling alas 68cm dan panjang salah satu diagonal alasanya 16 cm. Jika tinggi prisma itu 25cm. Volume prisma tersebut adalah....

Jawaban

Keliling alas = 68
4.s = 68
s = 68/4
s = 17 cm
panjang salah satu diagonal = 16 cm
panjang 1/2 diagonal = 8 cm
panjang 1/2 diagonal lainnya
= √(s^2 - panjang 1/2 diagonal^2)
= √(17^2 - 8^2)
= √(289 - 64)
= √225
= 15 cm
panjang diagonal lainnya adalah 30 cm

Volume prisma
= L alas x tinggi
= ((d1.d2)/2).t
= ((16.30)/2).25
= (8.30.25)
= 6000 cm^3
K. belah ketupat = 4 x panjang sisi
68 = 4 x panjang sisi
panjang sisi = 68 : 4
= 17

Maka,
Panjang d2 = 2 x [√(17² - (1/2 x d1)²)]
Panjang d2 = 2 x [√(17² - (1/2 x 16)²)]
Panjang d2 = 2 x 15
Panjang d2 = 30

V. Prisma = L. Alas x tinggi
= 1/2 x d1 x d2 x tinggi
= 1/2 x 16 x 30 x 25
= 6000 cm^3
