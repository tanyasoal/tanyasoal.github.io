Suku ke 3 dan suku 6 suatu deret aritmetika berturut turut adalah 40 dan 60. Jumlah sepuluh suku pertama deret tersebut adalah

Jawaban

Diketahui :
U3 = 40
U6 = 60

Ditanya :
S10 = ...?

Jawab :
Un = a + b(n-1)
U3 = a + b(3-1)
40 = a + 2b ...(1)
U6 = a + b(6-1)
60 = a + 5b ... (2)

=> subsitusikan ke persamaan (1) dan (2)
40 = a + 2b
60 = a + 5b
_________-
-20 = -3b
b = -20/-3
b = 6,6

=> eliminasikan ke persamaan (1) atau (2)
60 = a + 5b
60 = a + 5(6,6)
60 = a + 33
a = 60 - 33
a = 27

=> rumus Un :
Un = a + b(n-1)
Un = 27 + 6,6(n-1)
Un = 27 + 6,6n - 6,6
Un = 20,4 + 6,6n
U10 = 20,4 + 6,6(10)
U10 = 20,4 + 66
U10 = 86,4 => dibulatkan menjadi 86

=> Jumlah sepuluh suku pertama deret tersebut adalah :
Sn = n/2 (a + Un)
S10 = 10/2 (27+ U10)
S10 = 10/2 (27 + 86)
S10 = 5(113)
S10 = 565

Maka luasnya adalah 0,0063 are.Langkah Penyelesaian & Pembahasan : Materi : Karena bentuk tanah adalah persegi panjang, maka untuk mengukur luasnya kita menggunakan rumus luas persegi panjang. Sebelum itu kita akan mengkonversi satuan panjang agar keduanya ke dalam satuan yang sama. Di antara pilihan cm dan m, maka di sini akan saya coba kerjakan dengan mengubah semua satuan panjangnya ke bentuk cm. Langkah Penghitungan : Tips : Hitung luas tanah kemudian konversikan ke satuan are. Diketahui :
  • p = panjang tanah = 90 cm
  • l = lebar tanah = 7 m = 7 × 100 cm
Ditanya : L = luas lahan = ... are Jawaban :
  • Hitung terlebih dahulu ke bentuk cm²
  • Ubah ke bentuk are dengan catatan 1 cm² = 1/1.000.000 are.
Jadi, luas tanah adalah 0,0063 are.
