perbandingan panjanh lebar dan tinggi sebuah balok adalah 5 : 4 : 3 jika volumenya balok 1.620 cm tentukan ukuran balok trsbt

Jawaban

Mapel : Matematika
Materi : Volume Balok dan Kubus
Kelas : 8

Misalkan
⇔ Panjang = 5x
⇔ Lebar = 4x
⇔ Tinggi = 3x

◼ Tentukan Nilai x terlebih dahulu

Volume balok = p x l x t

1.620 = 5x • 4x • 3x
1.620 = 60x³

⇔ 60x³ = 1.620
⇔ x³ = 1.620/60
⇔ x³ = 27
⇔ x = ³√27
⇔ x = 3

Sehingga, Ukuran balok tersebut
◼ Panjang

= 5x
= 5(3)
= 15 cm

◼ Lebar

= 4x
= 4(3)
= 12 cm

◼ Tinggi

= 3x
= 3(3)
= 9 cm

Sebuah persegi panjang dengan p : l = 2 : 3 jika luasnnya 384cm² hitung : a. panjang b. lebar

Jawaban

→ Mapel : Math
→ Materi : Luas
→ Bab : Bangun Datar
————————————————

L = p x l
384 = 2a x 3a
384 = 6a²
a² = 384 ÷ 6
a² = 64
a = √64
a = 8 cm

Panjang = 2a = 2(8) = 16 cm
Lebar = 3a = 3(8) = 24 cm
Sebuah persegi panjang dengan luas 384 cm² dimana panjang nya 2 bagian dan lebar nya 3 bagian.

Misal x = satu bagian


Satu bagian senilai 8

Sehingga panjang dan lebar persegi panjang tersebut adalah

p = 2 bagian = 2×8 = 16 cm
l = 3 bagian = 3×8 = 24 cm
