1.Diketahui suatu balok dengan ukuran 26 cm×14 cm×10 cm.Hitunglah: A.luas permukaan balok
B.panjang rusuk/kerangka

Jawaban

 A.       LP balok = 2 * (p*l) + (p*t) + (l*t)
= 2 * ( 26*14 ) + ( 26*10 ) + ( 14*10 )
= 2 * 364 + 260 + 140
= 2 * 764 
= 1528 cm

B.      panjang rusuk balok
= 4 * ( p + l + t )
= 4 * ( 26 + 14 + 10 )
= 4 * 50
= 200 cm
