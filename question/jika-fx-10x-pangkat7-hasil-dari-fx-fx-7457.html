c. Carilah persamaan garis singgung itu!

Jawaban

y = x^2 -2x -3.
perpotongan kurva dengan sumbu y, x = 0
y = -3. Jadi koordinat titik singgung = (0, -3)
y' = 2x - 2
y' untuk x = 0
y' = 2.0 - 2 = -2 => adalah gradien garis singgung, misal m,
persamaan garis singgung
y - y1 = m (x - x1)
y - -3 = -2 (x - 0)
y + 3 = -2x
y = -2x - 3
