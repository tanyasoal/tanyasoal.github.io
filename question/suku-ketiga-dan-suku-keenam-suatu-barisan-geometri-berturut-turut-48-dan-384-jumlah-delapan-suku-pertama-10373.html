Pada barisan aritmatika suku ke enam adalah 24 dan suku ke sepuluh adalah 60 tentukan nilai dari su ke 20

Jawaban

Kelas 9 Matematika
Bab 6 - Barisan dan Deret

Penyelesaian :

Suku ke-6
Un = a + (n - 1) b
U6 = a + (6 - 1) b = 24
U6 = a + 5b = 24 ...........(1)

Suku ke-10
Un = a + (n - 1) b
U10 = a + (10 - 1) b = 60
U10 = a + 9b = 60.............(2)

Beda tiap suku
a + 5b = 24
a + 9b = 60
_________ _
- 4b = - 36
b = - 36 / - 4
b = 9

Suku Pertama
a + 5b = 24
a + 5(9) = 24
a + 45 = 24
a = 24 - 45
a = - 21

Suku ke-20
Un = a + (n - 1) b
U20 = - 21 + (20 - 1) 9
U20 = - 21 + (19) 9
U20 = - 21 + 171
U20 = 150

Jadi, nilai suku ke-20 adalah 150

Jika sudutA berpelurus dengan sudutB.dan besar sudutA=1\3B maka besar sudut A adalah

Jawaban

A + B = 180° (berpelurus)
1/3 × B + B = 180°
1/3B + 1B = 180°
1/3B + 3/3B = 180°
4/3B = 180°
B = 180° : 4/3
B = 180° × 3/4
B = 135°

sudut A :
A + B = 180°
A + 135° = 180°
A = 180° - 135°
A = 45°

atau

A = 1/3 B
A = 1/3 × 135°
A = 45°
