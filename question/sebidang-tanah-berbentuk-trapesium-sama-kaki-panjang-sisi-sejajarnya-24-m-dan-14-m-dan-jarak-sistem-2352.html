Sebidang tanah berbentuk trapesium sama kaki. Panjang sisi sejajarnya 24 m dan 14 m, dan jarak sistem sejajarnya 12 mau. Jika sekeliling tanah tersebut dibuat pagar, panjang pagar seluruhnya adalah

Jawaban

Kelas : 7
Mapel : Matematika
Kategori : Bangun Datar

Panjang Sisi Miring Trapesium
= √[{(24-14)/2}² + 12²]
= √[(10/2)² + 144]
= √[5² + 144]
= √[25 + 144]
= √[169]
= 13 cm

Maka Panjang Kawat Seluruhnya
= (2 x 13) + 24 + 14
= 26 + 38
= 64 cm

Jawaban

4mobil = a
motor = b

roda mobil = 4 = 4a
roda motor = 2 = 2b

mencari banyak motor dan mobil

cara eliminasi
a + b = 75 ~ x 4
4a + 2b = 210 ~ x 1

menjadi
4a + 4b = 300
4a + 2b = 210
___________-
2b = 70
b = 70/2 = 35

a + b = 75 ~ x 2
4a + 2b = 210 ~ x 1

menjadi
2a + 2b = 150
4a + 2b = 210
___________-
-2a = -60
a = -60/-2 = 30

pendapatan = (30 x 5000) + (35 x 2000)
= 150.000 + 70.000
= 220.000

Jadi pendapatan uang parkir adalah Rp220.000,00.
