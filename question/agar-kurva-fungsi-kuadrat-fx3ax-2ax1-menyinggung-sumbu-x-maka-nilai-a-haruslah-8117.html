Jawaban

X² - ax - 5 = 0
x = p atau x = q
(x-p)(x-q) = 0
x² - qx - px + pq = 0
x² - (q+p)x + pq = 0

pq = -5
q + p = a

p² + q² - 2pq = 36
(p-q)² = 36
p - q = 6
p = 6 + q


pq = -5
(6+q).q = -5
6q + q² = -5
q² + 6q + 5 = 0
(q+5)(q+1) = 0
q = -5 atau q = -1

Jika q = -5
p = 6 - 5 = 1

Jika q = -1
p = 6 - 1 = 5

Nilai a
a = q + p
a = -5 + 1
a = -4

atau
a = q + p
a = -1 + 5
a = 4

Jadi, a = -4 atau a = 4

Jawaban

Pembahasan :

kiper dirgantara dapat menggagalkan tendangan lawan sebesar 0,8 → merupakan peluang sukses
jadi peluang suksesnya 0,8 = 8/10 = 4/5
peluang gagalnya 1 - 0,8 = 0,2 = 2/10 = 1/5
terjadi 5 kali pinalti
2 gol → gagal menahan serangan

= 5C2 . (1/25) . (64/125)
= 5!/(5-2)!2! . (1/25) . 964/125)
= 5 x 4 x 3! / 3!2! . (1/25) . (64/125)
= 10 . (1/25) . (64/125)
