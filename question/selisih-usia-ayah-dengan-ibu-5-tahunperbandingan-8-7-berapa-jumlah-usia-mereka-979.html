Selisih usia ayah dengan ibu 5 tahun.perbandingan 8 : 7 ,berapa jumlah usia mereka?

Jawaban

A - b = 5
a : b = 8 : 7 

a + b = {(jumlah perbandingan a dan b) / selish perbadingan a dan b} x selisih sebenarnya

a+ b = { (8+7)/(8-7)} x 5
        = (15/1)  x 5
        = 75 tahun

Arbi Akan membuat dua buah aquarium tanpa tutup berbentuk balok dengan panjang 120cm, lebar 50cm dance tinggi 60cm berapa meter persegi, kaca yan diperlukan arbi untuk membuat dua akuarium tersebut?

Jawaban

P = 120 cm
l = 50 cm
t = 60 cm

Luas permukaan balok tanpa tutup
= (p x l) + 2(pt + lt)
= (120 x 50) + 2(120x60 + 50x60)
= 6.000 + 2(7.200 + 3.000)
= 6.000 + 20.400
= 26.400 cm²
= 2,64 m²
