Harga 3kg mangga dan 2kg jeruk Rp116.000,00, harga 4kg mangga dan 5kg jeruk Rp199.000,00. Jadi berapa harga 1kg jeruk dan 2kg mangga?

Jawaban

Jawaban ⇒ Rp 64.000,00

❏ Misalkan
mangga » x
jeruk » y

sehingga
3 kg mangga + 2 kg jeruk = 116.000
=> 3x + 2y = 116.000

4 kg mangga + 5 kg jeruk = 199.000
=> 4x + 5y = 199.000

❏ Eliminasi variabel x
4x + 5y = 199.000 ( ×3 )
3x + 2y = 116.000 ( ×4 )

12x + 15y = 597.000
12x + 8y = 464.000 --
-----------------------------
7y = 133.000
y = 133.000 ÷ 7 = 19.000

❏ Substitusikan y = 19.000 ke

3x + 2y = 116.000
3x + 2 (19.000) = 116.000
3x + 38.000 = 116.000
3x = 116.000- 38.000
3x = 78.000
x = 78.000 ÷ 3
x = 26.000

❏ Harga 1 kg jeruk dan 2 kg mangga
= 1x + 2y
= 1 (26.000) + 2 (19.000)
= 26.000 + 38.000
= 64.000

Jadi, harganya adalah Rp 64.000,00

Diketahui deret geometri dengan suku pertama 5 dan suku keempat 135.jumlah 6 suku pertama adalah

Jawaban

Diketahui :
Deret geometri
U1 = 5
U4 = 135

Ditanya :
S6 = ?

Dijawab :
U1 = a = 5
U4 = a.r^3 = 135

       U4/U1 = 135/5
a.r^3/a = 27
            r^3 = (3)^3
                r = 3

   a.r^3 = 135 
  a.3^3 = 135
   a. 27 = 135
       a = 135/27
       a = 5

Sn = a.(r^n -1)/(r - 1)
S6 = 5.(3^6 -1)/(3 - 1)
S6 = 5.(729 - 1))/(2)
S6 = 5.728/2
S6 = 3640/2
S6 = 1820

Jadi jumlah emapat suku pertama deret tersebut adalah 1.820
