Diketahui f(4x+5)=8x-2 dan g(x+7)=3x+1.Tentukan f(x)dan g(x)

Jawaban

F(4x + 5) = 8x - 2
g(x + 7) = 3x + 1

f(x) dan g(x) = ... ?

*misal
4x + 5 = a
4x = a - 5
x = (a - 5)/4

f(4x + 5) = 8x - 2
f(a) = 8 [(a - 5)/4] - 2
f(a) = 2 (a - 5) - 2
f(a) = 2a - 10 - 2
f(x) = 2x - 12

*misal
x + 7 = a
x = a - 7

g(x + 7) = 3x + 1
g(a) = 3 (a - 7) + 1
g(a) = 3a - 21 + 1
g(x) = 3x - 20

Kordinat titik balik minimum dari grafik fungsi kuadrat f(x)=x²-2x-3 adalah

Jawaban

BAB Fungsi Kuadrat
Matematika SMA

f(x) = x²-2x-3
titik balik terjadi jika f'(x) = 0
f'(x) = 2x-2 = 0
2x = 2
x = 1
f(1) = 1²-2.1-3
= 1-2-3
= -4

Jadi koordinat titik balik dari grafik fungsi kuadrat f(x) = x²-2x-3 adalah (1,-4)
