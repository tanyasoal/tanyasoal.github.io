Jika 1/a + 1/b + 1/c = o dan a+b+c =-8 berapa nilai a² + b² +c

Jawaban

1/a + 1/b + 1/c = 0 dan a+b+c = -8. Nilai a²+b²+c²? (saya anggap c²) 

1. ubah 1/a + 1/b + 1/c = 0 menjadi tidak pecahan dengan dikali abc
    1/a + 1/b + 1/c = 0  (×abc)
     bc + ac + ab = 0

2. cari nilai a²+b²+c²
    (a+b+c)² = (a+b+c)(a+b+c)
                   = a²+b²+c²+ 2ab+ 2bc + 2ac
                   = a²+b²+c² + 2 (ab+bc+ac)
           (-8)² = a²+b²+c² + 2 (0)
              64 = a²+b²+c²
     a²+b²+c² = 64
    
jadi, nilai a²+b²+c² adalah 64

Jika pelurus dari 75 derajat adalah 3r derajat maka yang memenuhi nilai r derajat adalah

Jawaban

Jika pelurus, maka :
3r° + 75° = 180°
3r° = 180° - 75°
3r° = 105°
r° = 105° / 3
r° = 35°
