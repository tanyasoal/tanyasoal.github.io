Penjelasan: 
Untuk membuat output segitiga pascal seperti itu, sebenarnya yang dibutuhkan adalah looping. Anda dapat mengkobinasikan dengan IF Then Else untuk memilih beberapa kondisi, seperti kondisi tinggi segitiga atau lainnya.

Contoh potongan kode untuk membuat output segitiga pascal dengan bahasa Java:

Scanner scan = new Scanner (System.in);
// meminta input tinggi segitiga
System.out.println("Masukkan tinggi segitiga: ");
int tinggi = scan.nextInt();
if (tinggi > 1) {
  // membuat output segitiga pascal
  for (int i = 0; i < tinggi; i++ {
    for (int k = 0; k < tinggi; k++) {
      System.out.print("* ");
    }
    System.out.println();
  }
} else {
  System.out.println("Tinggi segitiga harus lebih dari 1 !");
}

Tentukan modulus dari 43 mod 8

43 mod 8 = 3

• Penjelasan: 

Modulus merupakan hasil sisa dari pembagian terdekat dari bilangan tersebut

hasil kali dari 8 yang terdekat dari 43 adalah 40 (8 x 5)

43 - 40 adalah 3, maka 43 mod 8 adalah 3
43 : 8
=43-40( karena 5×8)
=3

intinya modulus itu sisa dari hasil pembagian terdekat dari target
