seorang pekerja bangunan membeli 2 kaleng cat dan 3 kuas seharga Rp101.500,00. esok harinya pekerja itu membeli 1 kaleng cat dan 2 kuas yang sama seharga Rp53.500,00. harga 1 kaleng cat dan 1 kuas adalah

Jawaban

→ Mapel : Math
→ Materi : Eliminasi dan Substitusi
→ Bab : SPLDV
————————————————

Kaleng = x
Kuas = y

Eliminasi
2x + 3y = 101.500 ¦ x 1
x + 2y = 53.500 ¦ x 2

2x + 3y = 101.500
2x + 4y = 107.000
——————————— (-)
-y = -5.500
y = 5.500

Substitusikan...
x + 2y = 53.500
x + 2(5.500) = 53.500
x = 53.500 - 11.000
x = 42.500

Maka :
x + y
= 5.500 + 42.500
= Rp 48.000

subtitusi

c = 42.500
