sebuah kapal belayar dari pelabuhan a ke arah utara sejahu14km kemudian belok ke arah 48km.jarak terdekat kapal dari pelabuhan a adalah

Jawaban

→ Mapel : Math
→ Materi : Jarak Terdekat
→ Bab : Teorema Pythagoras
————————————————

Jarak terdekat
= √48² + 14²
= √2304 + 196
= √2500
= 50 km
Gunakan rumus phytagoras

c² = a² + b²
= 14² + 48²
= 196 + 2.304
= 2.500
c = √2.500
= 50 km

Jadi jarak terdekatnya adalah 50 km.

maka
sn=a((r^n)-1)/r-1=765
sn=3((2^n)-1)/2-1=765
sn=3((2^n)-1)=765
sn=((2^n)-1)=765/3
