Diketahui deret geometri dengan suku pertama 5 dan suku keempat 135.jumlah 6 suku pertama adalah

Jawaban

Diketahui :
Deret geometri
U1 = 5
U4 = 135

Ditanya :
S6 = ?

Dijawab :
U1 = a = 5
U4 = a.r^3 = 135

       U4/U1 = 135/5
a.r^3/a = 27
            r^3 = (3)^3
                r = 3

   a.r^3 = 135 
  a.3^3 = 135
   a. 27 = 135
       a = 135/27
       a = 5

Sn = a.(r^n -1)/(r - 1)
S6 = 5.(3^6 -1)/(3 - 1)
S6 = 5.(729 - 1))/(2)
S6 = 5.728/2
S6 = 3640/2
S6 = 1820

Jadi jumlah emapat suku pertama deret tersebut adalah 1.820
