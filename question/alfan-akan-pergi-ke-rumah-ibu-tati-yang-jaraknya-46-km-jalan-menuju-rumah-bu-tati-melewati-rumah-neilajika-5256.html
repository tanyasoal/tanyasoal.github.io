Alfan akan pergi ke rumah ibu Tati yang jaraknya 4,6 km. Jalan menuju rumah Bu Tati melewati rumah neila.jika jarak antara rumah Alfan dan neila 25 hm. Berapakah jarak antara rumah neila dan Bu Tati..... Meter tolong di jawab pakai cara

Jawaban

Alfan - Bu Tati = 4,6 km

Melewati Rumah Neila *
Rumah Neila Dengan Bu Tati = 25hm
25hm = 2,5km

Jadi 4,6 km - 2,5km = 2,1 km ( jarak rumah Neila dan Bu Tati )
Ket. A = alfan, B = bu tati, dan C = neila
Jarak rumah A ke rumah B = 4,6 km = 4.600 m
rumah B melewati rumah C
Jarak rumah A ke rumah C = 25 hm = 2.500 m
berapakah jarak rumah C ke rumah B = 4.600 m - 2.500 m = 2.100 m

Diketahui m=16 2 per 3% dan n=66 2 per 3%
hasil m:n adalah ...

Jawaban

Jawabannya '


Mapel : Matematika
Bab : Operasi Bilangan
Sub Bab : Perbandingan

Pembahasan :

m = 16 2/3% = 50%
n = 66 2/3% = 200%

Maka '

m : n
= 50% : 200%
= 50/100 : 200/100
= 1/2 : 2/1
= 1/2 × 1/2
= 1/4 → Jwbnya
