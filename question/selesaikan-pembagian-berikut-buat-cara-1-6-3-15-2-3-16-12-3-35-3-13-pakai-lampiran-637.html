Grafik fungsi f(x) =x³-3x²-9x+15-5 naik pada interval ...

Jawaban

Pembahasan :

f(x) = x³ - 3x² - 9x + 15 - 5
naik pada interval ...

kita gunakan turunan
f(x) = x³ - 3x² - 9x + 15 - 5
f(x) = x³ - 3x² - 9x + 10
f'(x) = 3x² - 6x - 9 
naik pada saat f'(x) > 0

f'(x) = 3x² - 6x - 9 kita kecilkan dengan membagi 3
x² - 2x - 3 > 0
(x - 3)(x + 1) > 0
kita dapati dua bilangan -1 dan 3
kita cek nol fungsi atau x = 0

3x² - 6x - 9 > 0
3(0)² - 6(0) - 9 > 0
0 - 0 - 9 > 0
-9 > 0 salah
maka interval naik pada
x < -1 atau x > 3


jadi f(x) = x³ - 3x² - 9x + 15 - 5 naik pada interval x < -1 atau x > 3

Sola meminjam uang di koperasi sebesar Rp. 7.200.000 dan diangsur selama 12 bulan dengan bunga 2% per bulan. Besar angsuran tiap bulan yang akan Sola bayar?

Jawaban

Jawaban
diketahui :besar pinjaman =Rp7.200.000
lama angsuran =12 bulan
besar bunga =2%
ditanya. :Besar angsuran tiap bulan yang
akan Sola bayar?
dijawab

Bunga per bulan = 2%x7.200.000
= 2/100xx7.200.000
= 144.000

Angsuran per bulan = 7.200.000/12 + 144.000
= 600.000 + 144.000
= 744.000

Jadi, angsuran perbulan sebesar Rp744.000
