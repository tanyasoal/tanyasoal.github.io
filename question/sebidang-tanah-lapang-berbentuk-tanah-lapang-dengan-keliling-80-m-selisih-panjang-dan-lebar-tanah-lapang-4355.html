Jawaban

Dik : U2 + U6 = -6
U2 + U8 = -2
Dit : S5?

Jawab :
•U2 + U6 = -6 dipecah menjadi
a + b + a + 5b = -6
2a + 6b = -6
___________ : 2
a + 3b = -3

• U2 + U8 = -2 dipecah menjadi
a + b + a + 7b = -2
2a + 8b = -2
__________ :2
a + 4b = -1

•Dieliminasi
a + 3b = -3
a + 4b = -1
_________ -
-b = -2
b = 2

• Disubtitusi
a + 3b = -3
a = -3 - 3(2)
a = - 3 - 6
a = -9

•Mencari S5
Sn = n/2 (2a + (n-1) b)
S5 = 5/2 (2 x -9 + (5-1) 2)
S5 = 5/2 (-18 + 8)
S5 = 5/2 (-10)
S5 = -25
