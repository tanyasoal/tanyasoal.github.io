BAB: SPLDV



Jawab:

f(-2)= 14
f(3)= -1


=> -2p+q = 14
=> 3p+q = -1

3p+q = -1
---------------- --
-5p = 15
p = -3

Jika p=7a-8b-c dan Q= - 3a + 2b - 6c,hasil dari p-2Q adalah....

Jawaban

Materi : Aljabar 
Tingkat : SMP 

p = 7a - 8b - c
q = -3a + 2b - 6c 

Maka : 
⇒ p - 2q 
⇒ 7a - 8b - c - 2(-3a + 2b - 6c)
⇒ 7a - 8b - c + 6a - 4b + 12c
⇒ 7a + 6a - 8b - 4b - c + 12c
⇒ 13a - 12b + 11c
