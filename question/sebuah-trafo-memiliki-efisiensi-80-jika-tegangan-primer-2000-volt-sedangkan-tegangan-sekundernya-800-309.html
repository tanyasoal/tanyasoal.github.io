Sebuah trafo memiliki efisiensi 80%. Jika tegangan primer 2000 volt, sedangkan tegangan sekundernya 800 volt, tentukanlah kuatvarus primernya jika kuat arus sekundernya 4 ampere!

Diketahui :
η = 80% = 0,8
Vp = 2000 volt
Vs = 800 volt
is = 4 A

Ditanya :
Ip = ?

Penyelesaian :
η = Vs. Is / Vp. Ip
0,8 = 800.4 / 2000. Ip
0,8 = 3200/2000Ip
1600Ip = 3200
Ip = 3200/1600
Ip = 2 A

∴ Jadi, kuat arus primer trafo tsb adalah 2A
