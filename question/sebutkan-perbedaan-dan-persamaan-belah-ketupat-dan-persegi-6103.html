Panjang persegi panjang = 5/4 dari lebarnya luasnya 180 cm kuadrat maka kelilingnya adalah..........

Jawaban

Luas = 180cm2
panjang = 5/4 x lebar

p x l = L
(5/4 x l) x l = L
5/4 x l x l = 180cm2
l^2 = 180 x 4 : 5
l^2 = 720 : 5
l^2 = 144
l = √144
l = 12 cm
p = 180/12 = 15cm
keliling
= 2 x (p + l)
= 2 x (15 + 12)
= 2 x 27
= 54cm
