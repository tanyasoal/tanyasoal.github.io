Jawaban

= 20 kg + 30 kg

= 50 kg

Persentase berat Anggur Merah

= 20 kg/50 kg × 100 %

= 20/1 × 2 %

= 40 %

Persentase Anggur Hijau

= 30 kg/50 kg × 100 %

= 30/1 × 2 %

= 60 %

Jawaban

2 1/2 meter = 2,5 meter
Panjang pita dinda = 5,4 + 2,5 - 4,6 = 3,3 meter
Jawaban :

5,4 meter = 27/5 meter
2½ meter = 5/2 meter
4,6 meter = 23/5 meter

Sisa Pita Dinda
( 5,4 + 2½ ) - 4,6
= ( 27/5 + 5/2 ) - 23/5
= ( 54/10 + 25/10 ) - 23/5
= 79/10 - 23/5
= 79/10 - 46/10
= 33/10
= 3 3/10 meter
