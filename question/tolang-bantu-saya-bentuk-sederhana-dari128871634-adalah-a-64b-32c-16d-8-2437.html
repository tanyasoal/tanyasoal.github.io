Tolang bantu saya bentuk sederhana dari(128)8/7:(16)3/4 adalah.....
A. 64
B. 32
C. 16
D. 8

Jawaban

(128)8/7 : (16)3/4
(2^7)8/7 : (2⁴)3/4 (coret 7 sama 7 dan 4 sama 4)
jadi 2^8 : 2^3
256:8=32 (B)
(128)8/7 : (16)3/4
= (2^7)8/7 : (2^4)3/4
= (2)^8 : (2)^3
= 256 : 8
= 32

Arbi Akan membuat dua buah aquarium tanpa tutup berbentuk balok dengan panjang 120cm, lebar 50cm dance tinggi 60cm berapa meter persegi, kaca yan diperlukan arbi untuk membuat dua akuarium tersebut?

Jawaban

P = 120 cm
l = 50 cm
t = 60 cm

Luas permukaan balok tanpa tutup
= (p x l) + 2(pt + lt)
= (120 x 50) + 2(120x60 + 50x60)
= 6.000 + 2(7.200 + 3.000)
= 6.000 + 20.400
= 26.400 cm²
= 2,64 m²

Jawaban

Cari gradien dari persamaan 4x + 3y = 12

Bentuk umum persamaan garis
ax + by + c = 0

untuk persamaan tersebut :
a = 4
b = 3
c = -12

m = -a/b = -4/3

Karena tegak lurus,
m1 . m2 = -1
-4/3 . m2 = -1
m2 = 3/4

Persamaan garis melalui titik (-2, 5)
(y - y1) = m(x - x1)
(y - 5) = 3/4(x - (-2))
4(y - 5) = 3(x + 2)
4y - 20 = 3x + 6
3x - 4y = -20 - 6
3x - 4y = -26
