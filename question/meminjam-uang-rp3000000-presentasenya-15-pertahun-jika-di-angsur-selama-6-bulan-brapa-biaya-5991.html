Jawaban

Volume kolam renang = volume prisma dengan alas trapesium siku siku
= L alas.tinggi
= ((1 + 3).40/2).15
= 4.40/2.15
= 4.20.15
= 1200 m^3

Panjang persegi panjang = 5/4 dari lebarnya luasnya 180 cm kuadrat maka kelilingnya adalah..........

Jawaban

Luas = 180cm2
panjang = 5/4 x lebar

p x l = L
(5/4 x l) x l = L
5/4 x l x l = 180cm2
l^2 = 180 x 4 : 5
l^2 = 720 : 5
l^2 = 144
l = √144
l = 12 cm
p = 180/12 = 15cm
keliling
= 2 x (p + l)
= 2 x (15 + 12)
= 2 x 27
= 54cm

Arbi Akan membuat dua buah aquarium tanpa tutup berbentuk balok dengan panjang 120cm, lebar 50cm dance tinggi 60cm berapa meter persegi, kaca yan diperlukan arbi untuk membuat dua akuarium tersebut?

Jawaban

P = 120 cm
l = 50 cm
t = 60 cm

Luas permukaan balok tanpa tutup
= (p x l) + 2(pt + lt)
= (120 x 50) + 2(120x60 + 50x60)
= 6.000 + 2(7.200 + 3.000)
= 6.000 + 20.400
= 26.400 cm²
= 2,64 m²
