Jawaban

F(x) = x³ + 3x² + a

f(2) = (2)³ + 3(2)² + a
= 8 + 3.4 + a
= 8 + 12 + a
= 20 + a

f'(2) ??
f'(x) = 3x² + 6x + 1
f'(2) = 3(2)² + 6(2) + 1
= 3.4 + 12 + 1
= 12 + 12 + 1
= 25

f"(x) = ??
f'(x) = 3x² + 6x + 1
f"(x) = 6x + 6
f"(2) = 6(2) + 6
= 12 + 6
= 18

f"(2) + f'(2) + f(2)
= 18 + 25 + 20+a
= 51 + a
