Diketahui deret geometri dengan suku pertama 5 dan suku keempat 135.jumlah 6 suku pertama adalah

Jawaban

Diketahui :
Deret geometri
U1 = 5
U4 = 135

Ditanya :
S6 = ?

Dijawab :
U1 = a = 5
U4 = a.r^3 = 135

       U4/U1 = 135/5
a.r^3/a = 27
            r^3 = (3)^3
                r = 3

   a.r^3 = 135 
  a.3^3 = 135
   a. 27 = 135
       a = 135/27
       a = 5

Sn = a.(r^n -1)/(r - 1)
S6 = 5.(3^6 -1)/(3 - 1)
S6 = 5.(729 - 1))/(2)
S6 = 5.728/2
S6 = 3640/2
S6 = 1820

Jadi jumlah emapat suku pertama deret tersebut adalah 1.820

Jawaban

Lingkaran


x²+y²-4x-8y+15=0
(x-2)² - 4 + (y-4)² - 16 + 15 = 0
(x-2)²+ (y-4)² = 5

Maka
Pusat (2,4)
r = √5
Kelas 11 Matematika
Bab Persamaan Lingkaran

x² + y² - 4x - 8y + 15 = 0
x² - 4x + y² - 8y = -15
(x - 2)² + (y - 4)² = -15 + 2² + 4²
(x - 2)² + (y - 4)² = 5

r² = 5
r = √5

Pusat = (2, 4)
