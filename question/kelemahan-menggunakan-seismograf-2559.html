Diketahui :
h = 75 cm = 0,75 m
p = 800 kg/m³ (massa jenis minyak)
g = 10 m/s² (anggap)

Ditanya :
Ph = ?

Penyelesaian :
Ph = pgh
Ph = (800)(10)(0,75)

◾ Materi : Usaha dan Energi
◾ Sub materi : Usaha
◾ Mapel : Fisika
◾ Keyword : Usaha, Kerja

ㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡ

Diketahui :
F1 = +30 N
F2 = -40 N
F3 =+90 N
s = 250 cm = 2,5 m

Ditanya :
W = ?

Penyelesaian :
*Cari resultan gaya
Ftotal = F1+F2+F3
Ftotal = +30-40+90
Ftotal = +80 N

*Cari usaha yg bekerja
W = Ftotal x s
W = 80 x 2,5
W = 200 J

∴ Jadi, usaha yg bekerja pada meja adalah 200 J

Jawaban

KINEMATIKA
▶Persamaan Posisi

x = 12t² - 2t³

a. Posisi saat t = 3 s ???

x = 12 (3)² - 2(3)³
x = 54 m ✔

b. Kecepatan saat t = 3 s ???

v = dx / dt
v = d(12t² - 2t³) / dt
v = 24t - 6t²

v = 24 (3) - 6 (3)²
v = 18 m/s ✔

c. Percepatan pada t = 3 s ???

a = dv / dt
a = d(24t - 6t²) / dt
a = 24 - 12t

a = 24 - 12(3)
a = - 12 m/s²
a = 12 m/s² ✔

d. X maks = ????

x maks terjadi saat x' = 0
x = 12t² - 2t³
x' = 24t - 6t²
0 = 24t - 6t²
6t² - 24t = 0
6t(t - 4) = 0
t = 0 (ditolak) atau t = 4 sekon

x maks = 12(4)² - 2(4)³
x maks = 64 m

Koordinat maksimum = 64 m , saat t = 4 sekon ✔

e. v maks = ???

v maks terjadi saat v' = 0

v = 24t - 6t²
v' = 24 - 12t
0 = 24 - 12t
t = 2 sekon

v = 24(2) - 6(2)²
v = 24 m/s

v maks adalah 24 m/s , saat t = 2 sekon ✔

f. Percepatan partikel saat tidak bergerak adalah 0.

g. ṽ antara t = 0 s sampai t = 3 s = ??

saat t = 0 :
x = 12t² - 2t³
x = 12(0)² - 2(0)³ = 0

saat t = 3 s :
x = 12(3)² - 2(3)³ = 54 m

Maka :
ṽ = (54 - 0) / (3 - 0)
ṽ = 18 m/s ✔
