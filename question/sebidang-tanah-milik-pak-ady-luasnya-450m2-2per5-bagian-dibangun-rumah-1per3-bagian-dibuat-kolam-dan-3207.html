Sebuah kubus berisi air 4096 dam3 jika air hanya memenui setengah dari bak tersebut maka tinggi bak=...........dm

Jawaban

Kubus= sisi x sisi x sisi
kubus = sisi³
volume bak = 1/2 dari volume kubus 
      4096 dam³ = 1/2 sisi³
      4096 x 2 dam³ = sisi³
      8192 dam³ = sisi³
      sisi = ∛8192
            = 20,15 dam
 tinggi bak = 1/2 x 20,15
                  = 10,075 dam

diketahui suku ke-4 dan suku ke-7 suatu barisan aritmatika berturut-urut adalah 8 dan 17. suku ke-10 dari barisan tersebut adalah

Jawaban

Mapel : Matematika
Materi : Barisan dan Deret
Kelas  : 9

U₄ = 8
U₇ = 17

≡ Beda Barisan
    U₇ - U₄ = 17 - 8
    7b - 4b = 9
    3b        = 9
      b        = 9/3
      b        = 3

≡ Nilai suku Pertama
     U₄         = 8
     a + 3b   = 8
     a + 3(3) = 8
     a + 9     = 8
     a           = 8 - 9 
     a           = -1

Sehingga, Nilai suku ke - 10 barisan adalah :

Un  = a + (n - 1)b
U₁₀ = -1 + (10 - 1)(3)
U₁₀ = -1 + 9(3)
U₁₀ = -1 + 27
U₁₀ = 26
...,...,...,8,...,...,17,...,...,...,
mencari beda : 17 - 8 = 9 : 3 =3
beda = 3
maka:
-1, 2, 5, 8, 11, 14, 17, 20, 23, 26

atau dengan rumus:
Un = a + (n-1) b
U10 = -1 + (9 x 3)
U10 = -1 + 27
U10 = 26
