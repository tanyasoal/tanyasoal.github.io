Uno menabung di koperasi sekolah sebesar 750000 dengan bunga 10% setahun. Jika uno menerima bunga sebesar 112500.tentukan lamanya uang yang disimpan di koperasi

Jawaban

M = 750.000
b = 10%
L = 112.500
n = ?

L = M x b% x n/12
n = Lx12/(M x b%)
n = 112.500x12/(750.000 x 10%)
n = 1.350.000/75.000
n = 18 bulan
n = 18/12
n = 1½ tahun

seorang ibu mempunyai 5 orang anak, anak tertua berumur 2p tahun, dan anak termuda berumur p tahun. tiga anak yg lainnya berturut turut berumur 2p-2, p+2, p+1. jika rata rata umur mereka 17 tahun, maka umur anak tertua adalah

Jawaban

Rata² = (jumlah data) / banyaknya data
17 = (2p + p + 2p - 2 + p + 2 + p + 1) / 5
17 = (7p + 1) / 5
7p + 1 = 17 x 5
7p + 1 = 85
7p = 85 - 1
7p = 84
p = 84 / 7
p = 12

Umur anak tertua = 2p
                             = 2(12)
                             = 24 tahun
