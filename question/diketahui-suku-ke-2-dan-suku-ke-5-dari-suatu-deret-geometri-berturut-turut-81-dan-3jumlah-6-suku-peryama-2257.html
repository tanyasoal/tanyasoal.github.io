Jawaban

Bentuk |x - 3| ≥ 6 artinya

a) x - 3 ≥ 6, yaitu
x ≥ 6 + 3
x ≥ 9

atau
b) x - 3 ≤ -6
x ≤ -6 + 3
x ≤ -3

Sehingga himpunan penyelesaiannya
HP = { x | x ≤ -3 atau x ≥ 9, x € R }

Jika x1 & x2 adalah akar akar dari persamaan 2x2-3x+5=0 maka persamaan kuadrat baru yg akar akarnya x1-3 dan x2-3 adalah

Jawaban

2x^2 - 3x + 5 = 0
a = 2, b = -3, c = 5
X1 × X2 = c/a = 5/2
X1 + X2 = -b/a = 3/2
• (X1 - 3)(X2 - 3) = X1.X2 - 3X1 - 3X2 + 9
= X1.X2 - 3(X1 + X2) + 9
= 5/2 - 3(3/2) + 9
= 5/2 - 9/2 + 9
= 4/2
= 2
• (X1 - 3) + (X2 - 3) = X1 + X2 - 6
= 3/2 - 6
= -9/2
Persamaan Kuadrat Baru
x^2 - (X1.X2)x + (X1 + X2)
= x^2 - 2x - 9/2
= 2x^2 - 4x - 9
