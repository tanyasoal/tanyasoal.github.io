volume gas dalam suatu ruang 8cm pangkat 2 mempunyai tekanan 2 atm tentukan volume dalam ruangan jika tekanannya diperbesar 16 kali

Jawaban

◾ Materi : Teori Kinetik Gas
◾ Sub materi : Gas Ideal
◾ Mapel : Fisika
◾ Keyword : Isothermal

ㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡ

Diketahui :
v1 = 8 cm³
p1 = 2 atm
p2 = 16p1 = 16(2) = 32 atm

Ditanya :
v2 = ?

Penyelesaian :
p1v1 = p2v2
2(8) = 32(v2)
16 = 32v2
v2 = 16/32
v2 = 0,5 cm³

∴ Jadi, volume gas dalam ruangan itu berubah menjadi 0,5 cm³

ㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡ

N = 5000 lilitan
ΔΦ = 7000 - 1500 = 5500 Wb
t = 5 detik

Ditanya :
ε

Penyelesaian :
ε = N. ΔΦ/Δt
ε = 5000 (5500/5)
ε = 5000 ( 1100 )
ε = 5.500.000 volt
ε = 5,5x10^6 volt

∴ Jadi, besarnya ggl induksi yg muncul adalah 5,5 x 10^6 volt

= ((250) + (90)) ÷ (35)
= 340 ÷ 35
