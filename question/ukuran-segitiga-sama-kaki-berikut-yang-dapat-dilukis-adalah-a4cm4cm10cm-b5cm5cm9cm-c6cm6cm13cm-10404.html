Jika 1/a + 1/b + 1/c = o dan a+b+c =-8 berapa nilai a² + b² +c

Jawaban

1/a + 1/b + 1/c = 0 dan a+b+c = -8. Nilai a²+b²+c²? (saya anggap c²) 

1. ubah 1/a + 1/b + 1/c = 0 menjadi tidak pecahan dengan dikali abc
    1/a + 1/b + 1/c = 0  (×abc)
     bc + ac + ab = 0

2. cari nilai a²+b²+c²
    (a+b+c)² = (a+b+c)(a+b+c)
                   = a²+b²+c²+ 2ab+ 2bc + 2ac
                   = a²+b²+c² + 2 (ab+bc+ac)
           (-8)² = a²+b²+c² + 2 (0)
              64 = a²+b²+c²
     a²+b²+c² = 64
    
jadi, nilai a²+b²+c² adalah 64

Jawaban

N = 6
U6 = 8
S6 = 93

Sn = n/2(a + Un)
S6 = 6/2(a + U6)
93 = 3(a + 8)
a + 8 = 93 / 3
a + 8 = 31
a = 31 - 8
a = 23
