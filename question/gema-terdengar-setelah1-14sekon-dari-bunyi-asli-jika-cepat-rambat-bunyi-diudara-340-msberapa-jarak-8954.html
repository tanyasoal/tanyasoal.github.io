Jawaban

N(A) = ₂C₁ × ₃C₁ × ₅C₁
        = 2! /1!(2-1)!  ×  3!/1!(3-1)!  ×  5!/1!(5-1)!
        = 2 × 3 × 5
        = 30

n(S) = ₁₀C₃
        = 10! / 3!(10-3)!
        = (10×9×8)×7! / 3! 7!
        = (10×9×8)/ (3×2×1)
        = 120

P(A) = n(A) / n(S)
        = 30 / 120
        = 1/4

jadi, besar peluangnya adalah 1/4

A'( 2a - x , 2b - y )

2 - x = 3 ←→ x = -1
dan
-4 - y = 5 ←→ y = -9

Tentukan himpunan penyelesaian dari
|x - 3| ≥ 6

Jawaban

Bentuk |x - 3| ≥ 6 artinya

a) x - 3 ≥ 6, yaitu
x ≥ 6 + 3
x ≥ 9

atau
b) x - 3 ≤ -6
x ≤ -6 + 3
x ≤ -3

Sehingga himpunan penyelesaiannya
HP = { x | x ≤ -3 atau x ≥ 9, x € R }

Semoga jelas dan membantu

Materi : Nilai Mutlak

Pembahasan :

Kalau menurut saya begini.

Misal : 1 × 1 ≥ a → x ≤ -a atau x ≥ a

X ≤-a
|x-3|≤ -6
X-3≤-6
X≤ -6+3
X≤ -3


Atau
X ≥a
|x-3|≥6
X≥ 6+3
X≥9

*Jadi HP*
X ≥ 9, x ≤ -3
