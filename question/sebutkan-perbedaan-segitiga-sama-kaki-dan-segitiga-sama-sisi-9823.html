Persamaan garis melalui titik A(-3,4) fan B(2,-5) adalah

Jawaban

Melalui titik A(-3, 4) dan B(2, -5)

x1 = -3
x2 = 2
y1 = 4
y2 = -5

Persamaan garis
(y - y1) / (y2 - y1) = (x - x1) / (x2 - x1)
(y - 4) / (-5 - 4) = (x - (-3)) / (2 - (-3))
(y - 4) / (-9) = (x + 3) / 5
5(y - 4) = -9(x + 3)
5y - 20 = -9x - 27
9x + 5y = -27 + 20
9x + 5y = -7
atau
9x + 5y + 7 = 0

Jwb:5/8 × 32 ( 8 nya dri 5+3)

Banyak Kelereng Andi
> Andi / (Andi + Budi) x 32

> 5 / 8 x 32
> 5 x (32 ÷ 8)
> 5 x 4
