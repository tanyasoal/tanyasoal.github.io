tinggi

1180 = 60 T

Sebuah taman berbentuk persegi panjang berukuran panjang 32 m dan lebar 24 m si sekeliling taman akan dipasang lampu dengan jarak antar lampu 4 m berapa lampu yang diperlukan

Jawaban

Keliling=2×(p+l)
=2×(32+24)
=2×56
=112m
lampu yg diperlukan=112:4
=28 lampu
≡ penyelesaian :

⇒Tentukan keliling taman :

↬Keliling = 2 ( p + l )
↬K = 2 ( 32 + 24 )
↬K = 2 ( 56 )
↬K = 112 m

⇒maka, banyak lampu :

= Keliling / Jarak
= 112 / 4
= 28 buah

Luas lingkaran 1.386 cm .jari-jari lingkaran tersebut adalah

Jawaban

Mapel : Matematika
Materi : Lingkaran

◼Jari Jari Lingkaran

r = √[L/π]
r = √[1.386/22/7]
r = √[1.386 x 7/22]
r = √[441]
r = 21 cm
Luas Lingkaran = π.r²
1.386 = 22/7 . r²
r² = 1.386 : 22/7
r² = 1.386 x 7/22
r² = 63 x 7
r² = 441
r = √441
r = 21 cm
