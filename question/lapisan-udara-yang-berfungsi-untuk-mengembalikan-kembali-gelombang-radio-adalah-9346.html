cepat rambat bunyi di air laut adalah 240m/s getaran yang dikirimkan oleh kapal laut oleh kapal ke dasar laut akan diterima dalam waktu 3s. Hitunglah kedalaman laut disuatu tempat tersebut

Jawaban

Mapel : IPA Fisika 
Kelas : 8 SMP/VIII SMP
Bab : Gelombang dan Bunyi 
Tentang : Gelombang

Pembahasan Soal :
Cepat rambat bunyi di air laut adalah 240 m/s getaran yang dikirimkan oleh kapal laut oleh kapal ke dasar laut akan diterima dalam waktu 3 s. Hitunglah kedalaman laut disuatu tempat tersebut..

Pembahasan Jawaban :
DiKetahui :
v = 240 m/s
t = 3 sekon 

DiTanya :
s =...?

DiJawab :
Rumus mencari s
Yaitu :
s = v x t : 2 
s = 240 m/s x 3 sekon : 2 
s = 120 m/s x 3 sekon 
s = 360 meter
Jadi, Kedalaman laut tersebut adalah 360 meter.

Keterangan :
s = Jarak/kedalaman laut ( meter )
v = Cepat Rambat gelombang ( m/s )
t = Waktu yang ditempuh ( sekon )

benda masa 3kg kalori jenisnya 460j/kg°c dipanaskan dari suhu 10° menjadi 60°c tentukan banyaknya kalori yg di perlukan

Jawaban

Mapel : IPA Fisika 
Kelas : 8 SMP/VIII SMP
Bab : Kalor dan Suhu 
Tentang : Kalor Jenis 

Pembahasan Soal :
Benda masa 3 kg kalori jenisnya 460 J/kg°C dipanaskan dari suhu 10°C menjadi 60°C tentukan banyaknya kalori yg di perlukan...

Pembahasan Jawaban :
DiKetahui :
m(benda) = 3 kg 
C = 460 J/kg°C 
T1 = 10°C 
T2 = 60°C 

DiTanya :
Q =....?

DiJawab :
Rumus mencari Q
Yaitu :
Q = m(benda) x C x ΔT 
Q = m(benda) x C x ( T2 - T1 )
Q = 3 kg x 460 J/kg°C x ( 60°C - 10°C )
Q = 3 kg x 460 J/kg°C x 50°C 
Q = 1.380 x 50 
Q = 69.000 Joule 
Q = 69 kJ.
Jadi, Kalor yang dibutuhkan adalah 69.000 J/ 69 kJ.

Keterangan :
m(benda) = Massa benda ( kg )
T1 = Suhu pertama ( °C )
T2 = Suhu kedua ( °C )
ΔT = Selisih suhu ( °C )
C = Kalor Jenis benda ( J/kg°C )
Q = Banyak kalor ( Joule )
