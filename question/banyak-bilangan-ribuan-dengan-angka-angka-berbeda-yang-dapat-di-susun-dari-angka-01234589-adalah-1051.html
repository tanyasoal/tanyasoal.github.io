Jawaban

2√20 + √45 - √180 + √80
= 2√(4 x 5) + √(9 x 5) - √(36 x 5) + √(16 x 5)
= 2 x √4 x √5 + √9 x √5 - √36 x √5 + √16 x √5
= 2 x 2 x √5 + 3 x √5 - 6 x √5 + 4 x √5

= (4 + 3 - 6 + 4)√5
