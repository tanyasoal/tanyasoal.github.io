Tali sepanjang 1,44 m dipotong menjadi 3 bagian. perbandingan panjang potong tali 1,2,3 adalah 5 : 4 : . tentukan panjang tali setiap potong

Jawaban

1,44 m = 144 cm

panjang tali ke2 = 144 cm / 3
= 48 cm
panjang tali pertama = tali2 + (5+4+3)
= 48 + 12
= 60 cm
panjang tali ke3 = tali2 - (5+4+3)
= 48 - 13
= 36 cm

panjang tali setiap potong
tali1 = 60 cm = 0,6 m
tali2 = 48 cm = 0,48 m
tali3 = 36 cm = 0,36 m

60 : 48 : 36 = 5 : 4 : 3

Uno menabung di koperasi sekolah sebesar 750000 dengan bunga 10% setahun. Jika uno menerima bunga sebesar 112500.tentukan lamanya uang yang disimpan di koperasi

Jawaban

M = 750.000
b = 10%
L = 112.500
n = ?

L = M x b% x n/12
n = Lx12/(M x b%)
n = 112.500x12/(750.000 x 10%)
n = 1.350.000/75.000
n = 18 bulan
n = 18/12
n = 1½ tahun

Jawaban

Jika penjahit dapat menjahit 60 kaos dalam 3 hari, maka bila ia bekerja selama 2 minggu atau 14 hari, kaos yang dapat dijahit adalah

60 : 3 = 20, jadi dalam 1 hari dapat menjahit 20 kaos
20 x 14 = 280, jadi dalam 2 Minggu kaos yang dapat ia jahit adalah 280 potong kaos

4. Prisma Segi-n
Untuk prisma segi-enam, segi-tujuh, dst, Segi-n anda dapat menggunakan

Banyak sisi/bidang prisma segi-n = n + 2

Banyak rusuk prisma segi-n = 3n

Banyak titik sudut prisma segi-n = 2n

Rumus Prisma
1. Volume Prisma
Untuk menghitung besar volume prisma digunakan rumus:

Volume = Luas alas x tinggi

Dimana tinggi adalah tinggi prisma

Misalnya:

Volume Prisma segitiga = Luas alas x t

= (1/2 x alas x tinggi) x t

Volume Prisma segiempat = Luas alas x t

= (p x l) x t

2. Luas pemukaan prisma

Luas = Jumlah luas bidang-bidang sisinya

atau
