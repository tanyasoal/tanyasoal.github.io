Sebuah Transformator memiliki perbandingan lilitan primer dan lilitan sekunder 3:5 di hubungkan dengan sumber tegangan ac dengan tegangan 220 volt dan kuat arus 4 apmere.tentukan: efisiensi trafo

Jawaban

Diket :
Np : Ns = 3 : 5
Vp = 220 v
Ip = 4 A


Dit : Efisiensi trafo.......?


Jwb :

# Tegangan sekunder

Vs = ( Ns . Vp ) / Np
Vs = ( 5 . 220 ) / 3
Vs = 1100 / 3
Vs = 367 volt

# Arus sekunder

Is = ( Vp . Ip ) / Vs
Is = ( 220 . 4 ) /367
Is = 880 / 367
Is = 2,4 A

# Efisiensi

ef = ( Vs . Is / Vp . Ip ) . 100%
ef = ( 367 . 2,4 / 220 . 4) . 100%
ef =( 880,8 / 880) .100%
ef = 100%

Jadi efisiensi trafo tsb 100%

Jawaban

Penyelesaian :

Dik : s = 1,1 m  
        n = 5,5 x (1 bukit dan 1 lembah)
Dit : λ = ... ?

Jawab : λ = s/n
                = 1,1 m /  5,5 x
                = 0,2 m
                = 20 cm
