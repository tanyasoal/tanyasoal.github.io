Jawaban

Bab : Bilangan Berpangkat

Penyelesaian:
(-3)² + (-3)¹ + (-3) + (-3)³
= 9 + (-3) + (-3) + (-27)
= 9 - 3 - 3 - 27
= -24
(-3)² + (-3)¹ + (-3) + (-3)³
= (-3) × (-3) + (-3) + (-3) + (-3) × (-3) × (-3)
= 9 + (-6) + (-27)
= -24
Jawabannya adalah -24
