perbandingan jumlah lilitan primer dan sekunder 1:9 tegangan primer 100volt dan daya 750watt berapa kuat arus sekunder nya?

Jawaban

Diket :
Np : Ns = 1 : 9
Vp = 100 v
Pp = 750 w

Dit : Arus sekunder......?


Jwb :


# Tegangan sekunder

Vs = ( Ns . Vp ) / Np
Vs = ( 9 . 100)/ 1
Vs = 900 volt

# Arus primer

Ip = Pp / Vp
Ip = 750 / 100
Ip = 7,5 A


# Arus sekunder

Is = ( Vp . Ip ) / Vs
Is = ( 100 . 7,5 ) / 900
Is = 750 / 900
Is = 0,83 A


Jadi arus sekundernya 0,83 A
