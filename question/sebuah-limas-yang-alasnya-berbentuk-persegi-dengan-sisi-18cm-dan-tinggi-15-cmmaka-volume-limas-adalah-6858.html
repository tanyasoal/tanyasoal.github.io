Volume limas = Luas alas x tinggi x 1 / 3
                       = 18 x 18 x 15 x 1 / 3
                       = 18 x 18 x 5
                       = 1620 cm²
Volume limas= 1/3×Luas alas× tinggi

= 1/3×18×18×15
= 1620 cm^3

Jawaban

Jawab :

d^2 = (AB)^2 - (R - r)^2
15^2 = (AB)^2 - (10 - 2)^2
225 = (AB)^2 - 64
(AB)^2 = 225 + 64
(AB)^2 = 289
AB = √289
AB = 17 cm

Jadi jarak titik pusat 17 cm

2x + 6 (10.000) = 72.000
2x  + 60.000 = 72.000
2x = 72.000 - 60.000
2x = 12.000
x   = 12.000 ÷ 2
x   = 6.000


2x + 2y = n
2 (6.000) + 2(10.000) = n
