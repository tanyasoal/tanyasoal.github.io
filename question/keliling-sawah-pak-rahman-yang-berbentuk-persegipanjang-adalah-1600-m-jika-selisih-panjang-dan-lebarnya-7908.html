Keliling sawah Pak Rahman yang berbentuk persegipanjang adalah 1.600 m. Jika selisih panjang dan lebarnya 200 m, luas sawahnya adalah ....

Jawaban

P - l = 200
p = 200 + l

keliling = 2 x (p + l)
1600 = 2.(200 + l + l)
1600/2 = 200 + 2l
800 = 200 + 2l
800 - 200 = 2l
600 = 2l
600/2 = l
300 m = l

p = 200 + l
p = 200 + 300
p = 500 m

L = p. l
L = 500.300
L = 150000 m^2

Jawaban

Materi : Aljabar 
Tingkat : SMP 

p = 7a - 8b - c
q = -3a + 2b - 6c 

Maka : 
⇒ p - 2q 
⇒ 7a - 8b - c - 2(-3a + 2b - 6c)
⇒ 7a - 8b - c + 6a - 4b + 12c
⇒ 7a + 6a - 8b - 4b - c + 12c
⇒ 13a - 12b + 11c

Jawaban

40 org ----> 126 hari
x org ----> 80 hari

Maka:
Perbandingan terbalik,
40 . (126) = 80 . x
x. = (40 . (126)) / 80
x = 126 / 2
x = 63 orang

Sehingga tambahan orang:
= 63 - 40
= 23 orang
