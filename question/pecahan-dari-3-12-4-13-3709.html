di tempat parkir sebuah pertokoan terdapat 75 kendaran yang terdiri dari mobil dan motor.banyak roda seluruh nya ada 210.jika tarif parkir mobil Rp.5.000 dan motor Rp 2.000 maka pendapatan uang parkir saat itu adalah

Jawaban

4mobil = a
motor = b

roda mobil = 4 = 4a
roda motor = 2 = 2b

mencari banyak motor dan mobil

cara eliminasi
a + b = 75 ~ x 4
4a + 2b = 210 ~ x 1

menjadi
4a + 4b = 300
4a + 2b = 210
___________-
2b = 70
b = 70/2 = 35

a + b = 75 ~ x 2
4a + 2b = 210 ~ x 1

menjadi
2a + 2b = 150
4a + 2b = 210
___________-
-2a = -60
a = -60/-2 = 30

pendapatan = (30 x 5000) + (35 x 2000)
= 150.000 + 70.000
= 220.000

Jadi pendapatan uang parkir adalah Rp220.000,00.

Grafik fungsi f(x) =x³-3x²-9x+15-5 naik pada interval ...

Jawaban

Pembahasan :

f(x) = x³ - 3x² - 9x + 15 - 5
naik pada interval ...

kita gunakan turunan
f(x) = x³ - 3x² - 9x + 15 - 5
f(x) = x³ - 3x² - 9x + 10
f'(x) = 3x² - 6x - 9 
naik pada saat f'(x) > 0

f'(x) = 3x² - 6x - 9 kita kecilkan dengan membagi 3
x² - 2x - 3 > 0
(x - 3)(x + 1) > 0
kita dapati dua bilangan -1 dan 3
kita cek nol fungsi atau x = 0

3x² - 6x - 9 > 0
3(0)² - 6(0) - 9 > 0
0 - 0 - 9 > 0
-9 > 0 salah
maka interval naik pada
x < -1 atau x > 3


jadi f(x) = x³ - 3x² - 9x + 15 - 5 naik pada interval x < -1 atau x > 3

Sebuah lingkaran berpusat di O berjari-jari OQ=12 CM.Titik P terletak di luar lingkaran dan jarak dari titik P ke pusat lingkaran adalah 20 CM.Maka panjang garis PQ adalah....

Jawaban

OQ = 12 cm, OP = 20 cm

maka

PQ² = OP² - OQ²
⇒ PQ² = 20² - 12²
⇒ PQ² = 400 - 144
⇒ PQ² = 256
⇒ PQ = √256
⇒ PQ = 16 cm
