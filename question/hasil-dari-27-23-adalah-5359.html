Sebuah kaleng berbentuk balok berukuran 10dm×8dm×6dm berisi air penuh. Bila air itu di tuangkan pada kaleng lain berbentuk prisma yg luas alasnya 96 dm2 dan tingginya 9 dm. Berapa dm tinggi air pada kaleng berbentuk prisma?

Jawaban

Volume prisma = La x t= 96 x 9= 864 dm³
volume balok = p x l x t= 10 x 8 x 6= 480 dm³
tingginya  prisma =
volume yang terisi = volume prisma - volume balok
= 864 - 480 = 384

La x t = v96t = 384t = 384 / 96t = 4 dm
Jadi ketinggian airnya adalah 4 dm

Jawaban

Kalo sejajar berarti gradiennya sama

1. x + 4y = 5
gradiennya = -a / b = -1 / 4

2. 4x + 2y = - 12
gradiennya = -a / b = -4 / 2 = -2

3. y - 2x - 5 = 0
gradiennya = -a / b = - (-2) / 1 = 2

4. 3y = x >> x - 3y = 0
gradiennya = -a / b = -1 / -3 = 1/3

karena semua gradiennya berbeda maka tidak ada yg sejajar
