U10 = 45
U13 = 57

S8 = ...?

Jawab :
Un = a + b(n-1)
U10 = a + b(10-1)
45 = a +9b ...(1)
U13 = a + b(13-1)
57 = a + 12b

57 = a + 12b
__________-
-12 = -3b
b = -12/-3
b = 4

45 = a + 9b
45 = a + 9(4)
a = 27 - 36
a = -9

Un = a + b(n-1)
U8 = -9 + 4(8-1)
U8 = -9 + 28
U8 = 19

S8 = n/2 (a + U8)
S8 = 8/2 (-9 + 19)
S8 = 4(10)
S8 = 40

jumlah 8 suku pertama dari barusan itu adalah 40.

Persamaan garis singgung lingkaran x^2+y^2=32 dititik(4, -4)

Jawaban

X² + y² = 32 ---> bagi adil
x1.x + y1.y = 32
melalui titik (4,-4)
4x - 4y = 32 ---> kedua ruas dibagi 4
x - y = 8
jadi persamaa garis singgung adalah x - y = 8
