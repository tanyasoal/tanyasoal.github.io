Pola barisan 3,9,18,45 berapakah banyak pada pola ke 8

Jawaban

Kelas 9 Matematika
Bab 6 - Barisan dan Deret

Penyelesaian :

3, 9, 18, 45, .......
--6--9--27
----3

a = 3
b = 6
c = 3

Aritmatika tingkat 2
Pola ke-8
Un = a + ( n - 1 ) b + c ( n - 1 ) ( n - 2 ) / 2
U8 = 3 + ( 8 - 1 ) 6 + 3 ( 8 - 1 ) ( 8 - 2 ) / 2
U8 = 3 + ( 7 ) 6 + 3 ( 7 ) ( 6 ) / 2
U8 = 3 + 42 + 21 ( 3 )
U8 = 45 + 63
U8 = 108

Jadi, banyak pola ke-8 adalah 108

seorang ibu mempunyai 5 orang anak, anak tertua berumur 2p tahun, dan anak termuda berumur p tahun. tiga anak yg lainnya berturut turut berumur 2p-2, p+2, p+1. jika rata rata umur mereka 17 tahun, maka umur anak tertua adalah

Jawaban

Rata² = (jumlah data) / banyaknya data
17 = (2p + p + 2p - 2 + p + 2 + p + 1) / 5
17 = (7p + 1) / 5
7p + 1 = 17 x 5
7p + 1 = 85
7p = 85 - 1
7p = 84
p = 84 / 7
p = 12

Umur anak tertua = 2p
                             = 2(12)
                             = 24 tahun

Panjang persegi panjang = 5/4 dari lebarnya luasnya 180 cm kuadrat maka kelilingnya adalah..........

Jawaban

Luas = 180cm2
panjang = 5/4 x lebar

p x l = L
(5/4 x l) x l = L
5/4 x l x l = 180cm2
l^2 = 180 x 4 : 5
l^2 = 720 : 5
l^2 = 144
l = √144
l = 12 cm
p = 180/12 = 15cm
keliling
= 2 x (p + l)
= 2 x (15 + 12)
= 2 x 27
= 54cm

pada ulangan matematika. diketahui nilai rata rata kelas adalah 58. rata rata nilai matematika peserta didik pria adalah 66 dan rata rata nilai didik wanita adalah 54. tentukan perbandingan banyaknya peserta didik pria dengan wanita

Jawaban

MAPEL: MATEMATIKA
BAB: STATISTIKA



Jawab:
misal:
pria = X
wanita = Y


58(X+Y) = 66X+54Y
58X+58Y = 66X+54Y
58Y-54Y = 66X-58X
4Y = 8X

=> dibalik

=> 4X:8Y
=> 1X:2Y


jadi perbandingan banyaknya siswa pria dan wanita adalah 1:2
