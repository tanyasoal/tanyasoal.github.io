Perbandingan uang A : B = 2: 3 .
jika uang A rp 50.000 maka uang C =

Jawaban

Dik = A:B = 2:3
Uang A = Rp, 50.000

Dit = Uang C?

Jawab = 3/2×Rp, 50.000
= 3×Rp, 25.000
= Rp, 75.000

Maaf kalau salah
Uang A : Uang B = 2 : 3 ( kali 2) sehingga menjadi 4 : 6
uang B : uang C = 2 : 5 (kali 3) sehingga menjadi 6 : 15
Diperoleh perbandingan uang A : B : C = 4 : 6 : 15
jumlah perbandingan = 25

Maka :
> uang A = 4/25 x 125.000 = 20.000
> uang B = 6/25 x 125.000 = 30.000
> uang C = 15/25 x 125.000 = 75.000

Jawaban

Sifat Sudut dalam Jajar Genjang :
1. Jumlah Besar Semua Sudutnya 360 derajat
2. Sudut Yang Berseberangan saba besar( 3. Jumlah sudut yang sepihak adalah 180 derajat (
Contoh soal : Jika diketahui sudut a adalah 120, tentukan besar sudut yang lain !


Jawab :
Sudut a = sudut c, jadi sudut c = 120.

a + d = 180
120 + d = 180
d = 180 - 120
d = 60, jadi sudut d = 60.

sudut d = sudut b, maka sudut b = 60.
