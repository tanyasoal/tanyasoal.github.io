Uno menabung di koperasi sekolah sebesar 750000 dengan bunga 10% setahun. Jika uno menerima bunga sebesar 112500.tentukan lamanya uang yang disimpan di koperasi

Jawaban

M = 750.000
b = 10%
L = 112.500
n = ?

L = M x b% x n/12
n = Lx12/(M x b%)
n = 112.500x12/(750.000 x 10%)
n = 1.350.000/75.000
n = 18 bulan
n = 18/12
n = 1½ tahun

Bu darsi membeli 280 kue bolu,450 kue lapis,dan 500 kue donat.kue kue itu akan dimasuoan dalam kotak dengan jumlah masing masing jenis sama banyak berapa kotak terbanyak yang diperlukan bu darsi untuk menyimpan kue tersebut?

Jawaban

Ini cari FPB
Faktor prima dari 280 itu 2³ x 5 x 7
Faktor prima dari 450 itu 2 x 3² x 5²
Faktor prima dari 500 itu 2² x 5³

Jadi FPBnya itu 2 x 5 yaitu 10
Jadi kotak terbanyak = 10
