Jawaban

→ Materi : Rata rata
→ Bab : Penyajian Data
————————————————

10 siswa = 10 x 162
               = 1620
Seluruhnya = (10 + 5) x 160
                    = 15 x 160
                    = 2400

= ( 2400 - 1620 ) : 5
= 780 : 5
= 156 cm
