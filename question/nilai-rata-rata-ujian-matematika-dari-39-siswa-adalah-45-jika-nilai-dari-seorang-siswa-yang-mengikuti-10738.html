Jawaban

Kalo sejajar berarti gradiennya sama

gradiennya = -a / b = -1 / 4

2. 4x + 2y = - 12
gradiennya = -a / b = -4 / 2 = -2

gradiennya = -a / b = - (-2) / 1 = 2

4. 3y = x >> x - 3y = 0
gradiennya = -a / b = -1 / -3 = 1/3

karena semua gradiennya berbeda maka tidak ada yg sejajar
