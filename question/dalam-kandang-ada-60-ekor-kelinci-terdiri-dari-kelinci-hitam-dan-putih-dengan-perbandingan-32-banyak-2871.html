Dalam kandang ada 60 ekor kelinci, terdiri dari kelinci hitam dan putih dengan perbandingan 3:2 banyak kelinci putih dikandang ada

Jawaban

→ Mapel : Math
→ Materi : Pecahan
→ Bab : Perbandingan
————————————————

Hitam : Putih,
= 3 : 2
Jumlah → 3 + 2 → 5

Banyak kelinci putih
= 2/5 x 60
= 24 ekor
60 = 3x + 2x
60 = 5x
x = 60 : 5
x = 12

maka:
banyak kelinci putih = 2x
banyak kelinci putih = 2(12)
banyak kelinci putih = 24

Via membeli sepatu ditoko seharga Rp 350.000,00.toko sepatu tersebut memberikan diskon sebesar 5,5% untuk via.berapakah harga sepatu sekarang?

Jawaban

Harga sebelum diskon = Rp. 350.000

Diskon = 5,5% x Rp. 350.000
            = 5,5/100 x Rp. 350.000
            = Rp. 19.250

Harga setelah diskon = Rp. 350.000 - Rp. 19.250
                                   = Rp. 330.750
