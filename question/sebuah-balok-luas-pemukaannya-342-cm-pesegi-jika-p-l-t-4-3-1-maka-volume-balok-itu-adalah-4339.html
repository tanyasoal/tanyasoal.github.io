Sebuah balok luas pemukaannya 342 cm pesegi. Jika p : l : t = 4 : 3 : 1, maka volume balok itu adalah a.104 cm2 b.124cm2 c.224cm2 d.324cm2

Jawaban

Luas permukaan = 342 cm²
p : l : t = 4 : 3 : 1

misal :
p = 4x
l = 3x
t = x

Luas permukaan = 2(pl + pt + lt)
342 = 2(4x . 3x + 4x . x + 3x . x)
342 = 2(12x² + 4x² + 3x²)
342 / 2 = 19x²
19x² = 171
x² = 171 / 19
x² = 9
x = √9
x = 3

maka
panjang = 4x = 4(3) = 12 cm
lebar = 3x = 3(3)= 9 cm
t = x = 3 cm

Volume = p x l x t
             = 12 x 9 x 3
             = 324 cm³

Sebuah prisma dengan alas berbentuk belah ketupat mempunyai panjang diagonal 24cm dan 10cm.Jika tinggi prisma 8cm, maka luas permukaan prisma adalah?

Jawaban

Rumus luas permukaan prisma :
(2 X Luas alas) + (Keliling Alas X tinggi prisma)

cari dulu luas alasnya
luas alas=luas belah ketupat
=Diagonal1 X Diagonal 2
=24 X 10
=240 cm2

skrg cari keliilng alas
rumusnya kan tinggal ditambah2in aja sisi2nya
13+13+13+13 (ini pk tripel pythagoras)
=52 cm

udah deh tinggal masukin ke rumus luas permukaan prisma

(2 X Luas alas) + (Keliling Alas X tinggi prisma)
(2 X 240) + (52 X 8)
=896 cm2
