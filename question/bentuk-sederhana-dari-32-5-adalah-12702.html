Jika titik a (3,-2,-1) b (1,-2,1) c (7,p-1,-5) terletak pada garis lurus tentukan x+y

Jawaban

AB = b - a = (1,-2,1)- (3,-2,-1) = (-2, 0, 2)

BC = c - b = (7, p-1,-5) -(1, -2,1) = (6,p+1, -6)

AB = k BC
-2 =k.6
k = -2/6
k =-1/3

0 = k( p+1)
0 = -1/3 (p+1)
0= p+1
