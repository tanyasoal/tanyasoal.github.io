Alas sebuah prisma berbentuk belah ketupat yang panjang diagonalnya 2:3, jika tinggi prisma 20cm dan volume prisma 960 maka d2 nya adalah

Jawaban

Dik : V = 960 cm3
       tinggi prisma : 20 cm
       perbandingan diagonal = d1 : d2 = 2 : 3
dit : d2
jawab : gunakan pemisalan X pada perbandingan sehingga menjadi d1 = 2X dan d2 = 3X

V = luas alas x tinggi
960 = [(2X x 3X/2)] x 20
960 = 3X2 (kuadrat) x 20
960 = 60X2
X2 = 960/60
X2 = 16
X = 4

d2 = 3X
     = 3 (4)
     = 12

Jadi, d2 = 12 cm

Persamaan garis lurus mellaui tituk N(-2,5) dan tegak lurus dengan garis 4x+3y=12 adalah

Jawaban

Cari gradien dari persamaan 4x + 3y = 12

Bentuk umum persamaan garis
ax + by + c = 0

untuk persamaan tersebut :
a = 4
b = 3
c = -12

m = -a/b = -4/3

Karena tegak lurus,
m1 . m2 = -1
-4/3 . m2 = -1
m2 = 3/4

Persamaan garis melalui titik (-2, 5)
(y - y1) = m(x - x1)
(y - 5) = 3/4(x - (-2))
4(y - 5) = 3(x + 2)
4y - 20 = 3x + 6
3x - 4y = -20 - 6
3x - 4y = -26
