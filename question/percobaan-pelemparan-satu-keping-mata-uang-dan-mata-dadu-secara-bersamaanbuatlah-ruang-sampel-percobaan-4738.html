Diketahui dua buah lingkaran masing masing berjari jari 14cm dan 2cm jika titik kedua pusat lingkaran 20 cm makan panjang garis singgung persekutuan luar kedua lingkaran adalah...

Jawaban

Mapel = Matematika
Bab = Lingakaran
Materi = Garis Singgung Persekutuan Luar

Diketahui :
r1 = 14 cm
r2 = 2 cm
p = 20 cm

Penyelesaian :
L² = p² - (r1 - r2)²
L² = (20 cm)² - (14 cm - 2 cm)²
L² = 400 cm² - 144 cm²
L² = 256 cm²
L = √256 cm²
L = 16 cm

Jadi, panjang garis singgung persekutuan luarnya adalah 16 cm.

Arjuna menabung pada sebuah bank yang memberikan suku bunga 9% pertahun. Jika besar tabungan Bima setelah 30 bulan Rp 2.205.000,00, berapakah tabungan awal Arjuna…

Jawaban

Misal saldo awal = a

suku bunga 30 bulan = 30/12 x 9%
                                   = 5/2 x 9%
                                   = 22,5 %

saldo akhir = saldo awal + 22.5% x saldo awal
2.205.000 = a + 22,5% a
2.205.000 = 122,5 % a
a = 2.205.000 : 122,5 / 100
a = 2.205.000 x 100 / 122,5
a = 1.800.000

tabungan awal Arjuna adalah Rp. 1.800.000
→ Mapel : Math
→ Materi : Tabungan Awal
→ Bab : Aritmatika Sosial

= 100/122,5 x 2.205.000
= Rp 1.800.000,00

Sebuah tangga yang panjangnya 13meter disandarkan keujung tiang listrik jika jarak 5meter maka tinggi tiang listrik adalah?

Jawaban

Mapel          : Matematika
Kategori      : Teorema Pythagoras
Kata Kunci  : Teorema Pythagoras, Segitiga
Kelas           : 8
Kode            : 8.2.9 - [ Bab - Teorema Pythagoras ]


≡ Panjang Tangga                                = 13 meter
≡ Jarak pangkal tangga ke tiang listrik =  5 meter


Maka, Tinggi Tiang Listrik tersebut dihitung dengan menggunakan teorema pythagoras, dimana :
      b² = c² - a²
      b  = √(c² - a²)
      b  = √(13² - 5²)
      b  = √(169 - 25)
      b  = √(144)
      b  = 12 cm

Jadi, Tinggi tiang listrik adalah 12 cm
