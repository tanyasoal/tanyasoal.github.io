di tempat parkir sebuah pertokoan terdapat 75 kendaran yang terdiri dari mobil dan motor.banyak roda seluruh nya ada 210.jika tarif parkir mobil Rp.5.000 dan motor Rp 2.000 maka pendapatan uang parkir saat itu adalah

Jawaban

4mobil = a
motor = b

roda mobil = 4 = 4a
roda motor = 2 = 2b

mencari banyak motor dan mobil

cara eliminasi
a + b = 75 ~ x 4
4a + 2b = 210 ~ x 1

menjadi
4a + 4b = 300
4a + 2b = 210
___________-
2b = 70
b = 70/2 = 35

a + b = 75 ~ x 2
4a + 2b = 210 ~ x 1

menjadi
2a + 2b = 150
4a + 2b = 210
___________-
-2a = -60
a = -60/-2 = 30

pendapatan = (30 x 5000) + (35 x 2000)
= 150.000 + 70.000
= 220.000

Jadi pendapatan uang parkir adalah Rp220.000,00.

Perbandingan sederhana dari 1/4 lusin : 3/4 kodi adalah

Jawaban

Semua nya kita jadikan perbuah
1 lusin=12 buah
1 kodi=20 buah
Jadi,
1/4 lusin=1/4x12 buah=3 buah
3/4 kodi=3/4x20 buah=15 buah
1/4 lusin : 3/4 kodi = 3 : 15 (disederhanakan)
Jadi, 1 : 5
1/4 lusin : 3/4 kodi = 1/4 (12) : 3/4 (20) = 3 : 15 = 1 : 5
