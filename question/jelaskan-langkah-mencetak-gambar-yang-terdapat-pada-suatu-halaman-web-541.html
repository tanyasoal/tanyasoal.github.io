Soal MtK . 1. Harga 2 komponen A dan 3 komponen B adalah Rp28.000,00. Harga 3 komponen A dan 1 komponen B adalah Rp21.000,00. Harga 1 komponen A dan 2 komponen B adalah ?

Jawaban

Misalnya :
Komponen A = x
Komponen B = y

Penyelesaian :
Diketahui :
2x + 3y = Rp. 28.000
3x + y = Rp. 21.000

Ditanya :
x + 2y = ...?

Jawab :
=> subsitusikan ke persamaan (1) & (2)

2x + 3y = Rp. 28.000 | × 1
3x + y = Rp. 21.000 | × 3
____________________-
=
2x + 3y = Rp. 28.000
9x + 3y = Rp. 63.000
_________________-
-7x = -35.000
x = -35.000/-7
x = Rp. 5.000

=> Eliminasikan ke persamaan (1) atau (2)
2x + 3y = Rp. 28.000
2(Rp. 5.000) + 3y = Rp. 28.000
Rp. 10.000 + 3y = Rp. 28.000
3y = Rp. 28.000 - Rp. 10.000
3y = Rp. 18.000
y = 18.000/3
y = Rp. 6.000

Jadi, harga 1 komponen A dan 2 komponen B adalah :
=> x + 2y
= Rp. 5.000 + 2(Rp. 6.000)
= Rp. 5.000 + Rp. 12.000
= Rp. 17.000
