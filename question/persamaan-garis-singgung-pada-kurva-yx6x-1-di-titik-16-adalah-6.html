Koordinat banyak titik P(-1,3) jika didilatasi terhadap (0,2)dan dilanjutkan dengan (0,-90 derajat)

Jawaban

P(x,y) didilatasikan (0,k) ==> P'(kx,ky)
P(-1,3) didilatasikan (0,2) ==> P'(2×-1,2×3)
P'(-2,6)

P'(x,y) dirotasikan (0,-90°) ==> P''(y,-x)
P'(-2,6) dirotasikan (0,-90°) ==> P''(6,2)

Maka, bayangan titik P adalah (6,2)

Jawaban

•Cara (1)
y = (x2 + 3) (x - 2)
y = x3 - 2x2 + 3x - 6
y' = 3x2 - 2(2)x + 3 + 0
y' = 3x2 - 4x + 3

•Cara(2)
y = (x2 + 3) (x - 2)
Misal :
u = x2 + 3
u' = 2x
v = x - 2
v' = 1

y = u.v
y' = u'v + uv'
y' = (2x)(x-2) + (x2+3)(1)
y' = 2x2 - 4x + x2 + 3
y = 3x2 - 4x + 3

*angka sesudah x adalah pangkat
