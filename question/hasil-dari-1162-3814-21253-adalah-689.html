Banyaknya sulu barisan 8.18.28...898

Jawaban

Mapel = Matematika
Materi = Barisan Aritmetika

Diketahui :
U1 = 8
Un = 898
b = 10

Penyelesaian :
Un = a + (n - 1)b
898 = 8 + (n - 1)10
898 = 8 + 10n - 10
898 = 10n - 2
-10n = -2 - 898
-10n = -900
n = -900/-10
n = 90

Jadi, banyaknya suku pada barisan tersebut adalah 90
