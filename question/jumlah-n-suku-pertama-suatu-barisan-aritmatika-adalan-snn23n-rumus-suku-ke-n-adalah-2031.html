Jumlah n suku pertama suatu barisan aritmatika adalan Sn=n^2+3n rumus suku ke n adalah

Jawaban

Sn diketahui
Sn = n^2 + 3n

Rumus Sn yg ada Un-nya
Sn = n/2 [a + Un]

Sn = Sn
n^2 + 3n = n/2 (a + Un)
*dikali 2 semua, biar ga ada yg pecahan
2n^2 + 6n = n (a + Un)
2n^2 +6n = an + n(Un)
2n^2 + 6n -an = n(Un)
2n^2 + (6-a)n = n(Un)
*bagi dengan n semua, supaya hanya tersisa Un
2n + 6- a = Un
Un = 2n + 6 - a

Rumus suku ke-n nya adalah 2n + 6 -a

Diketahui deret geometri dengan suku pertama 5 dan suku keempat 135.jumlah 6 suku pertama adalah

Jawaban

Diketahui :
Deret geometri
U1 = 5
U4 = 135

Ditanya :
S6 = ?

Dijawab :
U1 = a = 5
U4 = a.r^3 = 135

       U4/U1 = 135/5
a.r^3/a = 27
            r^3 = (3)^3
                r = 3

   a.r^3 = 135 
  a.3^3 = 135
   a. 27 = 135
       a = 135/27
       a = 5

Sn = a.(r^n -1)/(r - 1)
S6 = 5.(3^6 -1)/(3 - 1)
S6 = 5.(729 - 1))/(2)
S6 = 5.728/2
S6 = 3640/2
S6 = 1820

Jadi jumlah emapat suku pertama deret tersebut adalah 1.820

Zaki berangkat dari kota A ke kota B yang berjarak 240 km dengan kecepatan rata-rata 40 km/jam. Apabila zaki berangkat pukul 07.00 WIB, pukuo berapakah ia sampai di kota B ?

Jawaban

→ Mapel : Math
→ Materi : Waktu
→ Bab : Perbandingan
————————————————

Waktu
= Jarak ÷ Kecepatan
= 240 km ÷ 40 km/jam
= 6 jam

Sampai
= 07.00 + 6 jam
= 13.00
Jadi Zaki tiba di kota B pada pukul 13.00
S = 240 km
v = 40 km/jam

t = s / v
  = 240 / 40
  = 6 jam

t₁ = 07.00

t₂ = t₁ + t
    = 07.00 + 06.00
    = 13.00

maka Zaki akan tiba di kota B pukul 13.00
