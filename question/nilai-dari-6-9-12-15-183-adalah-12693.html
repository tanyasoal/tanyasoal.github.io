Nilai dari 6 + 9 + 12 + 15 + ... + 183 adalah

Jawaban

A = 6
b = 9 - 6 = 3
Un = 183

Un = a + (n - 1)b
183 = 6 + (n - 1)3
3(n - 1) = 183 - 6
3(n - 1) = 177
n - 1 = 177 / 3
n - 1 = 59
n = 59 + 1
n = 60

Sn = n/2(a + Un)
S60 = 60/2(6 + 183)
S60 = 30 x 189
S60 = 5.670

Dik : U2 + U6 = -6
U2 + U8 = -2
Dit : S5?

Jawab :
•U2 + U6 = -6 dipecah menjadi
a + b + a + 5b = -6
2a + 6b = -6
___________ : 2
a + 3b = -3

• U2 + U8 = -2 dipecah menjadi
a + b + a + 7b = -2
2a + 8b = -2
__________ :2
a + 4b = -1

•Dieliminasi
a + 3b = -3
a + 4b = -1
_________ -
-b = -2
b = 2

• Disubtitusi
a + 3b = -3
a = -3 - 3(2)
a = - 3 - 6
a = -9

•Mencari S5

S5 = 5/2 (-10)
