Diketahui :
r1 = 14 cm
r2 = 2 cm
p = 20 cm

Penyelesaian :
L² = p² - (r1 - r2)²
L² = (20 cm)² - (14 cm - 2 cm)²
L² = 400 cm² - 144 cm²
L² = 256 cm²
L = √256 cm²
L = 16 cm

Jadi, panjang garis singgung persekutuan luarnya adalah 16 cm.

(-16) + 8 x (-4) : 2 - (-20)
= (-16) + (-32) : 2 - (-20)
= (-16) + (-16) + 20
= (-32) - (-20)
= (-12)
-16 + 8 x (-4) : 2 - (-20)
= (-16) + (-32) : 2 + 20
= (-16) + (-16) + 20
= (-16) - 16 + 20
= (-32) + 20
