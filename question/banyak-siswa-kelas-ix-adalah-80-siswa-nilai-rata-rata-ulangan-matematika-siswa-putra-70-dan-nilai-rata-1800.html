Pusat jari jari lingkaran dengan persamaan x^+y^-4x-8y+15=0 berturut turut adalah

Jawaban

Pembahasan :

diberikan persamaan lingkaran
x² + y² - 4x - 8y + 15 = 0
ditanya pusat dan jari-jari lingkaran

persamaan umum lingkaran
(x - a)² + (y - b)² = r²
dengan pusat (a, b)

x² + y² - 4x - 8y + 15 = 0 → kita buat menjadi bentuk persamaan umum lingkaran
x² - 4x + y² - 8y = -15
x² - 4x + (1/2 x -4)² + y² - 8y + (1/2 x -8)² = -15 + (1/2 x -4)² + (1/2 x -8)²
x² - 4x + 4 + y² - 8y + 16 = -15 + 4 + 16
(x - 2)² + (y - 4)² = 5

pusat lingkaran = (2, 4)
jari-jari lingkaran = √5

Arjuna menabung pada sebuah bank yang memberikan suku bunga 9% pertahun. Jika besar tabungan Bima setelah 30 bulan Rp 2.205.000,00, berapakah tabungan awal Arjuna…

Jawaban

Misal saldo awal = a

suku bunga 30 bulan = 30/12 x 9%
                                   = 5/2 x 9%
                                   = 22,5 %

saldo akhir = saldo awal + 22.5% x saldo awal
2.205.000 = a + 22,5% a
2.205.000 = 122,5 % a
a = 2.205.000 : 122,5 / 100
a = 2.205.000 x 100 / 122,5
a = 1.800.000

tabungan awal Arjuna adalah Rp. 1.800.000
→ Mapel : Math
→ Materi : Tabungan Awal
→ Bab : Aritmatika Sosial

Persentase bung,
= 30/12 x 9%
= 22,5%

Tabungan akhir
= 100% + 22,5%
= 122,5%

Tabungan awal
= 100/122,5 x 2.205.000
= Rp 1.800.000,00
