Sebuah fungsi dinyatakan dengan rumus f(x) = ax + b. Jika f(-2) = -4 dan f(3) = 21,maka f(7) adalah

Jawaban

F(x) = ax + b

f(-2) = a(-2) + b = -4
-2a + b = -4

f(3) = a(3) + b = 21
3a + b = 21

eliminasi b
-2a + b = -4
3a + b = 21
__________-
-5a = -25
a = 5

mencari b
3(5) + b = 21
15 + b = 21
b = 21-15
b = 6

f(7) = 5(7) +6
= 35 + 6
= 41
