U9 = a + b(9-1)
28 = a + 8b ... (2)

=> subsitusikan ke persamaan (1) dan (2)
13 = a + 3b
28 = a + 8b
_________-
-15 = -5b
b = -15/-5
b = 3

=> eliminasikan ke persamaan (1) atau (2)
13 = a + 3b
13 = a + 3(3)
13 = a + 9
a = 13 - 9
a = 4

=> menentukan rumus Un :
Un = a + b(n-1)
Un = 4 + 3(n-1)
Un = 4 + 3n -3
Un = 1 + 3n

=> U20 = 1 + 3(20) = 1 + 60 = 61

=> Jumlah dua puluh suku pertama deret tersebut adalah
Sn = n/2 (a + Un)
S20 = 20/2 (4 + U20)

Alas sebuah prisma berbentuk belah ketupat dengan panjang digonal masing masing 24 cm dan 18 cm. Tinggi prisma tersebut adalah 20 cm. Luas permukaan prisma adalah?

Jawaban



Lp = ( 2 × luas alas ) + ( keliling alas × tinggi )
= ( 2 × 1/2 × d1 × d2 ) + ( 4 × s × t )
= ( 2 × 1/2 × 24 × 18 ) + ( 4 × 12.7 × 20 )
= 432 + ( 50.8 × 20 )
= 432 + 1016
= 1448 cm2
