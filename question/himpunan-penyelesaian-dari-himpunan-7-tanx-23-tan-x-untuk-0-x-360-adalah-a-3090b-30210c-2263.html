Himpunan penyelesaian dari himpunan 7 tanx = 2√3 + tan x, untuk 0 < x <360 adalah A. 30,90
B. 30,210
C. 90,120
D. 90,180
E. 135, 180

Jawaban

7 tan x = 2√3 + tan x
6 tan x = 2√3
tan x = 2√3 / 6
tan x = 1/3 √3
tan x = 30°

x = 30° + n.180°
Untuk n = 0 → 30° + 0.180° = 30°
Untuk n = 1 → 30° + 1.180° = 210°

Himpunan penyelesaiannya adalah 30° dan 210°
Jawaban : B

Diketahui persamaan 2x_3y=20 dan x+4y=-1 nilai 2xy adalah

Jawaban

Kelas 8 Matematika
Bab SPLDV

x + 4y = -1
x = -4y - 1

2x - 3y = 20
2 (-4y - 1) - 3y = 20
-8y - 2 - 3y = 20
-11y = 20 + 2
y = 22/-11
y = -2

x = -4y - 1
x = -4 . (-2) - 1
x = 8 - 1
x = 7

2 . x . y = 2 . 7 . (-2)
2xy = -28

Jawaban

Mapel : Matematika
Materi : Perbandingan
Kelas  : 6

≡ Selisih perbandingan
     = 4 - 3
    = 1

≡ Kelereng Budi
     = 56 x 3
    = 168 Buah

≡ Kelereng Adi
     = 56 x 4
     = 224 Buah 

Maka, Jumlah kelereng Mereka adalah 
    = 224 + 168
    = 392 Buah
