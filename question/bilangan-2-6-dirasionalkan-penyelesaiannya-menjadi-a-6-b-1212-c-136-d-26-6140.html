Hitung luas trapesium sama kaki jika garis yang sejajar ukurannya 15 cm dan 27 cm.Panjang kakinya 10 cm!!

Jawaban

→ Mapel : Math
→ Materi : Luas
→ Bab : Bangun Datar
————————————————

Panjang sisi alas bagian segitiga
= ( 27 - 15 ) : 2
= 12 : 2
= 6 cm

Tinggi trapesium
= √( 10² - 6² )
= √( 100 - 36 )
= √( 64 )
= 8 cm

Luas trapesium
= x ( a + b ) x t
= x ( 15 + 27 ) x 8
= 42 x 4
= 168 cm²

Jawaban

sn=3((2^n)-1)/2-1=765
sn=3((2^n)-1)=765
sn=((2^n)-1)=765/3
