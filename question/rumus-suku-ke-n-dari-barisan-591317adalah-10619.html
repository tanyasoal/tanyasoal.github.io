Jawaban

A(3, 2) dan B(5, 4)x₁ = 3x₂ = 5y₁ = 2y₂ = 4
Persamaan garis(y - y₁) / (y₂ - y₁) = (x - x₁) / (x₂ - x₁)(y - 2) / (4 - 2) = (x - 3) / (5 - 3)(y - 2) / 2 = (x - 3) / 2y - 2 = x - 3y = x - 3 + 2y = x - 1
