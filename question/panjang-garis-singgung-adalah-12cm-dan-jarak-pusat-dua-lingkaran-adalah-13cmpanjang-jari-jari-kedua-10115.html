Jumlah panjang rusuk sebuah kubus adalah 96cm. Luas permukaan dan volume nya brp ya?

Jawaban

Mapel = Matematika
Materi = Bangun Ruang

12 . Rusuk = Jumlah panjang rusuk
12 . Rusuk = 96 cm
Rusuk = 96 cm / 12
Rusuk = 8 cm

Luas Permukaan Kubus
= 6(S x S)
= 6(8 cm x 8 cm)
= 6(64 cm²)
= 384 cm²

Volume Kubus
= S x S x S
= 8 cm x 8 cm x 8 cm
= 512 cm²

jumlah dua bilangan bulat 120 jika bilangan kedua sama dengan sama dengan dua kali bilangan pertama. tentukan kedua bilangan tersebut

Jawaban

Misal
bilangan pertama = a
bilangan kedua = b

a + b = 120
b = 2a

Substitusi
a + b = 120
a + 2a = 120
3a = 120
a = 120 / 3
a = 40

b = 2a
b = 2(40)
b = 80

maka :
bilangan pertama = 40
bilangan kedua = 80
