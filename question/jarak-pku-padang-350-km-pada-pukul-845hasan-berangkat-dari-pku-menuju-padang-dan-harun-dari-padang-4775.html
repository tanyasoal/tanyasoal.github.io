Perbandingan uang A : B = 2: 3 .
jika uang A rp 50.000 maka uang C =

Jawaban

Dik = A:B = 2:3
Uang A = Rp, 50.000

Dit = Uang C?

Jawab = 3/2×Rp, 50.000
= 3×Rp, 25.000
= Rp, 75.000

Maaf kalau salah
Uang A : Uang B = 2 : 3 ( kali 2) sehingga menjadi 4 : 6
uang B : uang C = 2 : 5 (kali 3) sehingga menjadi 6 : 15
Diperoleh perbandingan uang A : B : C = 4 : 6 : 15
jumlah perbandingan = 25

Maka :
> uang A = 4/25 x 125.000 = 20.000
> uang B = 6/25 x 125.000 = 30.000
> uang C = 15/25 x 125.000 = 75.000

Diketahui m=16 2 per 3% dan n=66 2 per 3%
hasil m:n adalah ...

Jawaban

Jawabannya '


Mapel : Matematika
Bab : Operasi Bilangan
Sub Bab : Perbandingan

Pembahasan :

m = 16 2/3% = 50%
n = 66 2/3% = 200%

Maka '

m : n
= 50% : 200%
= 50/100 : 200/100
= 1/2 : 2/1
= 1/2 × 1/2
= 1/4 → Jwbnya
