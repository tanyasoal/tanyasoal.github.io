Jawaban

Sisa hutang = 18.000.000 - 6.000.000 = 12.000.000
bunga hutang = 12.000.000 x 10% x 4 = 4.800.000
besar angsuran per bulan = (12.000.000 + 4.800.000)/(4x12)
= 16.800.0000/48
= Rp. 350.000
Utang budi = Rp 18.000.000. - Rp 6.000.000,
= Rp 12.000.000,-
bunga 4 tahun = 4 x 10% x Rp 12.000.000,-
= Rp 4.800.000,-
Jumlah utang + bunga
= Rp 12.000.000,- + Rp 4.800.000,
= Rp 16.800.000,
Besarnya angsuran setiap bulan
= Rp 16.800.000, - : 48
