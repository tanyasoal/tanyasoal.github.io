jumlah dua bilangan bulat 120 jika bilangan kedua sama dengan sama dengan dua kali bilangan pertama. tentukan kedua bilangan tersebut

Jawaban

Misal
bilangan pertama = a
bilangan kedua = b

a + b = 120
b = 2a

Substitusi
a + b = 120
a + 2a = 120
3a = 120
a = 120 / 3
a = 40

b = 2a
b = 2(40)
b = 80

maka :
bilangan pertama = 40
bilangan kedua = 80
