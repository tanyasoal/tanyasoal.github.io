Grafik fungsi f(x) =x³-3x²-9x+15-5 naik pada interval ...

Jawaban

Pembahasan :

f(x) = x³ - 3x² - 9x + 15 - 5
naik pada interval ...

kita gunakan turunan
f(x) = x³ - 3x² - 9x + 15 - 5
f(x) = x³ - 3x² - 9x + 10
f'(x) = 3x² - 6x - 9 
naik pada saat f'(x) > 0

f'(x) = 3x² - 6x - 9 kita kecilkan dengan membagi 3
x² - 2x - 3 > 0
(x - 3)(x + 1) > 0
kita dapati dua bilangan -1 dan 3
kita cek nol fungsi atau x = 0

3x² - 6x - 9 > 0
3(0)² - 6(0) - 9 > 0
0 - 0 - 9 > 0
-9 > 0 salah
maka interval naik pada
x < -1 atau x > 3


jadi f(x) = x³ - 3x² - 9x + 15 - 5 naik pada interval x < -1 atau x > 3

Sebuah balok berukuran panjang (5x+1)cm lebarr(3x-2)cm dan tinggi (x+5)cm sedangkan jumlah panjang rusuk rusaknya adalah 124 cm tentukan a. Nilai x? B. Luas balok? C.volume balok Tolong dijawab penting buat besok

Jawaban

MATEMATIKA
Geometri

Diketahui
- Balok
- (p) Panjang (5x+1) cm
- (l) Lebar (3x-2) cm
- (t) Tinggi (x+5) cm
- Jumlah semua rusuk 124 cm

Ditanya
- Nilai x
- Luas permukaan balok
- Volume balok

Jawab
a. Nilai x = ... ?

Panjang rusuk = 4(p + l + t)
124 = 4((5x + 1) + (3x - 2) + (x + 5))
31 = (5x + 1) + (3x - 2) + (x + 5)
31 = 5x + 1 + 3x - 2 + x + 5
31 = 9x + 4
31 - 4 = 9x
27 = 9x
x = 27 ÷ 9 = 3

Jadi, nilai x adalah 3.

b. Luas permukaan balok

Panjang balok = 5x + 1 = 5 × 3 + 1 = 16 cm
Lebar balok = 3x - 2 = 3 × 3 - 2 = 7 cm
Tinggi balok = x + 5 = 3 + 5 = 8 cm

Luas permukaan balok = 2(pl + pt + lt)
= 2(16 × 7 + 16 × 8 + 7 × 8)
= 2(112 + 128 + 56)
= 2(296)
= 592 cm²

Jadi, luas permukaan balok = 592 cm²

c. Volume balok

Volume balok = p × l × t
= 16 × 7 × 8
= 896 cm³

Jadi, volume balok 896 cm³

lebar = 5(3) = 15 cm
