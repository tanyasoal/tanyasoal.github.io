jika perbandingan jumlah lilitan primer dan sekunder pada sebuah trafo adalah 1 : 9, jika tegangan primernya adalah 100 volt dan dayanya 750 watt.. berapa besar kuat arus sekundernya?

Jawaban

Diket :

Np : Ns = 1 : 9
Vp = 100 v
Pp = 750 w


Dit : Arus sekunder.......?


Jwb :

# Tegangan sekunder

Vs = ( Ns . Vp ) / Np
Vs = ( 9 . 100 ) / 1
Vs = 900 volt

# Arus primer

Ip = Pp / Vp
Ip = 750 / 100
Ip = 7,5 A


# Arus sekunder

Is = ( Vp . Ip ) / Vs
Is = ( 100 . 7,5 ) / 900
Is = 750 / 900
Is = 0,83 A


Jadi arus sekundernya 0,83 A

sebuah trafo arus primer dan sekundernya masing masing 3A dan 2A. jumlah lilitan primer dan sekunder masing masing 4.000 dan 1.200, efisiensi trafo sersebut adalah...%

Jawaban

Diket :
Ip = 3A
Is = 2A
Np = 4000
Ns = 1.200

ditanya :
efisiensi trafo = ?

Jawab :
Np/Ns = Is/Ip = Vp/Vs
4000/1.200 = 2A/3A = Vp/Vs
4000 × 3 / 1.200 × 2 = Vp/Vs
12000 /2.400 = Vp/Vs
120 = Vp
24 = Vs

efisiensi trafo
= Ps/Pp × 100%
= Vs × Is / Vp × Ip × 100%
= 24 × 2 / 120 × 3 × 100%
= 48 / 360 × 100%
= 4/30 × 100%
= 13,3 %
