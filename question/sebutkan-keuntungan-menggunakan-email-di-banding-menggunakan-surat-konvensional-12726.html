Sebutkan langkah langkah untuk mengakses situs website Buku Sekolah Elektronik

1. Buka Web Browser

2. Pada Address bar, ketikan alamat Website Buku Sekolah Elektronik

3. Tekan tombol Enter

4. Website akan segera terbuka
- Klik tombol Start Windows
- Klik All Programs
- Klik Web Browser yang diinginkan (Misalnya Internet Explorer) atau dengan langsung klik kiri dua kali pada icon Internet Explorer
